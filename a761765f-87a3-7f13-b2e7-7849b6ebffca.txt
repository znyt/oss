绍洪被唐清山一顿奚落，顿时羞得满脸通红，不过面对唐家这位公子哥，他哪里有反驳的勇气？只好一脸讪讪的退了出去。<br/>　　<br/>　　    ……<br/>　　<br/>　　    唐清雅苏醒过来之后，又迷糊了好一会，方才看清楚眼前之人是叶尘，顿时露出一抹笑容，轻声道:<br/>　　<br/>　　    “我这是到天堂了吗？没想到这么快就见到你啦~”<br/>　　<br/>　　    叶尘微微一笑，伸手在唐清雅的琼鼻上轻轻捏了一下，道:<br/>　　<br/>　　    “傻瓜，有我在呢，怎么会让你死掉？你现在活得好好的，而且马上就能蹦能跳，能吃能睡，长命百岁！哦，不对，就算是活上一千岁、一万岁，也不是没有可能！”<br/>　　<br/>　　    “噗嗤~”<br/>　　<br/>　　    唐清雅听到这话，顿时笑的更厉害了，<br/>　　<br/>　　    “一千岁，一万年，那我岂不是成王八了？”<br/>　　<br/>　　    叶尘一本正经道:<br/>　　<br/>　　    “只要你想的话，我保证王八也没你活得久！”<br/>　　<br/>　　    ……<br/>　　<br/>　　    叶尘一边和唐清雅聊着天，一边继续用真元，为她调理经脉，修复身体的各个器官。<br/>　　<br/>　　    唐家众人见状，顿时都纷纷退了出去，唐清雅的妈妈原本想要上前，也被唐清山直接硬拖了出去。<br/>　　<br/>　　    很快，<br/>　　<br/>　　    在叶尘的真元滋润之下，唐清雅的身体开始迅速恢复，原本憔悴苍白的脸庞，也开始变得红润有光泽，甚至比之前的气色还要好。<br/>　　<br/>　　    而唐清雅，一开始还有些迷糊，渐渐地，神志也逐渐清醒，这才真的确信自己没死，叶尘也没有死，顿时坐起身来，哇哇大哭。<br/>　　<br/>　　    叶尘见状，顿时有些不明白了，连忙坐在床头，正准备劝慰她一番，却不料这妮子，居然直接埋头扑进了他的怀里，放声大哭起来。<br/>　　<br/>　　    足足哭了好一会，唐清雅这才从叶尘的怀中起来，双眼通红，抡起小拳头，在叶尘胸膛上一连捶了几下，一脸幽怨道:<br/>　　<br/>　　    “你既然没死，干嘛也不回来？让我白白为你伤心！”<br/>　　<br/>　　    叶尘苦笑道:<br/>　　<br/>　　    “我有重要的事情要去做，而且我哪知道你也在现场啊？”<br/>　　<br/>　　    说完这话，叶尘下意识的一低头，顿时差点没有喷出鼻血来！<br/>　　<br/>　　    原来，唐清雅因为重病，只穿了一件宽大的病号服，而且里面没有穿内衣，那白花花的一片，顿时一览无遗……<br/>　　<br/>　　    旋即，叶尘又想起之前，自己还按在那个地方足足好几分钟……<br/>　　<br/>　　    ‘没想到这丫头年纪不大，还挺有料！’<br/>　　<br/>　　    也不知道怎的，叶尘脑海中，忽然蹦出这么一道念头来，顿时老脸一红。<br/>　　<br/>　　    上架第一天，写了多少就更了多少，没有存稿，从明天开始，统一放在中午十二点更新，保底四更，加更的话就放在晚上**点左右。感谢打赏！感谢所有订阅支持的兄弟！感谢大家的月票、推荐票！谢谢！<br/><br/><br/>Ps:书友们，我是我意如刀，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。请您关注微信公众号：dazhuzaiyuedu（长按三秒复制）书友们快关注起来吧！<br/><br/><br/>
