　　第366章
<br/>　　程大雷身体倒是没有什么大事，主要是脱力引起的昏迷。他简单清理过伤口，又好好睡了一宿，便恢复了元气。
<br/>　　相比之下，樊梨花的情况就严重许多。刘悲亲自为她处理伤口，但伤筋动骨一百天，想要真正好转怕还得有些时日。
<br/>　　更严重的问题是，樊梨花状态有些不对，精神萎靡，食欲不振，好像……有些抑郁症的征兆。
<br/>　　当然，这也是可以理解的事情。她先遭遇了恐怖的崔弄海，最后算是侥幸活了下来，结果她的大夫却是刘悲。
<br/>　　刘悲就是个行走的负能量发射器，言谈话语间，随意泄出点负能量，就可以打破一个人生存的希望。如此，樊梨花患上抑郁症，是再正常不过的事情了。
<br/>　　这件事，倒是程大雷疏忽了，就不该让刘悲当樊梨花的大夫。
<br/>　　这一日，程大雷专程去探望樊梨花，如今的樊梨花就住在将军府，也安排了丫鬟照顾。
<br/>　　刚进去就吓了一跳，只见樊梨花穿着长裙，对着庭前的落叶长吁短叹。
<br/>　　这完全不是樊梨花的画风呐，程大雷踏入院子，谨小慎微的靠近樊梨花。
<br/>　　“歇着呢，身体好点没有？”
<br/>　　樊梨花幽怨的看了程大雷一眼，道：“好了又怎样，不好又怎样，活着，本就无所谓的。”
<br/>　　程大雷倒抽一口凉气，本来活蹦乱跳的樊梨花如何变成这个样子。自然，这是拜崔弄海所赐，可其中怕也有刘悲的很大原因。
<br/>　　身上的伤易好，可心里面受的刺激怕是不那么容易好转。如今她小腿还瘸着，只能被人搀扶着走路。程大雷坐在她身边，又陪着她说了一会话。
<br/>　　樊梨花始终还是那病恹恹的模样，言谈之中三句两句透着厌世情绪。这情绪定然是刘悲传染给她的，可看这劲头，也快传染给程大雷了。
<br/>　　程大雷离去的时候，正好撞见刘悲。
<br/>　　“你干嘛去？”
<br/>　　“去给樊寨主送药啊，我可是一点儿不敢耽搁，药都是我亲自让人熬的。”刘悲道。
<br/>　　“罢了罢了，这点小事，让手下人去做不就可以了么。以后，你还是少和樊寨主见面，对了，抓回来那人怎样了。”
<br/>　　“现在被关在地牢中，遵大当家吩咐，没让人去碰他，就等着大当家处理呢。”
<br/>　　所有人都知道，崔弄海是顿大餐，一定要程大雷亲自炮制。
<br/>　　“走，咱们一起去看看。”
<br/>　　程大雷揽过刘悲肩膀，迈步走向琴川关的地牢。
<br/>　　崔弄海被关进水牢中，虽然说没有过分处置他，但这两天他吃得苦头也不少。
<br/>　　程大雷进去的时候，见他半边身子泡在臭水中，整个人已经半人半鬼，可怖得紧。
<br/>　　但他的求生欲很强，丢进水池中的食物都被他抢过来吃了，双眼硕硕放光，生机很是旺盛。
<br/>　　程大雷走到牢房外，居高临下的看着他。崔弄海裂开嘴，也对程大雷笑了笑。
<br/>　　“怎样，对环境有没有要求，我们这里一直是优待俘虏的。”
<br/>　　崔弄海用一双眼睛盯着程大雷，发出桀桀的冷笑。
<br/>　　“你会杀了我么？”崔弄海突然收起脸上的笑容。
<br/>　　“杀你是迟早的事情。”程大雷道：“但那样多便宜你，我正在想如何处置你，可选的方案太多了，你能不能给我点参考意见。我会问你几个问题，如果你老实配合的话，我可能会让你少受点痛苦。”
<br/>　　程大雷的确要对这崔弄海做一些调查，这样的一个猛人，该不会是凭空冒出来的。总是要摸清楚他的来历，有什么同伙之类的。
<br/>　　崔弄海笑着：“从我嘴里你什么都不会得到。”
<br/>　　程大雷白他一眼：“你太天真了。”
<br/>　　程大雷和刘悲离开牢房，刘悲还劝程大雷：“大当家，你别被他气着了，不用和他生气。”
<br/>　　“他就是一个变态，我和一个变态生什么气。”
<br/>　　程大雷和刘悲来到将军府的大厅，心里琢磨着要怎么处置崔弄海。崔弄海给琴川关带来不好的影响，程大雷的意思是当众处置他，也提一下大家的士气。
<br/>　　只不过处置手段是什么程大雷还没想好，说白了，程大雷并不懂得怎样折磨一个人。
<br/>　　毁灭崔弄海的身体太容易了，程大雷是要毁灭他的灵魂。
<br/>　　想了会儿，程大雷差人将吴用喊了过来。
<br/>　　吴用来的时候有些困惑，道：“大当家，您身体好些了，我还正打算过来看你呢。”
<br/>　　程大雷道：“废话不聊了，你会不会对一个人用刑。”
<br/>　　程大雷手下兄弟很多，不乏心黑手很之辈，但程大雷一眼就看出来：心最黑的就是吴用。
<br/>　　“这太简单了。”吴用已经意识到程大雷说的是谁：“火刑，杖刑，剔指甲，扎针之类的。”
<br/>　　“你说的这些太普通了，我估计不会有什么用。”程大雷道：“如果只是这些，我用叫你来嘛。”
<br/>　　“大当家，我心肠软，那懂那么多酷刑。”吴用呵呵笑着。
<br/>　　然后吴用沉思片刻，就给程大雷说出了十几个法子，包括炮烙，蛇食，磨骨，割肉，千刀万剐，将崔弄海埋进土中，只有头部留出，然后用刀隔开一个小空，因为压力的作用，崔弄海的血会一瞬间冒出……
<br/>　　有些法子，程大雷莫说见，便是听都听得一身寒意。果然，专业的事就该交给专业的人来做，吴用果不愧是自己看中的：当之无愧的蛤蟆寨第一黑手。
<br/>　　“好，这人就交给你了，让他活着，但别让他好好活着。”
<br/>　　正这时，和珅走了进来，道：“大当家，凉州城那边来人了。”
<br/>　　“他们来干什么？”程大雷奇怪道。
<br/>　　“钟伟虎亲自来的，就在外面候着。是听说我们抓住了那恶人，所以特意过来看看。”和珅道：“大当家要不要见他？”
<br/>　　“让他进来吧。”程大雷坐直了身子，心中还是有些奇怪。平日里钟伟虎畏琴川关如同畏虎，今天怎么舍得过来。
<br/>　　“程当家，程当家……”钟伟虎高呼着跑进来：“程大雷果然不愧是豪杰，出手就擒住了那恶人。”
<br/>　　程大雷撇撇嘴：“钟大人有事？”
<br/>　　“那个人还活着么？”钟伟虎试探着问。
<br/>　　“活着。”
<br/>　　“能不能把他交给我？”
<br/>(本章完)
