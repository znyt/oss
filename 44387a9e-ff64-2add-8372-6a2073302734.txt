　　罗昊从一个装满了冰块的玻璃杯中，拿了两块冰块扔进玻璃杯中，冰块跟冰块相撞，冰块和玻璃相碰，发出清脆的声响，随后拿起一瓶标签上写满英文的可口可乐，拧开瓶盖，倒入放了冰块的玻璃杯，随后又在玻璃杯中，放了两片柠檬，对安妮说道：“这就跟去妓院的男人未必都是嫖客一个道理，来酒吧的人未必都是要喝酒。”
<br/>　　“很有趣的比喻。”安妮漂亮的眼睛在罗昊身上停留了一下，随后转身离开了吧台。
<br/>　　罗昊注视着安妮离开的背影，直到安妮完全消失，罗昊才收回了目光，对坐在自己身边的李然问道：“妖瞳，看出点什么了吗？”
<br/>　　李然是观察学高手，他在观察学上有很高的造诣，他不需要跟人进行交谈和接触，通过对方的衣着打扮、行为举止、说话谈吐就能对对方的职业身份做出一个相对准确的判断。
<br/>　　李然拿起加了柠檬片和冰块的可乐，喝了一口，随后轻轻摇了摇头，回答道：“这个女人不简单，她的身上就像是盖上了一层迷雾，我看不透他。”
<br/>　　“在我们身后靠近窗口盆景的位置，那个大胡子还有另外一个白西装的家伙，正在朝我们这边看。”罗昊手中拿着可乐杯，对身边的子午和李然轻声提醒道。
<br/>　　“我从那个大胡子的眼中，看到了四个字——‘别有所图’！”
<br/>　　李然把杯子中剩下的可乐全都喝完，随后把瓶子里剩下的可乐全都倒进了杯子中。
<br/>　　“不要掉以轻心。”子午对罗昊和李然说道：“来这里的人都不是什么善茬儿，抢劫杀人的事情不是没有发生过。”
<br/>　　洛菲克从西装口袋里拿出一根雪茄，正想要点燃，可是想到这里不能抽烟的规矩，又重新放了下来，对站在自己身后的David问道：“David，你对他们怎么看？”
<br/>　　“职业的军人，如果他们从事跟我一样的工作，他们会是我们这个行业中顶尖的杀手，开价也会是最高的。”David向洛菲克解释道：“他们来酒吧没有选择喝酒，是因为酒精会对他们的反应神经造成影响，这影响可能微乎其微，可能只有不到一秒钟的时间，但是在战斗中，这不到一秒钟的时间却能决定生死。他们选择喝可乐，那是因为可乐中含有很高的糖分，可以给人体补充能量，打职业拳赛的拳手在比赛中间的休息时间都会选择喝一瓶被放了气的可乐，来补充身体能量的消耗。”
<br/>　　“那就是说，我们没有机会了？”洛菲克皱眉对David问道。
<br/>　　“他们是专业的，我们同样也是专业的。”David充满自信的回答道：“更何况，我们的人比他们要多。但是我觉得，我们还是有必要搞清楚，坐在中间位置那个人到底是不是GHOST，不然惹上不该招惹的人，会带来无穷无尽的麻烦。”
<br/>　　洛菲克点头，表示自己了解。
<br/>　　罗昊、子午、李然三个人在情报酒吧中待到晚上九点才离开，期间他们也是花钱从安妮这边得到了一些还算有价值的东西。虽然没有从安妮这边得到这次“源代码拍卖会”的拍品信息，但是他们却得知了一部分将会参加这次拍卖会的人员信息。这次“源代码拍卖会”，世界顶级的军火公司都有派人来参加，其中除了洛克希德、通用原子以外，还有比利时的FN，瑞士的西格绍尔，俄罗斯的卡拉尼什科夫集团，德国的HK等等在全球享有很高知名度的军火公司。
<br/>　　根据安妮提供的情报，持有黑色卡片的参拍人员占这次拍卖会总人数的五分之一，持有紫色卡片的参拍人员占总人数的五分之一，剩下的五分之三全都是跟罗昊他们一样，手持红色卡片的自由者。
<br/>　　当然，这些“自由者”中，到底有多少人是真的自由者，而又有多少人是隐瞒了身份藏身在其中，那就有待考究了。
<br/>　　回到居住的旅馆，罗昊跟国内取得了联系，并且从云熙那里得到了一则令他和其他人都震惊的消息。
<br/>　　消息是巫师通过云凌风传回到国内的，因为“小黑龙”的出现，巫师立刻了“龙牙”总部，开始追查毒天使的下落，毒天使最后一次出现的地方是梵蒂冈，巫师推断，毒天使很有可能是奔着这次“源代码拍卖会”去的。跟这条消息一起传来的，还有一张从远处用手机拍摄的，并不算清楚的毒天使侧面照。
<br/>　　“这次‘源代码拍卖会’可算是热闹了。”罗昊把云熙传来的照片全屏放大，说道：“连‘暗月’的毒天使都来了。”
<br/>　　“‘暗月’的人来这里干嘛？”周柏皱眉道。
<br/>　　“反正不是来给我们拜年送礼的。”李然回答道：“大家还是做好准备吧，好戏马上就要开场了。毒天使的出现，或许可以说明一些问题，SHADOW真的跟‘暗月’有关系！”
<br/>　　第二天白天，罗昊、子午、李然他们六个人并没有离开旅馆，而是对晚上就要举行的拍卖会进行讨论。
<br/>　　“这次我们来这里的目的，就是要弄清楚SHADOW的下落。”李然说道：“但是毒天使的意外出现，让事情出现了些许的变故，我们还需要搞清楚‘暗月’的人出现在这个小镇是干什么，并且弄清楚SHADOW跟‘暗月’之间的关系。我们现在缺少情报，这对我们很不利，让我们变得非常被动，所以我们只能根据拍卖会上的情况，然后再见机行事；另外，我们还要注意昨天在酒吧看到那个大胡子和穿白色西装的家伙，我觉得他们似乎在针对着我们。”
<br/>　　“昨天晚上，我跟子午和狂龙讨论过，子午也查到了那个大胡子和白色西装的身份，他们分别是洛克希德和通用原子的人，子午认为他们有可能是认出了他是GHOST的身份，是冲着他手里那些当年从各大军火公司数据库里拿走的武器设计源代码来的，可能是我们潜在的敌人，必须要注意。”
