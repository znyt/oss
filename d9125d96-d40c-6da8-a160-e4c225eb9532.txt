<br/>    紫金山。
<br/>    
<br/>        金毛吼和长河正在对弈，长河落下一子道：“大哥，若是有机会让我杀掉陈明的弟子，一定要给我这个机会，我也让他尝尝，什么叫做丧子之痛！”
<br/>    
<br/>        金毛吼宽慰道：“贤弟莫急，燕山如今猖狂之极，定然有的是机会！”
<br/>    
<br/>        正在这时，金犴忽然走了进来，禀报道：“父王，燕山大宗主卓青瑶正在攻打我紫金山边境。”
<br/>    
<br/>        金毛吼抬头，疑惑道：“你确定是燕山大宗主，陈明那大弟子？”
<br/>    
<br/>        金犴点头道：“是的，可能是因为实力不够的原因，她并没有出手，只是坐镇中军。”
<br/>    
<br/>        长河道:“大哥，这一次机会难得，请让小弟前去，斩了那卓青瑶！”
<br/>    
<br/>        金毛吼微微点头道：“去吧，这一局棋，我等你回来再下！”
<br/>    
<br/>        长河拱手大笑道：“大哥稍待，我去去就回，回来之时，用卓青瑶的脑袋当酒盏！挖了她的心肝给我们下酒！”
<br/>    
<br/>        长河兴冲冲地出去了，金犴问道：“父王，我们要是杀了卓青瑶，那可就得罪死了燕山了。”
<br/>    
<br/>        金毛吼道：“怕什么，打得过燕山，那就没事，打不过，大不了把长河交出去赔罪而已。”
<br/>    
<br/>        一个时辰之后，金犴如同失了神一般走了回来，走进洞窟中，“父王，长河被卓青瑶一剑斩了......”
<br/>    
<br/>        金毛吼突然起身，问道：“怎么回事？长河怎么会被卓青瑶一剑斩了的！”
<br/>    
<br/>        金犴道：“那卓青瑶，挥剑如同泰山压顶，以她斩了长河的情形来看，恐怕和父王交手，也在伯仲之间......”
<br/>    
<br/>        就在这时，洞窟之外忽然响起了一声大喝：“金毛吼老儿，燕山大宗主，卓青瑶，奉师命前来取你首级！”
<br/>    
<br/>        金毛吼怒道：“黄毛丫头一个，也敢来挑衅于我，找死，拿我兵器来！”
<br/>    
<br/>        紫金山之外，山头之上有着十多位妖王正在看戏，他们亲眼看到了长河被卓青瑶一剑斩杀，心头已经是升起了万分恐惧，“这跟传言之中流传的似乎不太一样啊。”
<br/>    
<br/>        “燕山的大宗主，怎么有着这般恐怖的实力？”
<br/>    
<br/>        “这还只是燕山山主门下的大弟子，他的大弟子就有着这般横扫十万大山的实力了，那么燕山山主到底有多强？”
<br/>    
<br/>        “难道说，燕山是真的去攻打，不是为了给大君投诚？”
<br/>    
<br/>        “燕山山主之所以朝着妖城前进，是为了去攻打大君！”
<br/>    
<br/>        正在这时，金毛吼已经提着一柄环首大刀出来了，卓青瑶见到金毛吼出现，笑了笑，“终于是来了一个能让我好好打上一场的对手了，凌仙，你来指挥，我去对付金毛吼！”
<br/>    
<br/>        凌仙从卓青瑶的手中接过令旗，开始指挥血衣剑卫进攻。“怪不得师父让我跟着来，师姐一旦打上瘾了，根本管不了那么多了。”
<br/>    
<br/>        大军之中，刘芒骑着青眸玄牛，提着长枪走出，看了卓青瑶一眼，然后道：“放心吧，我不是来跟你抢对手的，你师父说，你没有坐骑，飞行不便，让我来看着，只要这金毛吼不离地，我就不会出手。”
<br/>    
<br/>        卓青瑶闻言，这才放心，握着满是鲜血巨阙，就是朝着金毛吼冲去，刘芒看了一眼卓青瑶，“这架势，真的还是闻道境？”
<br/>    
<br/>        凌仙一边摇旗指挥一边道：“大师姐修行的功法乃是九九玄功，一旦突破，便是超脱！”
<br/>    
<br/>        刘芒这才明白过来，“原来是这样，能把九九玄功修行到这个地步，你大师姐的天赋可是旷古烁今了。”
<br/>    
<br/>        凌仙道：“还不是师父教的好。”
<br/>    
<br/>        刘芒疑惑道：“你是说，你们师父，也会九九玄功？”
<br/>    
<br/>        凌仙道：“是啊，怎么了？师父对于九九玄功的领悟，比大师姐还要高，都是师父给大师姐讲解的，不过师父似乎是并不喜欢修行九九玄功。”
<br/>    
<br/>        刘芒此时是真的无语了，等于说宗师苦修也在修？比卓青瑶还要高，等于说宗师对于九九玄功的参悟，已经足以突破到大君了？
<br/>    
<br/>        他这已经不是修行路跑偏了，他这是修了假仙。
<br/>    
<br/>        看着场中卓青瑶和金毛吼大打出手，崩碎了周围数十座山峰，刘芒压根没有怎么在意，卓青瑶的实力，他清楚，就是他跟卓青瑶打，也有些犯怂。
<br/>    
<br/>        整个十万大山，已经是处处烽火狼烟，一片狼藉，战争的景象肆虐整个十万大山。
<br/>    
<br/>        本来众多妖王还觉得燕山只是做个样子，结果他们却发现，燕山不仅仅不是在做样子，而且还占了优势！
<br/>    
<br/>        那身披红甲的血衣剑卫，就是妖王也无法逃走，被他们一一斩杀，偶尔有受伤的，只要不是致命伤，都是血衣剑卫摸出一瓶丹药，服下疗伤丹药继续战斗。而且血衣剑卫乃是一个整体，想要杀了其中一人，更是难上加难。
<br/>    
<br/>        然后众多十万大山之中的妖族就发现，血衣剑卫所过之处，所向披靡，无人可挡。
<br/>    
<br/>        因为血衣剑卫的血煞之气为红色，加之血衣剑卫的模样，故而众多的妖族又将血衣剑卫称之为赤潮。
<br/>    
<br/>        赤潮所过，寸草不生。
<br/>    
<br/>        战事进行的并不快，赤潮虽然无可阻挡，但是这些大妖王都是拥兵自重，拥有着数万妖军，于是，在十万大山之中有着更多的消息开始弥漫开来。
<br/>    
<br/>        燕山大宗主斩杀大妖王长河，力战金毛吼，金毛吼败走，燕山大宗主追杀金毛吼，已经跋涉了三千里。
<br/>    
<br/>        而燕山的其他路兵马，所有的统帅几乎都已经展现了大妖王的实力，所有妖族，对于燕山的实力，已经是有了一个新的认识，燕山，有着抗衡整个十万大山的实力，燕山麾下，已经有了五位大妖王战力出现了。
<br/>    
<br/>        而燕山最为神秘而强大的存在，燕山山主陈明，还从未出手。
<br/>    
<br/>        不过大多数的妖族对于陈明的实力，也有了一定的猜测，能够让五位大妖王臣服的，唯有大君。
<br/>    
<br/>        燕山山主，恐怕已经有了大君的实力，达到了超脱境界。
<br/>    
<br/>        而燕山的大军，也开始喊出了燕山的命令，普天之下，凡妖族所属，不臣燕山者，杀无赦！
<br/>    
<br/>        整个十万大山，风起云涌，变幻莫测，前几日还不被看好的燕山，终于是展示了它欲要吞并整个十万大山，一统妖疆的雄心。
