　　司徒青扬处事之缜密，考虑之周到，让慕容飞和佟千雪两个人都有些受宠若惊。当然两个人也知道，司徒青扬会有如此周密的安排，全都是因为司徒空的关系。
<br/>　　司徒青扬既然这么安排了，慕容飞和佟千雪也就顺着他的安排在这里住了下来。慕容飞不知道“追魂”行动的确切启动时间，但是估计时间应该就在这几天，这次行动风险程度很高，出去了很有可能就回不来了，佟千雪安顿在这里好好调养身体也算是了却了慕容飞的一桩心事。
<br/>　　下午的时候，慕容飞拿到了那部iPhone手机，晚上的时候，慕容飞就收到了由一个未知号码发来的短信。
<br/>　　“明天晚上，‘蓝玫瑰’酒吧，A19卡座。从你接收到这条短信起，‘追魂’行动正式启动。”
<br/>　　阅读完短信，慕容飞立刻就把短信给删除了，装作什么事情都没有发生一样走进了屋子。
<br/>　　第二天傍晚的时候，慕容飞找了借口离开了宅院，打了辆车前往“蓝玫瑰”酒吧。
<br/>　　“蓝玫瑰”酒吧在东海市近郊，一个不算太过繁华的地方，而这也就说明了“蓝玫瑰”酒吧并不是一个很有档次的地方，它的老板很有可能还有道上的关系。不过这也符合“追魂”行动，如果是一个正儿八经的地方，反而就显得不够真实。
<br/>　　酒吧门口站着两个穿着黑色西装，腰间挂着甩棍，耳朵里塞着耳麦，负责看场子的男人。
<br/>　　慕容飞双手插在夹克衫口袋中，走进酒吧。刚走进酒吧，口袋中的手机便是震动了两下，又是一条未读短信。
<br/>　　“右边靠窗，牛仔外套。”
<br/>　　慕容飞删掉了短信，收起手机，站在酒吧中环顾了一下，很快就找到了符合短信中描述的男人，而他所坐的位置也是A19卡座。
<br/>　　走向A19卡座，慕容飞拉开男人对面的座椅，坐了下来，拿起他放在桌子上那瓶还没打开的啤酒，用牙齿咬开瓶盖，往自己嘴里灌了一口。
<br/>　　“我是‘司机’。”对方伸出手对慕容飞说道：“你这次行动的配合人。”
<br/>　　“无常。”慕容飞伸手跟对方握了握。
<br/>　　“跟我来。”
<br/>　　“司机”对着慕容飞招呼了一声，站起身朝酒吧后门方向走去。慕容飞站起身跟在对方身后，也朝着酒吧后门方向走去。
<br/>　　酒吧后门外的小巷子里，一对年轻男女正抱在一起卿卿我我，“司机”从内侧口袋中取出一把M92F递给慕容飞，说道：“接过这把枪，行动就正式开始，弹匣里第一发是空包弹，第二、第三发是实弹，后面五发也是空包弹，剩下的全都是实弹，不要记错，不然会出人命的。”
<br/>　　慕容飞深吸一口气，点了点头，当把这把M92F拿在手里的时候，慕容飞明白，他已经不再是那个龙隐部队战术中队的狙击手“无常”了，而是一个“背叛者”。
<br/>　　“司机”对着慕容飞使了个眼色，慕容飞抬手就是对着“司机”腹部开了一枪，枪声响起，“司机”腹部立刻就涌出了大股大股的“鲜血”，“司机”捂着肚子倒在地上。
<br/>　　那对正在拥吻，正想要“开战”的小情侣立刻被吓得缩到了一边，眼神惊恐的看着“开枪杀人”的慕容飞。
<br/>　　砰！砰！
<br/>　　慕容飞又对着那对小情侣旁边的小巷墙壁开了两枪，因为是实弹，墙壁上立刻多了两个弹孔。
<br/>　　慕容飞不是笨蛋，这对小情侣既然会出现在“追魂”行动的现场，十有八九也是云岳安排的人，甚至他们“情侣”的身份都有可能是假的。
<br/>　　对着小情侣开枪进行警告后，慕容飞拔腿就跑，很快就消失在小巷尽头。
<br/>　　等到慕容飞离开后，之前还被吓得瑟瑟发抖的小情侣立刻就恢复了正常，那个年轻男子看着留在墙壁上的弹孔，说道：“真是太帅了，真不愧是无常，这么近的距离都能这么好的控制弹道。”
<br/>　　“废话！”
<br/>　　一分钟前还“中弹”重伤倒地的“司机”，这个时候从地上爬了起来，说道：“野狼一手挑选出来的队员，能差吗？”
<br/>　　“司机”看了眼自己满手的“鲜血”，嘀咕道：“妈的，就算是空包弹这么近的距离挨上一枪，他妈还真疼。”
<br/>　　“我们刚才的表现怎么样？”年轻女子对“司机”问道。
<br/>　　“演得不错，到底是表演系科班出身。”
<br/>　　“司机”对着那对小情侣竖起大拇指，说道：“后面的考核会更加困难，等到真的加入了我们，就算是在特勤中队负责战术支援也随时会有生命危险，你们真的准备好了？”
<br/>　　“准备好了！”
<br/>　　小情侣对着“司机”敬了个礼，神情严肃。
<br/>　　“后面还要继续演下去，别穿帮了，在暗影公布行动细节前，必须要瞒住战术中队。”
<br/>　　“司机”对着那对小情侣叮嘱了一句，舔了舔手上的“鲜血”，说道：“刚才的吻戏很不错，很到位。妈的，疼死我了。”说着就转身离开了小巷。
<br/>　　按照剧情，“司机”是已经“中弹”生命垂危，所以他自然是不能留在这个小巷中，不然等到警察赶到就全都穿帮了，所以他必须离开。
<br/>　　医院那边，云岳也已经打点好，就算是警察去录笔录也绝对不会出现什么偏差。
<br/>　　等到“司机”离开后，小情侣用手机拨打了报警电话，说明了发生在小巷中的“枪击案”。
<br/>　　听到有枪击案发生，这还了得，东海市警局立刻接警出发，赶往现场。因为现场有两个用实弹留下的弹孔，加上目击了整个过程的小情侣作证，一切就变得真实无比。
<br/>　　“立刻向特警请求支援。”重案组组长周志斌对身边两个警员说道：“你们两个跟我去一趟医院。”
<br/>　　慕容飞此时则是已经来到了一栋烂尾楼中，并且在烂尾楼中找到了云岳提前给他准备好的装备——一把SVD狙击步枪！
<br/>　　慕容飞拿起狙击步枪，拉开枪栓看了看，说道：“嚯，还是原装货，暗影就是暗影！”
<br/>(本章完)
