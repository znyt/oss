　　第115章
<br/>　　牛本善去看伙计，发现伙计此刻已经吓得体若筛糠，瑟瑟而抖。他终是见过一些世面，此刻大着胆子又给程大雷上了一壶茶，视线偷看程大雷，却发现他表情不变，对身边发生的事情视若无睹，只专注在面前的茶杯中。
<br/>　　不知是怎样的人，才有这样的胆魄，可不管他是怎样的人，同时得罪金龙镖局和漕帮两大势力，怕都要不得好死。
<br/>　　黄天吉盯着程大雷：“是你将罗银山打伤的？”
<br/>　　程大雷没有回答，恍若未闻。
<br/>　　黄天吉冷哼一声，冲孙德龙道：“孙老爷子，这个人交给我，我要把他绑在旗杆上，暴晒七日。”
<br/>　　“慢。”孙德龙摆手。
<br/>　　“怎么，孙老爷子还是要给这小子出头。”
<br/>　　“出什么头，他只是伤了你一个人，却杀了我漕帮的弟兄，如果我让他死在别人手中，漕帮的面子还往哪里搁。”孙德龙盯着程大雷，眼神像要吃死人肉的老鼠：“李佳明，按漕帮的规矩，这种人该怎么办？”
<br/>　　“三刀六洞，杀了喂狗。”
<br/>　　“不行，这个人必须交给我金龙镖局，我金龙镖局的人不是谁想打伤就能打伤的。”
<br/>　　二人谁也不想把人交给对方，一时争得剑拔弩张。而不管在谁眼里，程大雷都已变成了死人。
<br/>　　“住了吧！”程大雷忽然喝了一声，手摁绷簧，锵啷拔出佩剑：“谁要动手，来！”
<br/>　　“小子，死到临头，还……”孙德龙斜睨程大雷一眼，目光落在剑上，嘴巴突然闭上。
<br/>　　他下意识站起身，目光又在剑上来回打量几遍，眸子里竟浮现一抹恐惧。
<br/>　　“敢问阁下贵姓？”
<br/>　　这下话里对程大雷用上了尊称，所有人都摸不着头脑，程大雷也是一头雾水。
<br/>　　“姓欧。”
<br/>　　“姓欧么……”孙德龙嘴里嘀咕了一句，视线盯着程大雷上下打量一遍，又问：“贵庚？”
<br/>　　“你问那么多做什么，还打不打？”
<br/>　　“不敢，不敢。”
<br/>　　孙德龙慌张摆手，腰微微晚低几公分。
<br/>　　黄天吉有些迷糊：“老爷子，你怎么了？”
<br/>　　面对黄天吉，孙德龙直起腰杆：“黄镖头，我劝你请回吧，欧公子贵足踏贱地，安全问题自然由我漕帮负责，我孙德龙不允许任何人威胁欧公子安全。”
<br/>　　黄天吉一楞：“你说得是什么话？”
<br/>　　“我这可是为你好啊。”孙德龙眨眨眼睛。
<br/>　　黄天吉摸不清里面深浅，这小子如此狂妄，漕帮孙老头什么样的身份，竟然也对他如此态度，莫非他背后藏着什么自己惹不起的势力。
<br/>　　不仅黄天吉不明白，程大雷也糊涂着呐。他举着剑，尴尬的立在哪里，却也不知道发生了什么。
<br/>　　孙德龙回过头来，布满皱纹的脸上堆出笑容：“如果我猜得没错，阁下应该不姓欧吧？”
<br/>　　程大雷一楞，将剑缓缓收鞘，却是没有回答。
<br/>　　他眼中错愕的表情自然瞒不过孙德龙这样的老狐狸，对方笑笑：“放心，阁下的身份我是不会泄露出去的，在城中的大小事务，都有老奴负责。”
<br/>　　我的身份？我什么身份？难道我是蛤蟆寨山贼的事情被这老不死看穿了。
<br/>　　还没等程大雷说什么，孙德龙和黄天吉已经离开客栈。程大雷一伙人面面相觑，都不知其中关窍。
<br/>　　几人回到丁字号院，大家聚在一起，其他人看向程大雷的眼神都充满困惑。
<br/>　　“大当家，莫非你还藏着什么我们不知道的身份？”
<br/>　　“狗屁身份，我现在也糊涂着呢。”
<br/>　　“可他为什么见到你拔剑就走了？”
<br/>　　“你问我我问谁去。”程大雷捧着佩剑，手指划过剑颚上那个小小的『陆』字，下意识想：“难道是这柄剑出了问题。”
<br/>　　剑是李行哉当初留给他的，很锋利，想也知道价值不菲，不过用处不大，还没有程大雷的斧头使得顺手，戴在身上只是作为装饰品的用处大些。
<br/>　　想来想去，想不明白，程大雷挥挥手道：“都散了吧，明天还有事情要忙呢。”
<br/>　　众人都怀揣着困惑离去，程大雷踱步到窗前，轻轻叹了口气。
<br/>　　“你是有心事？”
<br/>　　身后响起声音，程大雷回过头来，见苏樱站在自己身后。
<br/>　　“怎么还不睡？”
<br/>　　苏樱目光仔细打量着程大雷，慢慢道：“总感觉你来了扬州后，和以前不太一样了。”
<br/>　　“怎么不一样，难道是我脸上有灰？”
<br/>　　苏樱摇摇头：“有些狂妄。”
<br/>　　“有些狂妄……”程大雷手扶窗栏，轻叹口气：“时间不多，不得不狂啊。”
<br/>　　……
<br/>　　离开客栈不久，黄天吉便拦住了孙德龙。
<br/>　　“孙老爷子，你把话给我说明白，那小子是你什么人，你要这样护着他？”
<br/>　　“护着他，我是护着你啊。”
<br/>　　“什么意思，我黄天吉需要谁护着？”
<br/>　　孙德龙笑笑：“他……是一个你我都惹不起的人。”
<br/>　　黄天吉更加糊涂，脑海里把程大雷的模样过了一遍，没觉得程大雷有什么出奇之处。
<br/>　　“难道你没看到那柄剑？”
<br/>　　“剑……剑又如何，是把好剑，不过真正行走江湖的好手谁会佩剑，无非是那些大家族的酒囊饭袋好用一柄剑装门面而已。”
<br/>　　“可那是匹夫剑啊。”
<br/>　　“匹夫剑！”黄天吉一惊，结结巴巴问道：“难道他是……”
<br/>　　“唉，不可说，不可说，说破便是天机，而天机不可泄露呐。”
<br/>　　一乘小轿匆匆离开，只剩下黄天吉在那里脸白如纸。
<br/>　　大武明帝就是当今天子，明帝掌国三十六年，说不得有多昏庸，也谈不上有多圣明，普通而已。当皇帝的好处有一，女人是多的，三宫六院七十二嫔妃，虽然是泛指，但其实也差不多。
<br/>　　女人多了孩子就多，究竟明帝有多少子女，怕他自己都搞不清楚。但整个帝国的人都知道，最得明帝心的儿子有十三个，未来的皇位继承人必在他们中间出现。
<br/>　　明帝二十二年，采天山铁，铸剑十四柄，天子剑一，帝国天子执掌，诸侯剑十二，赏给十二位王子。
<br/>　　偏偏有个六皇子，自小不喜读书，偏爱和江湖豪强，贩夫走卒交往。明帝恶其不争，喜其率性，专门为其打了一柄剑。
<br/>　　此剑名曰匹夫剑。
<br/>　　所以剑上的『陆』字，代表的并非姓氏，而是『六』，六皇子的六。
