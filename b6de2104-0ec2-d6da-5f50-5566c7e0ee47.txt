　　元始指了指云中子、赤精子、广成子、黄龙真人、清虚道德真君、太乙真人、灵宝大法师七人，淡淡道：
<br/>　　“尔等去会一会这二十八星宿。”
<br/>　　“是，师尊！”七位真人领命，提剑上前，进入阵中。
<br/>　　柏林提枪归队，只见二十八名道人脚步错落，阵法流转，无边云气涌起，翻腾不休。茫茫云海汇聚而来，形成龙、虎、牛、羊、鼠、马等二十八头异兽，他们在云海之中奔腾，自虚空之中穿越过来。
<br/>　　阐教七位真人脚踏七星罡位，手中宝剑连成一环，真气绵延，与二十八星宿大战在一起。
<br/>　　只见紫气翻腾，白虹飞出，云光荡漾，星辰跳跃，争斗激烈。大战约有一刻钟，阵中世界，茫茫星空中龙吟虎啸，凤唳鹤鸣，声音响彻九幽，苍龙、白虎、朱雀、玄武四象神兽法身现出，身躯皆长达数万丈，火光冲天，咆哮声不断，向着七位真人不断扑击过来。
<br/>　　一时间青雷滚滚，黑雨纷纷，赤焰如海，金芒交织成一张大网，向着四方扑射而来。清虚道德真君高高举起混元幡，迎风招摇，立刻有无数风火从幡中涌出，如亿万风刃利刀，向着虚空胡乱切割，云中子将通天火柱祭出，八根巨大的火柱相互碰撞，声如雷震，烈焰腾飞，带出无数赤色柱影。
<br/>　　阐教七位真人真气相通，生生不息，打得四象神兽痛苦咆哮，残肢乱飞，天上流火四溢，散落下来，变成一片火海。
<br/>　　四象神兽一声声愤怒咆哮，紧接着便向中间扑了过来，如千万雷霆炸响，震耳欲聋，当空便炸出一团巨大的云雾蘑菇。而之后，那云雾蘑菇升腾而起，化为一个巨大的麒麟法相踏空而出，它四翼垂天，头顶有上百支角，浑身青白，口中咆哮声化为紫色雷霆在爆炸。那麒麟法相仰天咆哮，随即低下头来，张口一吐，便是一片雷海冲击过来。
<br/>　　阐教七位真人将法宝护在身前，紧接着腾空而起，那麒麟闷哼一声，前脚仰立，随即将那巨大的头颅扬起，向着七人撞击过来，头上百角锋芒，锐气逼人。
<br/>　　广成子抬手便是一击翻天印，一印翻天，巨大如山岳般的印章一下子便将那麒麟拍得倒飞回去。
<br/>　　七位真人连忙发剑，七道剑气化为数百万道金色剑芒，闪电般的刺出，俱皆射入那麒麟身体之中，那麒麟却浑然不觉，站立起来，仰天长吼，四脚踏空，继续向着七人冲击而来。
<br/>　　太乙真人当即便将九龙离火罩取出，九条火龙翻腾，来回咆哮，向着那麒麟冲去，之后纷纷爆炸，千万碎片纷飞，化为漫天流星，那麒麟腹中如雷霆般爆响，一声巨响，那麒麟庞大的身躯，竟纷纷炸裂，残云漫天飞舞。
<br/>　　二十八为道人纷纷惨叫，各自头顶飞出数道剑气，绕着众人一转，四维七政大阵便荡出了一片血雾，二十八道真灵飞向封神台去了。
<br/>　　“可恶，拿命来！”
<br/>　　万仙阵内，乌云仙见二十八位道兄就如此丧了性命，又悲又怒，他一跃而出，手持混元锤，向着七人打来。
<br/>　　七人刚刚经历过一场大战，气血翻腾，还未调理好，一时间躲不开，一声巨响，赤精子被打倒在地，广成子急忙上前相救，他手中翻天印再次拍出，一印便将那混元锤拍飞，救下了倒地的赤精子，而乌云仙则是去追他那被拍飞的混元锤，云中子等人吃此大亏，又怎么可能罢休，连忙手持法宝向他追去。
<br/>　　忽然西方一道人出现，他脚下云生水起，不过眨眼时间便到了面前，挡住了乌云仙的去路。
<br/>　　乌云仙抬眼看去，只见那道人面黄身瘦，好似长时间营养不良的样子，他头上插着两枝昙花，手持一树枝，但奇怪的是那树枝并没有枯萎或者死亡，依旧绿意葱葱。
<br/>　　乌云仙并不认识这位道人，便问道：“那道人，你是谁，为何拦住我的去路？”
<br/>　　那道人笑道：“道友，我乃西方准提道人，你与我西方有缘，今日特来度化你归我西方，共享极乐，岂不美哉？”
<br/>　　乌云仙闻言大怒：“西方野人，不去你西方那贫瘠之地守着，竟然到此蛊惑于我，你是圣人又如何？”
<br/>　　乌云仙手中利剑当即便向着准提道人砍来，准提不急不缓，面上依旧带着微笑，那剑到了头顶，竟然自己散去。乌云仙大怒，取出其他宝物去攻击准提，准提道人一笑，将手中那树枝向前一甩，那些法宝立刻便消失不见。乌云仙见此心中大感不妙，转身便向着万仙阵跑去。
<br/>　　准提道人取出一根竹子，笑道：“今日贫道便用这六根清净竹，来钓金鳌！”
<br/>　　准提道人将那六根清净竹凭空垂下，便有无限光华产生，裹住那乌云仙，此乃准提的至宝，先天灵根所化，只要六根未净，便不能逃脱。
<br/>　　准提道人大喝一声：“道友，还不速速现出原形！”
<br/>　　只见乌云仙将头一摇，立刻化作一苍老鳌鱼，有十余丈大小，遍体金色鱼鳞，准提道人的童子立刻出现，将手中钓竿一甩，那金色鳌鱼便被捉住，童子上前，扳住双角，骑在背上，那鳌鱼摇了摇头，风云涌动，便向着西方而去。
<br/>　　远处，阐教七位真人见此，却也不敢多说，上前行礼道：“见过准提圣人！”
<br/>　　准提圣人看着七人笑道：
<br/>　　“东方果然人杰地灵！”同时，他心中却想着要不要将这七人也度了过去。
<br/>　　“准提道友，来东方所为何事？”
<br/>　　但就在此时，元始那淡淡的声音自远处传来，让他心中一凛，不敢再多想。他笑着说道：“贫道此来正是为了一会万仙，以见东土道法，随缘超度几个有缘之人，以兴我西方道法，师兄不久便会到来。”
<br/>　　准提的意思便是，我是来见识一下这万仙大阵的，不是来与你阐教结怨的，甚至我们双方还可能合作，而且他提出接引道人，是为了让元始有所忌惮。
<br/>　　“哦？是么？”
<br/>　　元始不置可否道，让准提心中一松，不知为何，面对元始，他心中总有些紧张，但紧接着，他脸上的笑容便消失不见了，只听元始说道：
<br/>　　“此乃我阐、截二教之间的私人恩怨，与你西方何干？你西方又来凑什么热闹？难道是见封神榜上还差几个名额，欲送上几个西方名额补上？”
<br/>　　“呃？道友说笑了。”准提道人面色愁苦道。
<br/>　　“本座可不会说笑！”元始的声音之中，透露着一丝冷漠。
<br/>　　与此同时，万仙阵中，通天教主也知道了西方这位圣人的到来，他眉头微皱，不知在想些什么，不过，还是吩咐众弟子结好阵势，待阐教弟子前来破阵。
