　　缴纳奇物，楚风眠跟白萱便是踏入到混元大陆中。
<br/>　　一入混元大陆，楚风眠便是感觉到，在虚空海中，最为可怕的虚空乱流，如今却是一点都感觉不到。
<br/>　　若是说那星天峰，就像是海上的一艘小船的话，在海中飘摇，这混元大陆，就是一座海岛了，稳固无比。
<br/>　　在这虚空海中，稳定无比，感觉不到一丝一毫来自与外面的威胁。
<br/>　　好像这混元大陆，根本不像是在危险的虚空海中一样，而是北域的一方国度。
<br/>　　其中林立无数商会，无数强者，从中穿行，大部分的武者，都是神力境九重的境界，其中一些人的虽然不如落长恨，但是也跟曾经的白萱，相差无几了。
<br/>　　不过像是落长恨，沧澜霸这种级别的，就要少很多了，一千个人中，都难有一个。
<br/>　　这种人不管是放在那里，都是绝对的天才了。
<br/>　　“虚空海中，有着许多独行者，资质极高，却不愿意加入任何势力，便是一直在虚空海中探索资源，得到奇遇也足以支撑修炼。”
<br/>　　白萱看着一些熟悉的面孔，对着楚风眠出声解释道。
<br/>　　“这些人都是些亡命之徒，无法无天，不好对付，轻易还是不要招惹他们。”
<br/>　　不过说完，白萱才突然反应过来，在他身边的楚风眠，才是真正的一尊无法无天之人。
<br/>　　七大宗门的弟子都敢杀，这才是真正无法无天的人。
<br/>　　谁敢来招惹楚风眠，才是在自己找死。
<br/>　　“在这混元大陆中的商会有着千万，来自于各大势力，甚至是九域中其他几域的大势力，都有在这混元大陆中建立商会。”
<br/>　　白萱指着眼前林立的无数商会开口道。
<br/>　　“所以想要买什么奇物，都可以仔细挑选，不过我们可以先去天商会，虚空五霸的悬赏，便是天商会下的，我们去悬赏。”
<br/>　　“气源楼，万器阁，万宝楼……”
<br/>　　楚风眠一眼望去，便是看见许多熟悉的招牌，在这混元大陆中的不少商会，都是九域都赫赫有名的。
<br/>　　其中一些，更是跟七大宗门都有着千丝万缕的关系，这混元大陆，可以称得上是北域中最为繁华的区域之一了。
<br/>　　不过混元大陆，毕竟可是虚空海中最大的一片聚集地，是所有前来虚空海中历练武者的必经之地，如此之多的强者来往，自然足以造就出如此繁华。
<br/>　　“走，先去天商会吧。”
<br/>　　楚风眠点了点头道。
<br/>　　天商会也是九域中都有分店的大商会，想要购买需要的财富，在天商会中也可以探寻一二。
<br/>　　说着，两人便是走在混元大陆中，混元大陆，被设下禁制，不能飞行，不过以他们二人的实力，缩地成寸，也是轻而易举的事情。
<br/>　　几分钟的时间，白萱跟楚风眠，便是来到了一座巨大无比的商会前。
<br/>　　说是商会，但是如今这一眼望去，更像是一座巨大无比的城池，繁华无比。
<br/>　　在这混元大陆中，一座座商会，便是庞大无比的城池，如同一个个势力宗门一般，不过像是天商会如此巨大的，倒也不是很常见了。
<br/>　　在城池前，一座巨大的牌匾，上面便是写着“天商”二字。
<br/>　　这天商二字中，楚风眠都能察觉出其中的一丝丝玄妙，绝对是一尊几乎要超越生死的强大存在刻画出来的。
<br/>　　无数武者，从这城池前来回出入，大部分人的实力，都有着巅峰圣者之上的实力，不过没有达到巅峰圣者的武者，也并不少见。
<br/>　　“哦？这些没有巅峰圣者实力的武者，是如此来到混元大陆的？”
<br/>　　楚风眠有些诧异的开口道。
<br/>　　混元大陆，位于虚空海中，没有巅峰圣者的实力，根本无法来到这混元大陆。
<br/>　　一些强者，的确是可以帮助人抵挡虚空乱流，将一些小辈带进来，但是这混元大陆中，实力没有达到巅峰圣者实力的武者，却也不在少数。
<br/>　　“这些人，其实都并非是从外面来的，而是本来就生活在混元大陆中的。”
<br/>　　白萱听到楚风眠的疑惑，笑了笑开口道。
<br/>　　“这混元大陆中，本来也就有着一些土著，这些人一直都在混元大陆中生活，不过没有达到巅峰圣者的实力的话，只怕是一辈子都无法离开。”
<br/>　　“不过混元大陆中资源极多，所以在其中生活的武者，许多倒是都有着达到巅峰圣者实力的潜力。”
<br/>　　“走吧，我们先过去看看。”
<br/>　　楚风眠跟白萱，走入到这城池之中去，顿时在城门前的几尊武者，直接迎了过来。
<br/>　　这几尊武者的实力，都是有着神力境三重左右，在外面，就算是在北域中的繁华之地，也不算弱者了。
<br/>　　可是在这混元大陆中，也只能是最为底层的弱者，小厮，为一些势力做事。
<br/>　　毕竟来到混元大陆中的武者，大多数可都是有着巅峰圣者实力的。
<br/>　　一尊看起来仅有二十岁出头样子的女弟子，迎了上来开口道。
<br/>　　“两位阁下，可是来买东西的？我们天商会中，所有宝物，应有尽有，天地奇物，王品灵石都有，不知两位需要什么？”
<br/>　　“先带我们去任务殿，我们完成了天商会的悬赏。”
<br/>　　白萱开口道。
<br/>　　“等我们交完了任务，在去购买一些东西。”
<br/>　　“两位阁下，请随我来。”
<br/>　　那女弟子恭敬的点了点头，便是客气的带着楚风眠跟白萱两人，来到了一处宫殿前。
<br/>　　“李长老，有两位阁下完成了任务殿的任务，前来领取悬赏。”
<br/>　　女弟子恭敬的对着任务殿中喊道，一尊老者，便是从中，缓缓的走了出来。
<br/>　　这尊老者的实力，便是有着生死境高阶左右的地步，不过在生死境高阶中，只能算是稀松平常，跟沧澜逆等人差不多，比起不死魔尊那个等级，还要稍逊许多。
<br/>　　李长老打量了楚风眠二人一眼，开口道。
<br/>　　“两位完成了那个任务？拿出凭证来吧。”
<br/>　　“砰砰砰砰……”
<br/>　　白萱面无表情，打开空戒，顿时五具尸体，便是出现在了地上。
<br/>　　这五具尸体，可都是生死境强者的，千年不朽，如今出现，其中的样貌都是活灵活现。
<br/>　　“这是，虚空五霸？”
<br/>　　李长老打量了一眼这五具尸体，眼睛一下子都有着瞪得豆大，似乎有点无法相信。
