　　北邙圣域。
<br/>　　楚风眠跟寒月漓，两道身影，大摇大摆之间，便是直接回来了北邙圣域之中。
<br/>　　他们二人的遁光，并没有停下，而是直接向着北邙圣域中核心之处的一座巨大无比的山峰飞去。
<br/>　　“他们两人想要做什么？”
<br/>　　“哪里是？圣党的所在，以前夜天君的山峰，圣子峰？”
<br/>　　一道道目光，都看向了楚风眠二人遁光的所在，看着楚风眠飞行的方向，一个个都吃了一惊。
<br/>　　不少人都是急忙追了上去。
<br/>　　在场众人的心中都明白，这次楚风眠跟寒月漓的回归，肯定不会那般平静。
<br/>　　楚风眠跟寒月漓的遁光，一直是来到了一座巨大无比的山峰前，才停了下来。
<br/>　　这一座山峰，比起楚风眠师尊，玄北圣者所在的那一尊山峰，都要庞大，上面建造了无数宫殿，一尊尊武者，圣党弟子，都在其中修炼。
<br/>　　这一座山峰，在北邙圣域之中都是足以排列前三的山峰，圣党以此为老巢，足以看得出当初的夜天君，当初的圣党在这北邙学院之中到底是多么的一手遮天，所向无敌。
<br/>　　不过现在，随着夜天君的陨落，这圣党，也该随着夜天君一起消失了。
<br/>　　“都给我滚！”
<br/>　　楚风眠站在这圣子峰前，猛然冷喝一声。
<br/>　　这一声冷哼的声音，整个北邙圣域，都足以听得清。
<br/>　　在这圣子峰中，其中的无数宫殿，都是走出了不少弟子，甚至是那圣党的几尊核心弟子，都赫然位列其中。
<br/>　　“找死！”
<br/>　　“是谁？敢来找我们圣党的麻烦！”
<br/>　　“不想死的，赶紧滚！”
<br/>　　一声声狂傲的声音，从这圣子峰中直接传了出来。
<br/>　　听到这些话，楚风眠的脸色，冰冷到极点，猛然之间，他大手一挥，直接凝聚远古战龙血脉，化为了一道巨大无比的战龙之爪，从天而降。
<br/>　　“轰隆！”
<br/>　　一声巨大无比的轰鸣声。
<br/>　　这整个圣子峰，都突然摇摇欲坠起来，楚风眠的这一爪，将这圣子峰，都给撼动了。
<br/>　　那些一尊尊圣党弟子，都是愤怒无比，直接便是从这宫殿中，飞了出来，正好是看到了这一幕。
<br/>　　这圣子峰的禁制，在楚风眠的一爪之下，直接打成碎片。
<br/>　　“楚风眠，你想要做什么？这里是北邙圣域，不是你可以耀武扬威的地方！”
<br/>　　在其中最大的一座宫殿之中，一道遁光，突然从中飞出。
<br/>　　这一道遁光，正是这圣党的核心弟子之一，那北域百绝榜第七的狞厉。
<br/>　　在武斗大会上，楚风眠跟这狞厉没有碰面，这倒是让这狞厉活着回来了北邙学院，不然以楚风眠的性格，这狞厉肯定会被他，直接轰杀。
<br/>　　这狞厉看着楚风眠，高声喊道。
<br/>　　“速速退去，你傻了我们圣主，已经是死到临头了，还敢来嚣张，现在学院高层，已经是准备要惩罚你，你还敢来主动惹事？”
<br/>　　“惩罚我？我如今夺得这这一届武斗大会的榜首，有着大功，学院怎么可能惩罚我，倒是你们圣党弟子，在北邙学院中嚣张跋扈，惹下不少麻烦，现在夜天君死了，也没有人可以保得住你们的！”
<br/>　　楚风眠高高在上，俯视着下方，如今一尊真正的神明一般。
<br/>　　“都给我滚，三秒之内，谁不滚，谁就得死！”
<br/>　　“狂妄！”
<br/>　　狞厉听到楚风眠的话，也是脸色猛然一边，他的手中，一把长刀突然出现，直接一刀，已经是向着楚风眠斩了过来。
<br/>　　“蝼蚁罢了，也敢出手？”
<br/>　　楚风眠脸色平静，看着狞厉的刀光，他的手缓缓的抬起来，直接捏了过去。
<br/>　　只见这一把刀锋，被楚风眠直接捏在了手中，这狞厉的刀，甚至是就连楚风眠的皮肤，都无法划破，随着楚风眠稍稍用力，狞厉手中的刀，都是直接破碎。
<br/>　　“这里是北邙学院，我不杀你，但是你得罪了我，就以你身上的一半精血，来作为补偿吧。”
<br/>　　楚风眠手心一拍，这狞厉的身躯直接被拍飞出去，同时他的胸口，直接打出了一个伤口，无数精血，随着这伤口，直接全部被楚风眠抽取走，被楚风眠融入到了剑阵之中去。
<br/>　　这杯抽取了大半精血的狞厉，便是直接失去力气，从天上掉了下去。
<br/>　　“好狠的手段！”
<br/>　　“这回到学院中，就要来报复圣党了！”
<br/>　　一尊尊北邙学院弟子，看着楚风眠的手段，都是极为心惊。
<br/>　　狞厉在北域百绝榜上，都足以排进第七名的位置，在北邙圣域中便是一尊绝世天才。
<br/>　　可如今被楚风眠这样挥手之间，便是击败。
<br/>　　抽取了一半的精血，这狞厉以后若是没有惊天奇遇，想在恢复如今的实力，都没有可能，将来更是不可能在突破，可以说已经是成为了一个废人了。
<br/>　　看着狞厉被轻而易举的打飞，那圣子峰上的一尊尊圣党弟子，都是脸色苍白，纷纷化为一道道遁光，飞了出去。
<br/>　　他们可不想落得跟这狞厉一样的下场。
<br/>　　一道道遁光，纷纷离开，整个圣子峰，都是人去楼空。
<br/>　　曾经在北邙学院之中最为繁华的圣子峰，现在便是落得了这样一幅样子。
<br/>　　不少北邙学院弟子心中都清楚一点。
<br/>　　圣党，完了。
<br/>　　曾经在北邙学院称王称霸的圣党，如今在楚风眠的手中，彻底的完了。
<br/>　　“算他们识相。”
<br/>　　看着这圣子峰，已经是空无一人，所有的圣党弟子，纷纷逃走，楚风眠冷笑一声。
<br/>　　这些圣党弟子，楚风眠还没有兴趣去赶尽杀绝，一些小人物罢了，根本就引不起楚风眠的兴趣。
<br/>　　楚风眠身形一动，化为一道遁光，跟寒月漓一起，直接便是进入到了这圣子峰之中。
<br/>　　这圣子峰的灵气，在北邙圣域之中都足以算是最为顶级的了，这里正好是楚风眠现在最好的洞府所在。
<br/>　　“从今日起，这圣子峰便是改名为悟剑峰，是我楚风眠的洞府，任何人不得靠近！”
<br/>　　楚风眠进入到圣子峰中，顿时打出无数禁制，令这圣子峰，焕然一新。
<br/>　　不，这已经不再是圣子峰了，而是属于楚风眠的悟剑峰。
