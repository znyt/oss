<br/>    陈明从沙海之中露出一个脑袋，却是完全看不清妖月的身影，即便是自己散开感知，也无法穿过这飓风，瞬间便消失无踪。
<br/>    
<br/>        陈明看了周围一眼，早跟你说了，穷寇莫追，偏不信。
<br/>    
<br/>        他曾经在这沙海中试过，向上飞，根本就如同没有尽头一般，甚至于，自己连上下都分不清，所以陈明一直都控制着自己不要离开地面的沙海太远。
<br/>    
<br/>        若是能够有一枚定风珠，或是一座能够驱散抵抗这飓风的大阵，将燕山搬到这里来，那就真的是管他外面打个天翻地覆，十万大山怎么打，只要尸九泉不出来，那就和燕山没什么关系。
<br/>    
<br/>        可以的是，现在的陈明，还没有这个手段。
<br/>    
<br/>        掏出储物戒中指西针，陈明顺着一个方向前行，在一天之后，陈明终于是从沙海之中出来了，回头看了一眼沙海，风沙如同便是整个世界，就让妖月在里面自己玩吧。
<br/>    
<br/>        陈明现在必须尽快回到燕山，只要他回到燕山，而妖月没有回去，那就够了，历史总是由活下来的那个人书写，妖月到底是怎么死的，还不是他说了算？
<br/>    
<br/>        不仅仅是妖月王庭附近，就是整个十万大山，都在流传妖月追杀陈明的事情，“你听说了吗？有大妖王出手了，据传是追杀什么燕山之主。”
<br/>    
<br/>        “燕山，燕山是什么地方？”
<br/>    
<br/>        “没听说过！”
<br/>    
<br/>        “据传闻，燕山想要和妖月王庭分庭抗礼，妖月大人何等强大，直接击毙了燕山之主！”
<br/>    
<br/>        “这个什么燕山之主，就是以跳梁小丑而已，什么东西，也敢挑衅大妖王！”
<br/>    
<br/>        妖月王庭境内，神府境妖王，道种境妖王现在对于燕山，可谓是相当的看重，虽然陈明不敌妖月大人，但是陈明才崛起多久，声名鹊起也不过一年的时间，居然可以和妖月王庭分庭抗礼，燕山，必定是一块宝地。
<br/>    
<br/>        “燕山，可谓是钟天地之造化，日月之精华，也唯有如此宝地，才能养出陈明此等大能！”
<br/>    
<br/>        “不错，妖月王庭不日便要攻打燕山，我等必须早早的出发，才能分到一杯羹！”
<br/>    
<br/>        “燕山，乃是王起之地！”
<br/>    
<br/>        燕山附近，瞬间风声鹤唳，乱作一团，众多妖王聚兵，想要瓜分燕山的财宝。
<br/>    
<br/>        燕山之中，银翼等了几天，最后也是察觉到了燕山周边的变化，燕山，已经不能继续待下去了，燕山，实在是太危险了！
<br/>    
<br/>        银翼找到四位亲传弟子，已经三千六百记名弟子，站在高台上，银翼道：“现在的燕山，实在是太危险了，我们不能继续待在这里了。”
<br/>    
<br/>        这些记名弟子还不知道发生了什么事，即便是亲传弟子，对于燕山的战事，都不是很清楚，银翼解释道：“在数日之前，妖月王庭的大妖王妖月，齐聚妖月王庭所有妖王，准备商议如何针对燕山之事。你们的师父自然不会让他们如意，为了你们的安全，他孤身前往，想要阻止他们对付燕山，却不想那妖月蛮不讲理，不分青红皂白就要杀了你们师父。至于现在，你们师父的坟头，怕是草都有两寸高了。”
<br/>    
<br/>        卓青瑶的双眼瞬间一红，鞘中玉女剑瞬间出鞘，直指银翼，“这不可能！”
<br/>    
<br/>        凌仙也是道：“师父不可能死的！”
<br/>    
<br/>        李素衣更是毫不犹豫，“你在欺骗我们，师父根本不可能会死在妖月的手里！”
<br/>    
<br/>        陈灵鱼对于陈明的实力并没有一个清楚的认知，但是大师姐和两位师兄都这么说了，“大师姐和两位师兄说的对啊！”
<br/>    
<br/>        众多记名弟子见四位亲传弟子不信，也是表示不信，银翼道：“目前燕山的情况你们应该看到了。至少有着上百位妖王，已经在进军燕山了，我们现在不走，就没有走的机会了！”
<br/>    
<br/>        卓青瑶将玉女剑插在地面上，盘膝坐在地面上，“我不走，势与燕山共存亡！”
<br/>    
<br/>        见到威严如山的大师姐这么表态，其他弟子也是有样学样，“我们不走，势与燕山共存亡！”
<br/>    
<br/>        银翼无奈了，毕竟是陈明的徒弟，不可能因为他两句话就放弃燕山，“你们走不走算了，反正我是决定要散伙了，散伙的时候大家分一分家当啊，我就带个一两千瓶丹药就满足了，愿意跟我走的，我会尽全力带你们走的，我已经仁至义尽了！”
<br/>    
<br/>        银翼忽然发现身前的众多弟子脸色一变，齐齐跪倒道：“见过师父！”
<br/>    
<br/>        银翼愣了愣，随意摆摆手，“别开玩笑了，你们师父坟头草都两寸高了，该散伙就散伙，吓我没用的！”
<br/>    
<br/>        这个时候，银翼的背后，传来一句淡淡的声音，“都起来吧。”
<br/>    
<br/>        银翼瞬间脸就白了，这绝对是陈明的声音，银翼缓缓回头，可以看得到浑身是血的陈明正在目光灼灼地看着他，浑身鲜血，如同刚从地狱之中爬出来，“银翼，刚刚就是你说的要散伙是吧？”
<br/>    
<br/>        银翼飞速摇头，“不是，我没有。我不是这个意思！刚刚，我是为了考验山主门下弟子的忠心，现在看来，他们对于山主，还是极为敬重的，很好，很好啊！我很是欣慰。”
<br/>    
<br/>        陈明微微点头，道：“很好，既然你很是欣慰，想必你对燕山也是极为忠心的，现在，燕山之外有着那么多的妖王，你应该明白吧？”
<br/>    
<br/>        银翼此时怎么可能不明白，“属下，这就去杀敌！”
<br/>    
<br/>        道完，银翼直接飞走了，一边飞还一边在想，到底要杀多少个，山主才会放过自己？
<br/>    
<br/>        而且，山主回来了，但是妖月却是没有回来的消息，难不成，妖月，已经死在了山主的手里不成？
<br/>    
<br/>        浑身鲜血的陈明，看向身前众多的弟子，笑了笑，“你们还没事就好。”
<br/>    
<br/>        众多弟子看向陈明，浑身浴血，可见这一战有多艰辛，而这一刻，他们也是真正明白了，一日为师终身为父到底是个什么概念。
<br/>    
<br/>        陈明朝着弟子们微微挥手，然后道：“别怕，只要师父还在，就没有任何人可以伤害你们，不过最近一段时间，不要外出了，就待在燕山吧。”
<br/>    
<br/>        陈明并没有说太多，转身朝着燕山走去，卓青瑶第一个上前，直接紧紧抱住陈明的胳膊，生怕眼前的陈明并不是真的，“师父，我扶着你。”
<br/>    
<br/>        剩下三个亲传弟子也是跟上，卓青瑶安排道：“凌仙，去打水来。”
<br/>    
<br/>        “李素衣，你去找疗伤丹药来。”
<br/>    
<br/>        “陈灵鱼，速速去准备饭菜。”
