　　航母上的餐厅窗户正对着航母甲板，早晨的阳光通过餐厅窗户洒进餐厅中，给整个餐厅镀上了一层金色的光辉。
<br/>　　罗昊坐在靠近窗户的那张餐桌上，眼睛看着窗户外航母甲板。甲板上，两架F/A-18E“超级大黄蜂”战斗机在地勤人员的引导下，进入弹射起飞的位置。
<br/>　　这两架“超级大黄蜂”是对影子部队在中东的战略目标进行精确打击的，自从Parker交代了他所知道的有关于影子部队在中东的部署后，航空母舰上的战斗机就不断地起飞降落，对一些重点战略目标进行打击摧毁。
<br/>　　西伦科特在中东经营多年，影子部队在这里不仅储备足以应付中等规模局部战争的军火，甚至还部署了防空导弹基地，挖掘了地下防空通道，坦克部队，以及可以供战斗机起飞的机场！
<br/>　　而这些东西现在都成了美国海军航空部队的重点打击目标，不仅是美国，北约的一些盟国也是加入了这场定点清除的行动中。
<br/>　　罗昊怔怔地看着窗户外的航母甲板，就连云熙走进餐桌都是没有发现。云熙知道罗昊在想什么，他在想被救回来的司徒空、刘子锦、林晓阳、陈应天、张子航五个人的伤势，他在想其他队员的情况，在考虑应该怎么告诉张子航他弟弟牺牲的事情。
<br/>　　这就是罗昊和蛰龙最大的区别，这也是为什么罗昊只能成为战术中队队长，而无法担任战略行动组组长的原因。
<br/>　　“别想太多。”云熙把手里的盘子放在餐桌上，说道：“先吃东西。”
<br/>　　云熙转身去倒了一杯温水和一杯牛奶，放在罗昊面前说道：“先喝点水，然后再吃。”
<br/>　　罗昊拿起一个花式馒头，看了眼坐在自己对面的云熙，问道：“你吃了没？”
<br/>　　“你先吃。”云熙微笑着说道：“你吃完我再吃。”
<br/>　　“我吃完你就不吃了。”罗昊把馒头放在云熙面前，说道：“我还不了解你？你如果不吃，信不信我来喂你。”
<br/>　　罗昊把“喂”这个字咬得很重，令得云熙俏脸绯红。云熙当然知道罗昊所说的“喂”是什么意思。
<br/>　　一次云熙生病了，发着低烧，躺在床上发着小孩子脾气，说什么也不肯吃药，最后是罗昊把药含在嘴里，用嘴对嘴的方式给云熙喂药，才让她把药给吃了，当然这喂药喂到最后演变成了什么，也只有罗昊和云熙两个人知道了，反正第二天云熙的感冒好了很多，但罗昊却成了病号。
<br/>　　云熙娇羞着瞪了罗昊一眼，餐桌下用脚用力的踩着他的脚背，“亏我还帮你做早餐，气死我了！”
<br/>　　当然，最后云熙还是把罗昊放在自己面前的馒头给吃了，因为云熙知道，自己如果不吃，罗昊是真有可能来“喂”自己的。而且如果罗昊真的这么做，餐厅中这些美国大兵非但不会阻止，反而还会起哄，这些美国大兵是不嫌事儿大的。
<br/>　　因为担心还会有人给西伦科特通风报信，走漏消息，所以这次詹姆斯启用了美国海军陆战队的情报系统，由海军陆战队情报室对Parker所交代的那几个西伦科特秘密设置的安全屋进行侦察和情报收集，而且这次担任情报的人员全都是詹姆斯亲自挑选出来的，可靠性上绝对不存在任何问题。
<br/>　　海军陆战队情报室的工作效率很高，再次体现出了一个超级军事大情报系统的强大。
<br/>　　联合行动司令部，参加联合反击行动的多国军事代表再次聚集在房间中那张巨大的三维电子沙盘前，除了美国方面汉密尔顿被太平洋舰队司令詹姆斯替换外，其他国家的军事代表还是原来那些人，只不过这一次不同的是，他们对中国这边的宋萱、李然、罗昊、云熙的态度比起之前要好的多，就连最开始反对中国派出代表参与到联合反击行动情报收集和作战计划制定的意大利、法兰西国、荷兰这些国家的军事代表看到中方这边四个人的时候，都是主动打招呼示好。
<br/>　　詹姆斯在安排座位的时候，更是将中方代表的座位安排在了他的左手边，而不是像汉密尔顿那样把他们的座位安排在角落里，这些细节上的变化，都说明了联合行动司令部对中方态度的改变。
<br/>　　他们是强者，强者就应该得到强者应该有的尊重！
<br/>　　“先生们，这次联合反击行动可以说是失败的，在这次行动中，我们每个国家都有巨大的损失。”詹姆斯抬起头缓缓扫过沙盘周围各国军事代表，最后把视线落在李然的身上，说道：“我为汉密尔顿向你表示道歉，如果当时他能听取你的意见，或许我们每个国家都不会蒙受现在这样的损失。”
<br/>　　坐在李然和罗昊身后的云熙跟宋萱对视一眼，为詹姆斯的说话技巧竖起大拇指，比起汉密尔顿那种表面奉承，但实际轻视的态度，詹姆斯的确要更会说话，也更会为人处世，这也难怪詹姆斯能成为美国太平洋舰队的司令，而汉密尔顿却不行。
<br/>　　“MCIA根据‘八爪鱼’所交代的情况，对西伦科特所设立的几个秘密安全屋进行了调查，并且进行情报收集，我们得到了一些有价值的东西。”
<br/>　　詹姆斯手指滑动，将一张在远距离用单反相机拍摄到的照片，在三维电子沙盘上放大，说道：“现在我们可以肯定，西伦科特这个家伙逃到了南美洲！”
<br/>　　所有人把视线注意到照片上，因为是在远距离拍摄的，所以照片并不是特别清晰，照片上只能看到一个穿着迷彩服的男人从一栋楼房中出来，走向一辆停在路边的越野吉普，但是只是通过照片不能看清楚照片上的男人就是西伦科特。
<br/>　　詹姆斯把照片上的男人放大，说道：“我们技术部门进行了面部骨骼对比，可以确认这个人就是我们要找的西伦科特无疑，他确确实实从我们的眼皮子底下溜掉了，逃到了南美洲躲藏起来！”
