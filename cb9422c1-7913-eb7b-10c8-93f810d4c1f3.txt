　　影天使用垃圾话试图对子午造成影响，子午也用同样的方法回敬给影天使。
<br/>　　只是到了子午和影天使这样的级别，垃圾话能对他们所产生的影响，实在太低太低了。
<br/>　　听到子午的回复，影天使也只是微微一笑，并没有放在心上。
<br/>　　子午和影天使之间的对话很短，可是坐在七人座MPV中的李然还是抓住了其中的关键。
<br/>　　刚才的对话中，影天使对从这栋写字楼中撤离显得很有信心，似乎是早就有了撤退方案。
<br/>　　这栋写字楼楼顶是直升机停机坪，可是想要利用这种办法撤离显然显然是不可能做到的，如果他们真的胆子大到敢搭乘直升机撤离，李然可以保证，在他们直升机刚从写字楼楼顶停机坪起飞的时候，就会被防空导弹打下来，以智天使的精明是绝对不会用这么愚蠢的办法来撤离的。
<br/>　　地面撤离？也不现实，因为写字楼周围的道路全都已经被封锁，别说智天使现在没有坦克，就算是他有坦克，他也跑不出去，唯一的办法就是用他布置在K国还没有被发现的炸药进行威胁。可是智天使真的会用这种办法来逃脱吗？他真的只是准备这一种撤离方案？
<br/>　　李然对此深感怀疑。
<br/>　　地下防空通道！？
<br/>　　李然突然想到了一个可能。这座城市在第二次世界大战中也曾经是反法西斯主战场之一，也曾在战火中遭到严重破坏，所以这座城市一定是建有地下防空设施的，而一座城市的地下防空设施一般来说都非常复杂，就跟一座地下迷宫一样，一旦进入到其中，基本不可能再找到。
<br/>　　“地下防空通道！”
<br/>　　李然通过无线电跟位于指挥中心的赵天宇进行联系，说道：“智天使会利用地下防空通道脱身！立刻跟K国政府进行联系，拿到这座城市的地下防空设施图纸，即便是过去了几十年，但是我相信K国政府一定还保存着。”
<br/>　　“我立刻跟K国政府联系。”赵天宇回答道。
<br/>　　赵天宇的办事效率很高，不到二十分钟的时间，李然就拿到了这座城市地下防空设施图纸的照片，通过照片，可以发现这些图纸已经保存了很长时间，但却依然清晰。
<br/>　　城市地下防空设施的图纸很复杂，李然一张一张翻看着照片，李然翻看照片的速度很快，而且没有返回重复看。尽管李然表面上显得很平静，可是他内心很着急，他不断的告诉自己，还有时间，还有时间进行布置。
<br/>　　罗昊看了眼赵天宇传来的照片，沉吟道：“这图纸是几十年前的，还会有用吗？”
<br/>　　“一般来说，一座城市的地下防空设施不会轻易变动，即便是被用作他途也只是在原有基础上进行改造。”李然一边翻看着照片，一边说道：“出口和入口的位置不会做出改变，所以这些图纸还是有价值的。”
<br/>　　李然翻看照片的同时，将图纸上地下防空设施的出口和入口记忆下来。
<br/>　　地下防空设施内部非常复杂，但是只要守住出入口，就能破坏智天使的撤离计划。李然不清楚是不是这样就能抓住智天使，但是他总需要去尝试一下。
<br/>　　李然的目光停留在一张照片上，说道：“这里！”
<br/>　　“什么‘这里’？”罗昊不解问道。
<br/>　　“如果我是智天使，我就会选择这个地方作为我撤退的路线。”李然指着照片说道：“从这里出去再往东走不到十公里就是渔港，如果提前在那里准备了船只和车辆，就是一条理想的撤离路线。”
<br/>　　“我联系师傅，让K国政府进行布置。”罗昊说道。
<br/>　　“K国快速反应部队拦不住他们。”李然摇了摇头，回答道：“你留在这里，我带战备一队一组人过去。”
<br/>　　“小心。”罗昊嘱咐道。
<br/>　　李然点了点头，立刻带着战备一队的一组队员前方他所说的那个地下防空通道出口。
<br/>　　……
<br/>　　写字楼内，吴云自然是看到了李然带人离开，他通过无线电跟智天使取得了联系，“智天使，他们有人过去了。”
<br/>　　智天使对于这个情况显然是不感到意外，嘴角微微上扬，说道：“果然还是猜到了，如果猜不到那就太让我失望了，毕竟他是一个难得能让我重视起来的对手。可惜啊，他这次注定要失望了，他只是猜到了一部分。”
<br/>　　智天使松开耳麦，看向桌子上的显示器，通过安装在两个房间里的摄像头，智天使可以看到这次影天使和子午之间对决的全过程，而这样的现场直播是全世界各国黑客都梦寐以求的。智天使看了眼时间，距离完成门锁程序设置还有最后十分钟。
<br/>　　通过曲线图，智天使发现两个人APM曲线基本趋于一致，从最开始最高超过500，接近600的APM，逐步下降到100多接近200，然后一直维持在这个水平，没有很大的波动。他们两个人全都保持着自己的手速，没有让手速失控。
<br/>　　一小时倒计时结束，子午和影天使键盘的背光灯同时熄灭，两个人的键盘同时被锁死无法再继续使用。
<br/>　　“五分钟时间休息。”影天使说道：“五分钟后，我们进入生死搏杀。赢的人可以逃出去，输的人就留在这里灰飞烟灭。”
<br/>　　“听说你在EG工作了几年时间。”子午看着墙壁上显示屏中的影天使说道。
<br/>　　“没错。”影天使回答道：“在那里我学到了很多东西，今天你所面对的SHADOW，再也不是以前那个SHADOW，我会把你彻底击败！”
<br/>　　“我不是这个意思。”子午摇晃着手指，说道：“我的意思是，你在办公室里待了那么长的时间，是不是缺乏锻炼，别一会儿来不及跑，就算是赢了都跑不出去还被炸死在这里，那就真的太丢脸了。”
<br/>　　影天使冷哼一声，说道：“还是先管好你自己吧，别到时候根本就没有机会跑！”
<br/>　　子午微笑没有说话，而是拿出一盒烟，抽出一根塞到嘴里点燃，慢慢的抽着……
