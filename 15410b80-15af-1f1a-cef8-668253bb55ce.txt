<br/>				缓步而来的美妇人，一袭月牙白的衣裙着身，素雅而端庄，那盘起的发上并没有过多的饰品，只斜插着一根碧绿钗子，腰间流苏垂落，随着她缓步的走动而轻轻摆动着，煞是好看。<br/>　　<br/>　　    越看着这美妇人，她越觉得奇怪，一个无法修炼的女子，遇到危险时也会惊慌娇弱的躲在男人怀里的女子，为何有时展现出来的气质那般神秘？<br/>　　<br/>　　    不错，就是神秘，至今，她都看不透这美妇人，总觉得，她不应该如此简单。<br/>　　<br/>　　    “上马车吧！”美妇人对一双儿女说着。<br/>　　<br/>　　    谢玉堂两人先上了马车，而后凤九便扶着美妇人上了马车，待他们都坐好后，这才在外面坐了下来，驾着马车往阮家而去。<br/>　　<br/>　　    阮家离谢家也不是很远，再加上昨天凤九探过路了，路况熟悉了，很快便也到了阮家大门那里，只是，意外的是前面的马车已经有七八辆，将路都堵了起来。<br/>　　<br/>　　    “夫人，前面过不去了，马车将前面路堵住了，是否要在这里下车？”她回头询问着，从这里再往前走去也就十几米左右，将马车停到一旁倒也方便。<br/>　　<br/>　　    “也行，就这里下吧！”美妇人先出来，下了马车后对凤九道：“你把马车停附近就好，交待一下阮家的门房，让他们看着点就行了，等会诗思跟着我，你就跟着玉堂吧！”<br/>　　<br/>　　    闻言，她点了点头，应了一声：“是。”<br/>　　<br/>　　    她将马车停好后，交待了门房一声，便跟着他们一起往里面走去，如同小厮般的跟在谢玉堂的身后，抬头往前看去，见走在前面的美妇人与其他马车的一位妇人聊了起来，一路有说有笑。<br/>　　<br/>　　    她收回目光，朝周围打量了一眼，见这阮家里面也甚是气派，跟谢家主家能有一比，就这往后院而去的小道周围，暗处都有数名暗卫在守着。<br/>　　<br/>　　    院家主母宴客的地方是在后院，少年和少女所在的地方是分隔开的。到了后院院门那里，凤九因是小厮随从的身份所以不得进内院，于时，美妇人便让凤九在院门那里等一会。<br/>　　<br/>　　    她带了一双儿女进去后不久，见过阮家主母后，谢玉堂便先出来。<br/>　　<br/>　　    “走吧！据说城中不少家族的子弟都来了，去看看。”谢玉堂来到凤九的身边说着，便迈步跟着一名带路的婢女往另一边而去。<br/>　　<br/>　　    凤九跟了过去，来到另一处院落，进了里面就见已经有十几名十几到二十岁上下的男子在里面坐着，而在每个人的身后，都跟有一名小厮。<br/>　　<br/>　　    谢玉堂的到来，让那些人都回过头来打量着，作为主人家的阮家公子见又有客人来了，便笑着上前：“我是阮家大公子，不知这位是……”<br/>　　<br/>　　    “我是谢家的，我叫谢玉堂。”谢玉堂打量着眼前之人，见眼前之人二十上下，剑眉星目容颜出色，身形修长气宇不凡，不由多看了两眼。<br/>　　<br/>　　    院中众少年听说他是谢家的，便以为他是谢家嫡系子弟，于是，便都招呼着他上前。<br/>
