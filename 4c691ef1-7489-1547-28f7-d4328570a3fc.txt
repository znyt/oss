<br/>    “对了，王婆还说，下一次去镇子上，给我带一匹好布，做新衣服，女儿想了，让王婆多带两匹布，我给爹爹也做两身衣服。”
<br/>    
<br/>        步留痕很是欣慰，“好好，只要你开心，什么都好，这里与世无争，确实是个好地方。”

<br/>        步留痕忽然露出一抹恭敬之色，步生莲问道：“爹爹，怎么了？”

<br/>        步留痕认真道：“无面者阁下在呼唤我了。”
<br/>    
<br/>        步生莲道：“无面者阁下对我们有大恩，我们可不能怠慢，爹爹快去吧。”
<br/>    
<br/>        步留痕道：“我这就去，带我去你的房间。”
<br/>    
<br/>        来到步生莲的房间，步留痕在房间中央席地而坐，双膝盘坐，“尊敬的无面者阁下，我听从你的召唤。”
<br/>    
<br/>        意识一变，步留痕已经是来到了大殿之中，大殿的上首，陈明端坐其中，步留痕恭敬地朝着陈明拱手道：“无面者阁下。”
<br/>    
<br/>        陈明呵呵一笑，微微点头，“你是我的刀，既然是刀，就会有出鞘的时刻，现在，该是出鞘的时候了。”
<br/>    
<br/>        步留痕当即起身道：“我早就已经准备好了！请无面者阁下吩咐！”
<br/>    
<br/>        陈明微微点头道：“在武王星，大荒山脉，三百三十三丈断崖处，其中有个洞口，进入洞口，来到最底部，有一块大石，上面有着我的记号，大石之下，有一柄弓，你去将那柄弓带走。”

<br/>        步留痕拱手道：“属下记住了！”
<br/>    
<br/>        陈明嘿嘿一笑，继续道：“哦，对了，拿到那一柄弓之后，不要试图开弓，或者是任何冒犯那柄弓的行为，会死的。”
<br/>    
<br/>        步留痕心中一震，这是英魂兵器啊！
<br/>    
<br/>        步留痕道：“我会郑重的请这柄弓的。”
<br/>    
<br/>        陈明继续道：“你应该知道孽龙星河的鬼门在哪儿吧？”
<br/>    
<br/>        步留痕拱手道：“鬼门就在孽龙星河，天庭和瑶池都没有掌控，也是因为这个原因，天庭和瑶池的摩擦才没有那么严重，鬼门的位置，属下早就已经知道了。鬼门，乃是收纳灵魂的位置，每百年一开，只开七天，这也是天庭和瑶池争斗最为凶猛的七天。”
<br/>    
<br/>        陈明点头，然后道：“带着你的人，将瑶池驻扎鬼门附近军队，暗哨，全部斩除！”
<br/>    
<br/>        步留痕心中一震，无面者阁下这是要动手了，他也想要参加这一次的鬼门之争。有着惊讶，但是同样有着狂喜，在星空之中，想要得到变强的机会，只是修行有啥用，修个一辈子，到死也就是天仙了，想要继续变强，唯有争！
<br/>    
<br/>        如何争？
<br/>    
<br/>        便是跟随这些大势力去争，立了大功，便有赏赐，大势力的赏赐，才是晋升的机缘！
<br/>    
<br/>        他步留痕想要晋升到金仙，唯有在无面者的指挥下去和天庭和瑶池争斗，才能得到更大的机缘。
<br/>    
<br/>        步留痕道：“冒充天庭的身份去吗？”
<br/>    
<br/>        步留痕想的很简单，无面者的规矩，无面者是很少，极少会出手的，必定是他去争，但是他如今的实力，和天庭瑶池对拼？
<br/>    
<br/>        那不是找死？
<br/>    
<br/>        所以最好的办法就是让瑶池觉得是天庭想要动手，挑起天庭和瑶池的争斗，水浑了，才好浑水摸鱼。
<br/>    
<br/>        陈明呵呵一笑，还是有点嫩啊，“你冒充天庭的身份，只会弄巧成拙，暴露了我们，距离鬼门打开，还有差不多十年的时间，天庭现在可不会动手。提前动手，只会引起更大的惊变，引起更多的损伤。至少明面上不会，所以，只能以暗杀的形式进行。”
<br/>    
<br/>        “天庭和瑶池并不知道我的存在，瑶池驻扎鬼门的人没了，天庭的人却还在，瑶池和天庭的视角之中，只有对方，不用天庭的身份去，才会让瑶池觉得就是天庭做的。”
<br/>    
<br/>        步留痕这才明白过来，拱手道：“无面者阁下真是神机妙算，这样一来，天庭和瑶池争斗，我们就有一丝机会浑水摸鱼了。”
<br/>    
<br/>        陈明哈哈大笑道：“不，这样还是没有机会的，还不够，有另外一个无面者的执刀人，也会来此。”
<br/>    
<br/>        说实话，步留痕并不是很怕天庭和瑶池，而另外一个无面者，却是让他怕了，无面者在星空的名声，太过凶狠。
<br/>    
<br/>        步留痕道：“这要怎么办？还请无面者阁下示下。”

<br/>        陈明道：“所以，一把刀是不够的，我还需要另外一把刀。挑起瑶池和天庭的争斗之后，瑶池的河府防卫必定松懈，等到瑶池大举进军孽龙星河，带着你的人，截了瑶池河府的大狱，将那柄弓交给一个独腿的老者，告诉他，交易达成了，另外记得告诉他，别忘了我的加价。
<br/>    
<br/>        你有着八步登龙，想必离开也是很简单的一件事吧。”

<br/>        步留痕这才明白了过来，原来无面者阁下早就已经确定了另外一个执刀人，让自己前去营救。

<br/>        步留痕拱手道：“无面者阁下请放心，属下一定能够完成任务！”

<br/>        步生莲道：“爹爹放心吧，步生莲会照顾好自己的，这个无面者阁下神秘之极，爹爹在他的手下做事，定是前途一片光明。”
