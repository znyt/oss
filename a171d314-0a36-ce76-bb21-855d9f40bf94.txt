　　场中的气氛瞬间僵持了起来，大有一副即将开打的模样。
<br/>　　店长冷冷的盯着刘哔，语气也不像开始那般谦和了：“难道，刘先生不能体谅一些我们吗？”
<br/>　　嗤笑一声，刘哔直视着店长的双眼，道：“体谅……体谅是什么东西，委屈我自己让你们开心吗？”
<br/>　　店长正欲说些什么，却被刘哔接下来的话语打断。
<br/>　　“你又是以什么样的身份要求我？20区的喰种统治者？枭？或者是别的什么？”
<br/>　　“别做梦了，你睁眼看看这个世界，已经乱成了什么样子，CCG，青铜树，小丑，月山家，等等等等，在这样的地方，处处都充满了欲望和杀戮！”
<br/>　　“而你，又有什么资格，用这样的语气，倚老卖老的告诉我们，叫我们收敛！”
<br/>　　“你玩够了，累了，疲了，可我们没有，我们还年轻，没有活够，我们还要生存，没有食物，我们拿什么活下去！”
<br/>　　“像你们这些人，总是站在大义的角度，用最正气凛然的话语，劝我们去死，可我们又怎么可能答应这种要求呢？”
<br/>　　“你去劝劝人类，告诉他们要保护动植物，不要吃饭了，试试会有人听你们的吗？”
<br/>　　“真是的，老是说着这些不知所谓的话语，还以为自己很对的样子，这个世界的本质，就是弱肉强食，你又拿什么东西来改变呢？”
<br/>　　“你要真的这么慈悲心肠的话，那我告诉你，我现在很饿，快饿死了，你会让我吃掉吗？”
<br/>　　拿起桌上的白水喝了一口，湿润了一下嗓子，刘哔冷冷的盯着店长，嘴角勾起一抹嘲讽的笑容。
<br/>　　“看吧，遇到有关自身利益的事情，果然就不会再那样说了吧，归根结底，我们都是一样，只是立场不同而已，哪有什么对错是非。”
<br/>　　停止了讲话，刘哔将咖啡杯放在茶几上，站起了身来，伸了伸衣服上因为坐姿而新起的皱褶。
<br/>　　“谢谢你的白水，可惜我不喝咖啡，即使旁人都说你泡的咖啡很好喝。”说罢，刘哔推开屋门，顺着时的路径走了出去。
<br/>　　前厅，已经有了一些客人，店里的服务人员四处忙碌着，脸上挂满了笑意。
<br/>　　这些可怜的家伙，宁愿像懦夫一样躲在自己编织的梦境里，也根本不愿接受残酷的现实。
<br/>　　脸上露出一抹得逞的笑意，刘哔大步流星的离开了古董咖啡厅。
<br/>　　“滴滴……滴滴……”
<br/>　　走了一段路程，腰间的联络器又响起了警报，站在人来人往的人行横道上，刘哔掏出了联络器，上面显示有十几条未读消息。
<br/>　　这是CCG的专用联络器，形状和BB机差不多，但功能却要强大了不是多少倍。
<br/>　　这上面有卫星定位的功能，可以让搜查官随时能探测到队员的位置，当遭遇到危险时，也能用它发出警报，附近一个区域内的白鸽都可以接收到。
<br/>　　利用这个功能，刘哔击杀了许多白鸽，那些身处危险的白鸽发出警报时，非但没有获得援助，反倒是引来了一个死神。
<br/>　　前方的人潮中，一个手提银色密码箱的青年男子走了过来，在他面前站定。
<br/>　　黑色短发，身材偏瘦，长相很普通，让人很难留下印象，但一双眼睛，却是明亮的有些过分。
<br/>　　“刘君，终于找到你了呢，给你发消息你也没回，这几天你没来上班，也没回宿舍，是出什么事情了吗？”青年男子收起了手中的联络器，一脸关切的问道。
<br/>　　这个家伙，自己很是陌生，脑海中完全没有印象，但看他和自己熟识的样子，应该是系统给他安排的身份发挥了作用。
<br/>　　“哦，我这几天在调查一个十分机密的事情，一直没来机会回消息，一直到今天才有了结果。”刘哔嘴角带着笑容，随口找了一个理由。
<br/>　　这边的事情，已经解决的差不多了，也是时候回到CCG了。
<br/>　　毕竟，讨伐青铜树的战役也快要开始了，他可不想错过这个剧情。
<br/>　　“这样啊，是很重要的事情吗，可是明天有一场很重要的会议，刘君能来参加吗？”停顿了一下，青年男子继续说道：“大家都很担心你呢。”
<br/>　　脸上露出有些感动的神情，刘哔点了点头：“当然，我也有些事情需要禀报呢。”
<br/>　　闻言，青年男子笑了一下，伸出手腕，看了一眼时间。
<br/>　　“那么我们一起回去吧，搜查时间差不多也要结束了，虽然小鱼小虾抓住了两三只，但一点都没有修罗的消息呢。”
<br/>　　像是突然想到了什么，青年男子解释道：“哦，你这几天没再总部，可能不太清楚，等回去之后我在和你好好说说。”
<br/>　　应了一声，刘哔跟随着青年男子，一路往CCG总部大楼的方向走去。
<br/>　　或许是因为级别不够，男子并没有使用CCG的专用配车，现在也只能多花费一些时间，步行走回去。
<br/>　　……
<br/>　　到达总部的时候，已经到了下班时间了，青年男子领着刘哔到了一间办公室，写了一份报告之后，一起回到了宿舍。
<br/>　　作为政府大力扶持的公务人员，CCG的福利待遇还是蛮高的，虽然只是免费提供的公用宿舍，却和一般的家庭住宅没有什么区别。
<br/>　　四室两厅，家具应有尽有，除了两人之外，还有另外两名男性搜查官，一起住在这里，想必都是同一个小组内的成员了。
<br/>　　一晚上的时间，他都没有和这几人有太多交流，所幸的是，另外两人除了礼貌的问候了他一下，就没有继续搭理他。
<br/>　　看起来，小组内的关系也并不像是原著中所说的那样和睦。
<br/>　　反倒是领他回来的那个青年男子，一直和他絮絮叨叨的说着这段时间发生的事，比如，20区新出现的未知等阶喰种‘修罗’，比如有关明天会议的一些小道消息。
<br/>　　经过他有意无意间的引导，最终也知道了小组内几名成员的姓名。
<br/>　　松下立智，藤崎阳牧，以及在他身边一直很啰嗦的旧二多福。
<br/>　　前两个人他都没什么印象，估计只是普通的搜查官，但这个旧二多福，这个名字他总是觉得有些熟悉。
<br/>　　他们四人，都是由一名准特等搜查官‘木嶋式’带领着，木嶋式有自己的房子，平日并不和大家居住在一起。
<br/>　　对于这个家伙，他还是有些熟悉的啊。
