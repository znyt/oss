　　在这一战前。
<br/>　　在场众人中，绝没有任何一尊武者，会觉得能够有人以剑术，抗衡如此强大的力量。
<br/>　　夜天君的实力，的确有所隐藏，但是却也并不代表着，楚风眠所展现出的，就是真正实力。
<br/>　　起码这一剑，比起楚风眠当初击败风无心的剑术，还要强大。
<br/>　　隐藏的人，不止是只有楚风眠一人。
<br/>　　“哦？”
<br/>　　夜天君的脸色，终于开始变化了。
<br/>　　在他心中，本来以为他展现出真正实力，击败楚风眠，不过是轻而易举的事情。
<br/>　　楚风眠的剑术，虽然令他惊艳，却是没有被他放在眼中。
<br/>　　因为在他看来，他的力量，已经是足以碾压一切，就算是楚风眠的剑术再强，也只能被他碾压。
<br/>　　楚风眠的剑锋，居然是足以破开他的混乱长河，的确是令他有些惊讶。
<br/>　　但是夜天君是什么人物，他的脸色，很快便是恢复过来，露出了与之前一模一样的冷笑。
<br/>　　“好，楚风眠，我果然是有些小看了你，不过任凭你的剑术在高，想要跟我抗衡，也绝无可能！”
<br/>　　夜天君冷喝一声，他的身躯，如同一尊太阳一般，就在这擂台之中，缓缓升起，身上散发这一种异样的光芒。
<br/>　　这光芒，不是其他，正是最为纯粹的灵力，一团团的灵力围绕在夜天君的身边，令他看起来如同一尊烈日。
<br/>　　“我倒是要看看，你到底足以破开多少次！有种你就以剑术，将我的所有拳风，都给破开吧！”
<br/>　　夜天君一边狞笑这，一边已经是疯狂凝聚这灵力。
<br/>　　就在夜天君的身上，仿佛便因为了无数的混乱之力，令他的身躯，都是膨胀起来。
<br/>　　下一刻，这夜天君的身躯，猛然分化，足足是化为了九个一模一样的夜天君。
<br/>　　这分化出来的，并非是虚影，而是每一尊都是真正的夜天君，他的混乱之力，凝聚出的影子，每一尊的力量，都跟夜天君一模一样。
<br/>　　足以说每一道虚影，都是真正的夜天君。
<br/>　　这九尊虚影，同时出手，突然之间无数拳风，全部向着楚风眠袭来，这漫天拳风，将这擂台上都给占据。
<br/>　　如今一下子凝聚出九尊虚影。
<br/>　　也就意味着，夜天君的实力，又暴涨了九倍。
<br/>　　这无数拳风，几乎是源源不断的，向着楚风眠轰击过去。
<br/>　　要是一般武者，绝不敢做如此疯狂的举动，因为这样做，每时每刻所消耗的灵力，都是一个天文数字，以他们的灵力，根本无法支撑。
<br/>　　但是夜天君，却是先天神体，这先天神体的强大，更是体现在他的力量上，夜天君的灵力雄厚，要超越同境界武者的数十倍，数百倍。
<br/>　　所以他才足以这样肆无忌惮的动用灵力，来轰击楚风眠。
<br/>　　“不拿出一点真本事来，这种拳风，来多少都没有丝毫意义。”
<br/>　　楚风眠站在原地，脸色平静，他手持祖龙至尊剑。
<br/>　　就在这漫天拳风袭击过来的一刻，他手中的剑锋动了。
<br/>　　楚风眠的剑锋，看似缓缓的催动，斩出，但是瞬息之间，居然便是出现了万道剑气，在这一瞬之间，楚风眠的剑锋，都不知道斩出了多少次。
<br/>　　他的剑锋，实在太快。
<br/>　　快到任何人都没有看出，楚风眠的剑锋，到此是出窍了多少次。
<br/>　　这一道道拳风，在楚风眠的剑气下，被不断摧毁。
<br/>　　一息之间，这夜天君打出的漫天拳风，居然都被楚风眠这样轻而易举的化解。
<br/>　　这种攻势，都给楚风眠，造成不了任何影响。
<br/>　　“杀！”
<br/>　　同时就在这将所有拳风化解的一刻，楚风眠的身躯也是动了。
<br/>　　他的身躯，直接化为了一道虚影，速度快的吓人，几乎一瞬间，便是就来到了这夜天君的面前。
<br/>　　随着楚风眠的剑锋闪动，直接斩向了其中一尊夜天君。
<br/>　　“嘶！”
<br/>　　这夜天君被楚风眠直接斩破胸口，猛然破碎，显然是其中一道虚影，被楚风眠直接斩杀。
<br/>　　这突然的出手，令夜天君似乎都没有料到，他的心中都是惊出了一身冷汗。
<br/>　　楚风眠刚刚身形剑锋斩杀的速度，实在是太快了。
<br/>　　令他都没有反应过来。
<br/>　　好在楚风眠斩杀的，只是他的一道虚影。
<br/>　　若是刚刚，楚风眠斩杀的正好是他的本体的话，他就算是不死，也要重伤。
<br/>　　这其中任何一道虚影，都代表着夜天君真正的实力。
<br/>　　楚风眠可以这样，轻而易举的斩杀一道虚影，也就代表着，刚刚楚风眠想要斩杀夜天君的本体，都足以做到。
<br/>　　这得是什么力量，才足以做到一剑能够斩杀夜天君？
<br/>　　众人都有些不敢想象，楚风眠到底是有着多么强大的实力。
<br/>　　“楚风眠，看来我的确是小看了你，你的实力，足以让我认真起来。”
<br/>　　夜天君看向楚风眠，突然开口道。
<br/>　　“本来对付你，我没有打算动用全力，能够让我动用全力，你，已经足以自傲了，你的实力的确强大，但是在我的面前，你，依然没有胜算！”
<br/>　　“现在，该让你见识混乱神体的真正力量了，也让你明白，天才跟凡人之间的差距！”
<br/>　　夜天君一边说着，一边就在他身躯之中，似乎是有着一股灵力，被引动起来。
<br/>　　楚风眠看的出来，这是夜天君神体之中蕴含的一枚种子，混乱种子。
<br/>　　但是跟那些圣党弟子的混乱种子不同，这一枚混乱种子，是夜天君混乱神体所自己凝聚出来的。
<br/>　　跟楚风眠身体之中的本命剑元一样，代表的便是他最为强大的力量。
<br/>　　“轰隆！”
<br/>　　随着夜天君引动混乱种子的力量，他身边的气息，便是跟曾经相比，截然不同起来。
<br/>　　一尊巨大无比的虚影，笼罩在夜天君的背后，这虚影，看起来混乱无比，一些武者看到这虚影，内心之中似乎都已经是开始变的疯狂起来。
<br/>　　从这虚影之中，庞然的威压，顿时笼罩下来，如同一方领域一般。
<br/>　　将这整个擂台，都是化为了混乱的领域，楚风眠被笼罩在其中，他的一举一动，如今都受到了极大的限制，任何举动，都要需要耗费数十倍的力量。
