　　玄天回到人族后，便召来众多人族首领，商量以后人族发展的事宜。
<br/>　　人祖殿中，玄天高居于首座之上，旁边是燧人氏三祖，之后便是太昊氏、太康氏、少昊氏、大弈氏等人族的部落首领，再之后便是赤松子、雨伯等人族高手。
<br/>　　看了一眼众人，虽然此次三族大战人族损失惨重，但经过战争的洗礼的人族，就是不一样。如今人族的风气面貌都已经变了，人族已经自强起来了，不需要玄天再操多少心了。
<br/>　　玄天开口问道：
<br/>　　“先汇报一下人族此次大战伤亡情况。”
<br/>　　燧人氏心灵神会，看了一眼众人，说道：
<br/>　　“此次大战，我们人族也是损失惨重，转轮皇者境死去三个，盘皇盘王、风皇风后以及容皇容成子。其他各大强者也多有损伤，没有三五百年的时间，恢复不过来。
<br/>　　至于法相武相境，死去更是多大数千人，洞天尊者境，死去数万有余，神通王者境，更是死去近百万，人族伤亡惨重。”
<br/>　　听了这话，每个人的心头都变得极为沉重，更是感到一股心痛，一阵哀伤。这些人都是为了人族崛起而陨落之人，如今虽然人族胜利了，但他们却也不在了。
<br/>　　“这是不可避免的事情，人族想要崛起，这些牺牲是在所难免的。人族是不会忘了他们的，给我在首阳山上建立一个群英殿，里面刻上这些陨落英雄们的名字，日后人族可以前去祭拜他们。他们为人族崛起做出了巨大的贡献，人族是不会忘记他们的。
<br/>　　还有，那些英雄们若是有家属，以后便好好照顾，我们既然已经对不起这些为人族牺牲的群英，但却可以让他们死得瞑目，死得好受些。”
<br/>　　玄天一样心头沉重，但他并没有太过悲伤，也许这就是强者的无奈。他收拾好自己的心情，对着下面的人吩咐道。
<br/>　　“是，武祖！”
<br/>　　“多谢武祖大人！”
<br/>　　“……”
<br/>　　其他人听了这话，纷纷向玄天感谢道，有一个能够体恤人民的首领，的确是他们的幸运。
<br/>　　“好了，过去的伤痛我们要时刻牢记在心，时刻激励着自己，但人族还要往前看。人族从微末之中崛起，经历了各种大战，饱受了无数困苦艰辛，牺牲了无数英豪，如今终于战胜了巫、妖二族，走向了辉煌大道。不过，人族还没有完全兴盛起来，我今日叫大家来商量的，便是为了人族未来之事。”
<br/>　　玄天对着众人族高层缓缓说道。
<br/>　　“人族想要走上主角之路，仅仅这些还不够，我们要让整个洪荒大地都踏满人族的足迹。所以，接下来，人族要做的便是走出去。”
<br/>　　“走出去？如何走出去？”众人疑惑道。
<br/>　　“不错，走出去，我们需要搬迁，以首阳山为源地，慢慢向着洪荒其他地方搬迁。人族想要发展为洪荒第一大族，守着一个首阳山是不行的。我希望不久的将来，洪荒不论是东边、还是西边、北边与南方，都有我人族的部落。不过，这洪荒之中，隐藏的高手也有很多，想要做到这一切，还需要我人族有更多的高手护航。”
<br/>　　“所以，我人族可以先搬迁到一些安全之地，之后等人族发展起来，高手变多了，便可以一步一步的向着其他地方进发了。”
<br/>　　在场众人只听得热血沸腾，好似能够看到未来洪荒是人族的天下，人族为洪荒的主角，洪荒遍地都是人族的宏大场景。他们想想就觉得热血沸腾，心中顿时觉得豪情万丈。
<br/>　　“不过，人族想要大兴，还需要定三皇，立五帝，以此镇压人族气运，方可使人族气运永昌，久劫不衰。”
<br/>　　“大哥，这三皇五帝指的是哪三皇，哪五帝？是指修为到了皇者境界或者帝者境界之人，还是其他特殊之人？”缁衣氏开口问道。
<br/>　　此话一出，其他人便一同看向玄天，想看看他如何回答，若是任何人都可以，那岂不是他们自己也可以，一想到此处，他们心头便火热无比。三皇占据人族气运，修炼也必定飞快无比。
<br/>　　玄天赞许的看了缁衣氏一样，缓缓道：
<br/>　　“洪荒开辟，分立天、地、人三才大道。数起于一，立于三，成于五，盛于七，处于九。是故，天道当有九位圣人，洪荒六圣、道祖鸿钧、本来还有红云老祖，但奈何……还有一道遁去的鸿蒙紫气，总共九位。
<br/>　　地道，则是幽冥界六道轮回，也当有九位神皇，即轮回后土、五方鬼帝，三大冥主。所以，人道也应当有九位大帝，三皇五帝，即为就为人道大帝中的八位，剩下的一位，是为遁去的一。”
<br/>　　“这三皇五帝岂不是等同等那洪荒圣人了？！”有人惊骇道。
<br/>　　“说是，但也不是。天道圣人、人道大帝，地道神皇，在一定程度上，可以说是地位等同。但，开天之初，天道便已完善了，而人道与地道却是之后才补上的，天道力量又最为强大，统治着地道与人道。所以，天道圣人也最为强大。但，天道圣人想要对人道大帝与地道神皇出手，也会遭受因果天罚。”
<br/>　　先天想了想，如此说道。
<br/>　　“嘶！那岂不是说，三皇五帝也可以向天道圣人一般，历万劫而不灭？”人们惊骇道。
<br/>　　“在一定程度上，可以说是的。只要人族不灭，人道不毁，人道大帝便不会死亡。”
<br/>　　玄天说道。同时心中却暗暗道，三皇五帝看似是好，其实也有限制。他们的气运完全与人族连在一起，一荣俱损，只有一方有损，另一方便会遭受反噬。
<br/>　　“选择三皇五帝也有限制吧？”
<br/>　　缁衣氏身为女子，很是细心，她不相信什么人都可以成为三皇五帝，定是存在某种限制。
<br/>　　玄天深深的看了一眼缁衣氏，缓缓道：
<br/>　　“不错，这三皇五帝的选择，不是任何人想做便可以做的。选择的标准，不仅需要对人族做出巨大贡献，而且还需要有强大的修为，更重要的是，这是天道在选择，不是人为控制的。人可以干预，但不可控制。”
<br/>　　“这样么，倒也说得通了。”
<br/>　　人们心中的那些火热缓缓散了，他们也觉得这样很有道理，如果真是什么人都可以，那么天下岂不是都乱套了吗？
<br/>　　“好了，日后可能会有圣人的门人出现在人族之中，是收徒，还是别的，你们多多关注一番，不要被别人算计了。”
<br/>　　“是，武祖大人！”
<br/>　　众人族高层都若有所思，今日得到的消息实在是太大了，他们需要回去好好想想，更何况，人族还有许多大事在等着他们呢，人族还不是可以一劳永逸的时候。
<br/>　　不过，他们都明白一个道理，人族的新时代，来临了。
<br/>　　……
