　　　　难怪如此，一个平日里其笨无比的小师弟，突然间一鸣惊人，并且在一甲子一次的七脉会武上，杀进了最后了一轮，任谁也无法在短时间内接受。
<br/>　　在田不易咄咄逼人的目光下，张小凡额头上的汗水涔涔而下，有那么一刻，他几乎要冲口而出告诉师傅，是慕师弟在背后指点他修行。
<br/>　　然而，话到嘴边，他终究还是咽了下去，他已经不是五年前，无知懵懂的少年，知道有的事情能说，有的事情不能说。
<br/>　　“弟子愚笨不堪，这些年里修行境界一直进展不大。”
<br/>　　张小凡低下了头，不敢面对田不易的目光，斟言酌句缓缓开口道：“前些日子，弟子忽然发现能够驱动事物，但弟子自己都不能置信，所以不敢禀告师父师娘，没想到......”
<br/>　　田不易冷笑一声，道：“没想到这次却一鸣惊人，大出风头！”
<br/>　　田不易岂是这么好蒙骗过去的，当下声音变得冷冽道：“你说你能驱物，但这至少要有玉清境四重的修为，我问过大仁，他只传了你第二层的法决，那你可否告诉我，我这个孤陋寡闻做师傅的，你究竟是如何绕过第三层修炼至第四层境界的呢？”
<br/>　　他说到这里，声音已经变得冰冷无比，甚至带有几分煞气，听得众弟子骤然色变。
<br/>　　张小凡选择沉默，客房中一片寂静。
<br/>　　许久，就在田不易脸色越来越难看，众弟子担忧之情越来越重，在田灵儿好奇的目光中，跪了下来。
<br/>　　田不易丝毫没有动容，冷声道：“怎么？”
<br/>　　张小凡深埋下头，没有向四处在看一眼，低声道：“师父，请您惩罚我吧。”
<br/>　　众人耸然动容，田不易更是气的勃然变色，苏茹秀眉皱了皱，道：“小凡，你若是有什么顾忌便与你师父直说就是，何必如此？”
<br/>　　张小凡跪在地上，一动不动。
<br/>　　田不易冷笑两声，气极反笑，道：“很好，你倒是个硬骨头，我也是收了个好弟子，对我竟如此隐瞒。”
<br/>　　张小凡匍匐在地上，身子一颤，也不知道他此刻是什么心情与表情，片刻后，只听见他低声道：“一切都是弟子的错，请师父责罚我吧。”
<br/>　　田不易豁然起身，‘蓬’一声，在他身下的椅子竟是四分五裂的炸了开来，众人色变，只见他指着张小凡怒道：“都是你的错，呵...你可知道背师偷艺乃是我青云门中大忌，轻则面壁数十年，重则废去道行逐出青云，你可知道？”
<br/>　　张小凡猛的抬起头来，看着田不易，只见师父脸上满是怒容，但决无一丝夸张，心中不由得一沉。
<br/>　　他当然知道青云门背师偷艺的后果，但他终究没有回头看上一眼，客房中死一样的寂静，没有人开口说一句话，只剩下或高或低，焦急的喘息声。
<br/>　　就在这时，慕清霄上前一步，微微行礼道：“师父，是弟子斗胆，将后面的法决一并传给了张师兄。”
<br/>　　张小凡身躯一颤，心中唯有浓浓的感激之情，明明可以置身事外，却没想到慕清霄在这种时候，竟愿意站出来为他说话。
<br/>　　田不易愣了愣，张小凡是他带上大竹峰的，当年资质如何，没有人比他更清楚，哪怕法决传授于他，也不可能在短时间内提高这么多，期中肯定有猫腻。
<br/>　　慕清霄与张小凡在这些年里走的很近，大竹峰一脉弟子都清楚，将法决私传给张小凡也不是没有可能，他心中硬是给自己找了一个饶过张小凡的理由。
<br/>　　有慕清霄领头，以宋大仁为首强撑着虚弱的身体跪下，其他众弟子都在田不易面前跪下，道：“师父，你就绕了小师弟吧。”
<br/>　　田灵儿也是连忙跑到田不易身旁跪下，脸色煞白，此刻的张小凡在她的心目中非常特殊，自然不希望出后者受到一点伤害，道：“爹，求你饶了小凡吧。”
<br/>　　田不易看着跪在脚下的这群弟子，还有自己的爱女，又盯着身前的张小凡，看了一眼慕清霄，慢慢闭上双眼，摇了摇头，一甩袖袍走了出去。
<br/>　　苏茹看了众人一眼，摇着头轻叹一声，对宋大仁几人说道：“你们真是不让人省心，快都起来吧，我去看看你们师父。”
<br/>　　“是，师娘。”
<br/>　　屋内，众人面面相觑，半响过后，田灵儿走上前来，搀扶着张小凡，张小凡脸上尽是微笑，紧接着两人就窃窃私语起来。
<br/>　　......
<br/>　　......
<br/>　　此时，夜幕早已降临，云海广场上，依旧是那边云气飘渺，美如仙境。
<br/>　　田不易站在广场之中，昂首看天，但见夜空繁星无数，月冷如霜，身后响起熟悉的脚步声，苏茹走到他的身边，抬头看了看星空，淡淡的笑道：“心情好些了吗？”
<br/>　　田不易冷哼一声，却不说话。
<br/>　　苏茹微微一笑，道：“我们在一起这么多年，知根知底的，哪怕小凡背师偷艺，也最多也就小惩一下，你偏偏要装作愤怒的模样，都几百岁的人了，真是死要面子活受罪。”
<br/>　　田不易看着夜空，一声不吭。
<br/>　　田不易转过头来，瞪了一眼苏茹，道：“你又不是没看见，那臭小子跟什么似的，师父，请惩罚我吧，明明是他的错，居然还说得十分委屈的模样，真是岂有此理。”
<br/>　　苏茹回头向住宿居所方向看了一眼，道：“我就不信你没看出来？”
<br/>　　田不易道：“什么？”
<br/>　　“灵儿的样子有些古怪，她与小凡之间......那你准备回去以后如何收场，背师偷艺这个罪名可大可小，要不我们看在灵儿和清霄的份上就不要太过分，明日就让小凡会大竹峰，在后山面壁个三五十年也就是了。”
<br/>　　田不易一愣，哼了一声，却道：“面壁就算了，清霄不也说了，法决是他私下传授的，若真如此，灵儿那丫头恐怕也不会罢休。”
<br/>　　“而且，我大竹峰好不容易挺进七脉会武最后一轮，这可是百年来第一次，可不能便宜了苍松和商正梁他们，明日让他继续比试，若是夺得桂冠，让他与灵儿在一起，也未尝不可。”
