　　林峰瞧了眼刘峰，后者笑道：“走吧，上去露个脸。”
<br/>　　林峰点了点头就和刘峰上了擂台，就好像是那得了奖的员工上去领奖一样。
<br/>　　干事人不多，一共才十二个人。都是黑铁战士。
<br/>　　本来有十五个人，但是有三个据说是被军部挖走了，跳槽了。
<br/>　　“这些就是干事啊。”
<br/>　　“看起来也不怎么强嘛。”
<br/>　　“还有个学生。”
<br/>　　“握草，这特么成年没有，学生都可以当干事？”
<br/>　　许多人都议论纷纷，甚至许多人都不服气。
<br/>　　林峰操蛋的心情都有了，他发现许多人似乎都不屑的盯着他，他这招谁惹谁了。
<br/>　　他看了眼其余干事，都是大人，其中有几个满脸横肉一看就不是好人的家伙，这些家伙，没有一个人敢不屑的看着他们。
<br/>　　反倒是全都看着林峰。
<br/>　　“这尼玛老子以后也去剃个光头，也去贴个小猪佩奇的纹身在身上。”
<br/>　　林峰心中操蛋，学生怎么了，有志不在年高，英雄不问出处。
<br/>　　“诸位，这就是我们战士会目前的十二个干事，以后也只会有这么多。”
<br/>　　寒鸦扫了林峰等人一眼微笑道，他也是第一次看到十二个干事。
<br/>　　“会长，我斗胆问一句，这十二个人是怎么评选出来的。”
<br/>　　“没错，干事代表的是我们战士会的脸面，既然是脸面，那肯定其一就是要自身有实力吧。”
<br/>　　“就是，凭什么，他们可以当干事。我不服！”
<br/>　　“还有个学生，这尼玛，作业写完了吗？”
<br/>　　一群人都闹哄哄起来。
<br/>　　寒鸦举了举手示意每个人都安静。
<br/>　　瞬间每个人都不说话了。
<br/>　　因为排行榜的缘故，每个人都知道了寒鸦是个九阶的。无比尊敬无比敬畏。
<br/>　　“你们说的没错，干事的确是我们战士会脸面，这群人选拔，实际上也没有多么严厉，但是至少一点，这站着的十二个人，都是五阶的，至于以后该怎么选拔，那自然是拳头说话，不服气的可以去挑战他们的地位，若是打败了对方，那么你们就可以替代别人成为新的干事。”寒鸦道。
<br/>　　“五阶的？会长，那个学生也是五阶的？”
<br/>　　一群人问道。
<br/>　　“嗯，都是五阶的。”寒鸦看了眼林峰道。
<br/>　　一群人顿时不说话了。
<br/>　　“会长，五阶的又如何，等级和力量可不是一个人战斗力的全部，我自认学习过几年格斗术，实力还算可以，我要挑战他们，就那个学生！这家伙也不知道走了什么狗屎运，拿了这么多晶石变成五阶的！说不定就是绣花枕头一包草！”
<br/>　　一个大汉道。
<br/>　　“哦？你叫林峰吧，你意下如何。”
<br/>　　寒鸦看着林峰道。
<br/>　　林峰微微点头：“既然有人要丢人，那么我自然成全他。”
<br/>　　寒鸦笑道：“不错，既然林峰干事同意了，那么你们今天就在大家眼皮子底下决斗吧，当然，点到为止。”
<br/>　　大汉立即哈哈大笑：“多谢会长，这么看来，今天老子是要当干事了！”
<br/>　　一群人立即全部散开，把中间一块地方留给林峰和大汉。个个都抱起胳膊看好戏。
<br/>　　“不知道这家伙能不能成。”
<br/>　　“我估计十之八九没问题，没听他说吗，学习过格斗术，而这个学生，虽然五阶，但战斗力如何还未知。”
<br/>　　“没错，这五阶要是被四阶的打趴下了，那可真够丢人的，反正我是没脸再继续呆下去了。”
<br/>　　一群人都议论纷纷。
<br/>　　场中。
<br/>　　大汉看着林峰，挥舞了几拳，“看到没有，你将被我的铁拳打趴下来，我的拳头，会快到你根本看不清。”
<br/>　　林峰呵呵一笑，“放马过来吧。”
<br/>　　“哼，那你可要看好了，别被我一拳打趴下！”
<br/>　　大汉冷笑一声脚下一点地面就是朝林峰冲了过去，他的爆发力非常的强大。
<br/>　　然后一拳就是朝林峰的下颚击打过去，这一拳就是单纯的快而狠，虽然没有什么招式，但简单利落，就好像毒蛇一样，一击必中。
<br/>　　砰！
<br/>　　然而下一刻。
<br/>　　在一群人目光之中，大汉来得快去得快，整个人都还没碰到林峰的衣服就被一拳轰飞出去，在地上滚了七八圈，然后就直接晕厥了过去。
<br/>　　众人都愣住了。
<br/>　　“我去，这学生这么厉害？”
<br/>　　“英雄不问出处，还真是英雄出少年啊！”
<br/>　　许多人都彻底的震惊了，然后看着林峰的目光也是变了，一个个都无比的尊敬。
<br/>　　林峰收起拳头，吐了口气。
<br/>　　啪啪啪。
<br/>　　寒鸦鼓起掌来，微微点头：“很不错，大家对林峰干事可还有什么质疑？”
<br/>　　众人谁还敢质疑了，一个个都彻底的心服口服。
<br/>　　随后又有人挑战干事，不过没有人挑战林峰，而是挑战其余干事。
<br/>　　而林峰除了一开始那一战后，后面就没有再出手过。
<br/>　　最后。
<br/>　　十二个干事依然还是之前的那一群人，没有什么改动，现阶段能够成为五阶的，没有一个是善茬，岂能是那么容易被打败的。
<br/>　　随后大家就吃吃喝喝，这一次宴席之中准备了不少的吃的喝的。
<br/>　　当然了，寻常的宴席那最起码也是红酒外加各种精致的糕点。
<br/>　　可是这个宴席，全都是矿泉水外加辣条和一些肉铺之类的零食。
<br/>　　不过众人依然无比兴奋，一个个都大吃特吃。
<br/>　　林峰和陈勾等人也是拿着个八宝粥狂吃，不吃白不吃，这些东西外面现在卖的可贵了。
<br/>　　吃完东西后。
<br/>　　十二个干事留了下来。
<br/>　　其余的人都离去了，各回各家。
<br/>　　“诸位以后都是我战士会的干事了，大家以后要同心协力，把战士会继续壮大下去。”
<br/>　　寒鸦道。
<br/>　　“会长，你放心吧，战士会以后就是我们家了。”
<br/>　　“没错，一荣俱荣，一损俱损。”
<br/>　　立即有干事表态了。
<br/>　　其中一个男人引起了林峰的注意，因为这个男人给林峰的感觉似乎很强的样子。
<br/>　　“莫非他就是那个七阶的韩风？”
<br/>　　林峰在打量对方，对方也在看林峰。
<br/>　　“我叫韩风，你刚才那一拳很犀利。”
<br/>　　让林峰意外的是，对方主动走了过来。
