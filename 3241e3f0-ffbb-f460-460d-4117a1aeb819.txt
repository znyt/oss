　　“那下面的水，是从哪来的呢？”梧桐追问。
<br/>　　叶少阳只好耸了耸肩，“想弄清楚，那就只能下去看看了。”
<br/>　　陈晓宇说道：“我看可以，叶天师要不要下去看看？”
<br/>　　“你真当我傻比啊，你怎么不去？”叶少阳两眼一翻，这水虽然看着清，但里面可是有鬼鱼的，水里肯定泡过尸体，想想都觉得恶心，谁也不愿意下去。
<br/>　　曹雨兴提议把石棺砸了，让水流出来，然后再探寻下面那个洞穴，但是妙心等人都不同意，生怕这样做会触发什么机关或者不可逆转的事情，在一切弄清楚之前，最好还是不要破坏古墓里的东西。
<br/>　　最后还是叶少阳提议，去旁边的墓室看看，石棺里是不是跟这里一样。一句话点醒了大家，结伴来到隔壁的墓室，然后跟之前一样把石棺打开，果然还是一池子水，中间有个孔洞，里面躺着一只鬼鱼，一动不动的。
<br/>　　“咱们把这鱼捞出来看看怎么样？”陈晓宇提议。
<br/>　　妙心立刻表示不妥。
<br/>　　“捞出来，我也同意。”叶少阳说道。妙心有些诧异地看着他。从下墓开始，叶少阳始终表现得比较沉稳，站在她这一边，刷新了妙心对他的印象分，觉得虽然实力不咋样，但不是个鲁莽的人，不知道为什么会同意陈晓宇这个鲁莽的建议。
<br/>　　叶少阳说出了自己的想法，妙心想了一下，豁然开朗，说道：“那根本不用抓出来。”
<br/>　　当下从包里摸出了一个木盒，打开来，里面是一大把针，不过不是金属的，而且金黄色的稍微有点弯曲。
<br/>　　“这是我家常用的松针，被我用莲心水泡过，七子连心阵所禳，能十里追踪。”
<br/>　　叶少阳听她说完，想要问她怎么用，妙心拈起一根松针，朝水中掷去，正好扎在那鬼鱼的身上，鬼鱼摆了一下尾巴，纵身朝水底钻了进去，潜入了水下的洞穴之中。
<br/>　　妙心从随身小包里摸出一只拳头大小的莲花，金色的，也不知道是真的还是什么材料做的。
<br/>　　妙心把莲花捧在手中，上手一上一下地覆盖在上面，闭上眼睛，口中念起了咒语，手中的莲花亮起金光，并且幻化出了虚拟的叶片，在妙心的手中旋转起来，突然，莲花打开，金光收敛成为一束，落在妙心的手中，然后幻象消失。妙心深吸一口气，说道：“跟我走吧。”
<br/>　　“什么意思？”叶少阳问，“你知道那鬼鱼去哪了？”
<br/>　　妙心不说话，转身出了墓室，朝墓道深处快步走去。
<br/>　　一行人紧随其后，路上遇到好些个墓室，里面都摆着石棺，虽然不知道这些石棺和里面的鬼鱼都是用来干什么的，但是这么大的阵势和透露出的诡异的气氛，让所有人都好奇不已，内心也有些紧张。
<br/>　　一路走过去，最终来到一扇拱形的石门前，石门是开着的，阵阵阴风从里面吹出来。
<br/>　　“先别进去。”妙心一马当先，将大伙挡在身后，从叶少阳手中拿过三色莲花灯，对着石门上照过去，发现石门上有一个凹陷处，中间镶嵌着一个好像轮盘的所在，轮盘是若干小石块拼凑而成，每一块石条上都有一个天干地支的标记，轮盘一共有两圈，凑在一起就是天干地支的整数。
<br/>　　不用说，这一看就是个用九宫八卦凑成的“密码锁”。
<br/>　　“这里之前有人进去过？”卢晓清看看密码锁，又看看敞开的石门，纳闷道。
<br/>　　妙心咬了咬嘴唇，说道：“可能是我祖先，当初从这里逃出来，一路被炼尸缸里的邪物追杀，自然是没有时间关闭石门。”
<br/>　　说完，她认真打量了一会轮盘，半晌说道：“幸亏我祖先打开了石门机关，不然我们还真不好进去。”
<br/>　　卢晓清上前看了一会，说道：“这是天干配地支，一共六十种变化，只有一组是正确的，确实不好判断。”他是佛门弟子，不过这么简单的八卦阵法，他一个和尚也是能看出个大概。
<br/>　　曹雨兴脱口说道：“一个一个试就是了。”
<br/>　　这句话一说出口，连卢晓清都觉得自己这个猪队友智商有问题，瞥了他一眼说道：“如果可以一组一组试过去，还要这机关做什么？我怀疑，这机关附近一定是有埋伏，如果组合错误，想来是会发生不可逆转的事情。”
<br/>　　大伙抬头在石门附近寻找起来，虽然没有找到机关的所在，但是对卢晓清的话都不怀疑。
<br/>　　“没有这么简单。”叶少阳用三色莲花灯照着轮盘，说道，“这个不是简单的八卦机关，而是利用了一组先天八卦的易数。”
<br/>　　“先天八卦？”陈晓宇鼻子里哼了一声，不假思索地就是反驳起来，“你开什么玩笑，这世上哪有什么先天八卦，就算有，凭你，怎么可能会知道？”
<br/>　　妙心狠狠瞪了他一眼，转头看着叶少阳，一脸的狐疑，“你怎么知道是先天八卦？”
<br/>　　“道生一，一生二，二生三，三生万物……八卦机变，九爻辅助，这是后天八卦，你看这轮盘，虽然只有两柱，但彼此对应的是癸亥和乙寅，你们自己推算一下，要想在后天八卦中出现这两组搭配，是完全不可能的……”
<br/>　　在场几人一听，除了梧桐和卢晓清是佛门弟子，对八卦没什么深入研究，站着发呆之外，其余人都各自推算了一下，这才发现，叶少阳说的是对的。
<br/>　　其实这个道理非常浅显，就算是初学八卦的人也能看出其中的问题，就因为这是太基础的问题，所以反而被人忽略了存在，经过叶少阳这么提醒才恍然大悟。
<br/>　　“那就说明，这轮盘把天干地支的位置搞错了。”曹雨兴不假思索地说道。
<br/>　　叶少阳瞥了他一眼说道：“你以为人家跟你一样猪头，能在这里用禁制封锁石门的人，会犯这么低级的错误吗？就算他弄错了，那妙心的祖先是怎么进去的，一个错的题目，还能有正确答案吗？”
<br/>　　“这……”曹宇讯张口结舌，一时间想不到话来反驳。
