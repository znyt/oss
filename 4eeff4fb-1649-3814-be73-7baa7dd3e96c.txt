
<br/>　　<br/>　　    手术室的门在凌晨前，一共来来回回开关了几次，主要是以输血为主，丁武失血过多根本都止不住，到最后医院血库的血都不够用了，又从其他医院调来的，才勉强输上，不然人可能当晚就死了。<br/>　　<br/>　　    一直到早上五点多钟左右，手术才结束，丁武出来的时候浑身上下被包的跟个粽子似的，都看不出本来面目了，整个上身全是白色的纱布。<br/>　　<br/>　　    “医生，我朋友他怎么样了？”齐泉在外面等了半夜，看见丁武被推出来还没醒就有点急了。<br/>　　<br/>　　    “现在还不清楚，伤势暂时稳定了，不过他的身体机能还没有恢复，危险期也没过，如果明天晚上能醒来的话还好，醒不过来的话还得继续观望，现在人得送到重症监护室去了”<br/>　　<br/>　　    “好，好，麻烦您了”齐泉连连点头跟着去了ICU后，在外面他才想起来要跟老马说一声，但电话打过去两遍那边都没接，齐泉暂时也就没打了，然后让人去下边交住院费和手术费，留下个人在医院等着他就回去了。<br/>　　<br/>　　    早上七点左右，酒店里面，一夜宿醉之后，王惊蛰和茅小草差不多是同时醒来的。<br/>　　<br/>　　    两人四目相对，一顿尴尬。<br/>　　<br/>　　    憋了半天，王惊蛰嘴里吐出两字：“早啊”<br/>　　<br/>　　    “早！”茅小草从床上爬了起来，看了自己身上完好无损的衣服，嘴角略微翘起点弧度就往卫生间走去了。<br/>　　<br/>　　    “这他么的，尴尬不尴尬？”王惊蛰叹了口气，卫生间门被“咣当”一声关上了，茅小草手拄着洗手台，看着镜子里的自己，笑了笑：“憧憬就从这个时候开始了……”<br/>　　<br/>　　    半个多小时后，王惊蛰茅小草两人之间隔着挺远的离开了酒店，站在马路边。<br/>　　<br/>　　    “唰”他俩同时抬手，两辆出租车先后开了过来。<br/>　　<br/>　　    “咣当”王惊蛰跟落荒而逃似的，上了车关上车门，跟司机说了地址车就开走了。<br/>　　<br/>　　    茅小草萌萌的看着消失的车尾灯，半天后才愤愤的骂道：“就好像吃干抹净脚底擦了油似的，你跑个毛线啊……你不没吃到最里呢么”<br/>　　<br/>　　    王惊蛰坐在车里，抹了下头上的虚汗：“我他么的紧张啥啊，我不啥也没干么！”<br/>　　<br/>　　    记忆虽然不在了，但本性还在，王惊蛰在面对茅小草的时候，仍然跟一只受惊的兔子差不多，本性难移了！<br/>　　<br/>　　    回到家里的时候，菜刀文刚从床上爬起来，他一把按上王惊蛰的肩膀，眯着眼打量了好几眼，促狭着说道：“呵呵，春宵一度了呗？”<br/>　　<br/>　　    王惊蛰扒拉开他手，说道：“别闹，她喝多了我给她送到酒店里去了，啥事没有”<br/>　　<br/>　　    “哦……”菜刀文拉着长音说道：“她喝多了，嗯，酒店”<br/>　　<br/>　　    王惊蛰都懒得和他解释，打着哈欠刚要回卧室，就看见丁武的房间没开车里面没有人，问道：“小武这么早就出去了啊？”<br/>　　<br/>　　    “和你一样，都没回来”<br/>　　<br/>　　    “唰”王惊蛰顿时皱眉，说道：“他也没回来？不是跟齐泉出去办事了么，办了一夜？”<br/>　　<br/>　　    “那谁知道了”<br/>　　<br/>　　    王惊蛰也没觉得一夜未归的丁武有什么不妥，都是成年人了，夜不归宿很正常，就比如昨天的他。<br/>　　<br/>　　    等到了下午的时候，丁武还没有回来，王惊蛰就跟菜刀文说了要不咱俩去一趟酒吧看看吧，丁武这个点估计应该过去了。<br/>　　<br/>　　    “你是为了丁武，还是为了偶遇？”菜刀文眨了眨眼睛问道。<br/>　　<br/>　　    “呃！”忽然有那么一瞬间，王惊蛰确实想到了那位小草姑娘，但他肯定不能承认啊，就硬着头皮理所当然的说道：“当然是为了小武啊，一天都没个动静，我能不担心么？”<br/>　　<br/>　　    菜刀文冷笑着说道：“为了丁武？那你可以打电话啊，你至于巴巴的特意跑一趟酒吧么？哥们，别嘴硬了，你的心是骗不了自己的”<br/>　　<br/>　　    “我真的是惦记丁武的……”王惊蛰弱弱的说道。<br/>　　<br/>　　    “那打电话就行了”菜刀文咬着字眼说道。<br/>　　<br/>　　    王惊蛰悲愤的捏了捏拳头，为了掩饰内心深处的念头，他不得不拿出手机给丁武拨了过去，但打了几遍之后那边都无法接通，于是又给缪斯那边拨了过去，这才发现丁武也没去酒吧。<br/>　　<br/>　　    王惊蛰和菜刀文顿时都愣了，丁武现在已经属于失踪了，一天一夜没见人影又联系不上了。<br/>　　<br/>　　    王惊蛰就从陈臣那里要来了齐泉的电话，于是给他打了过去：“泉哥？我是小武的朋友，惊蛰……”<br/>　　<br/>　　    一个电话打完，王惊蛰和菜刀文匆忙的从家里出来赶往了医院，电话里齐泉告诉王惊蛰丁武在重症监护室，被砍了十几刀，人还没有脱离危险。<br/>　　<br/>　　    坐在车里，王惊蛰的脸色很阴沉，丁武和王冬至是他醒来之后所见到的第一波人，尽管他当时不记得对方了，但他仍然记得丁武曾经和他说过的那句话。<br/>　　<br/>　　    “我这辈子就你一个朋友……你也一样，就我一个哥们”<br/>　　<br/>　　    到了医院，来到ICU病房，站在透明玻璃窗外，王惊蛰静静的看着床上昏迷不醒的丁武，他身上包裹的全是纱布，这说明人身上几乎是没有一块完好无损的地方了。<br/>　　<br/>　　    “当时追人是小武先追过去的，等我们赶过去的时候前后没差五分钟，但却已经来不及了，小武躺在了地上，前胸口上被砍了十几刀，肠子都漏出来了……”<br/>　　<br/>　　    菜刀文冷冷的盯着他说道：“你们毛都没事，就丁武被砍了？”<br/>　　<br/>　　    “也死了一个，小武追的太快了我们根本都没来得及跟他一起过去”齐泉皱眉说道。<br/>　　<br/>　　    “为了陈重的事？”王惊蛰突然转身问道。<br/>　　<br/>　　    齐泉一滞，随即嗯了一声：“是的”<br/>　　<br/>　　    就这么一瞬间，王惊蛰对陈重不满的情绪瞬间蔓延开来，丁武躺在这里生死不知，就一个齐泉在这里，他连面都没漏，真拿丁武当小马仔看呢？<br/>　　<br/>　　    其实，王惊蛰不知道的是，陈重对于丁武的问题，到现在还是不知情的，只有老马那里下午的时候被齐泉通知了一声，老马让他赶紧救治，其他的倒也没说什么。
