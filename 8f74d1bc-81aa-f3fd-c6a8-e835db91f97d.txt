        刚刚在下头的时候，保安连线上来说，有个戴着假面眼镜的女孩要找他，他二话不说就让保安放她上来了。
<br/>
<br/>        傅夜沉也没想到，童筱颖真的会来。
<br/>
<br/>        他以为，起初他在电话里说的那些话，会令她生气，甚至以她那性子，一定会赌气不来。
<br/>
<br/>        因为，“夜惑”应该不值得筱颖这么做。
<br/>
<br/>        但是，她来了，却让他感到意外。
<br/>
<br/>        “坐。”傅夜沉迎了迎手。
<br/>
<br/>        他的办公室里未开大灯，只有办公桌上的那盏台灯，将这偌大的办公室里笼罩上一层朦胧的微光。
<br/>
<br/>        童筱颖听从指令，在沙发上抬头挺胸，端端正正地坐下。
<br/>
<br/>        “你随意，我得去把我的文件看完。”傅夜沉淡淡地说，然后自顾自地坐回了老板椅上，开始接着看文件。
<br/>
<br/>        童筱颖目光呆滞地看着前方，呼吸平缓，双膝并拢，双手轻轻叠放在大腿上，一句话也不说，一动也不动，完完全全就像一尊雕像一样坐在那里。
<br/>
<br/>        因为她太过安静，傅夜沉也没多想，一门心思地将自己的文件看完后，才发现已经快凌晨了。
<br/>
<br/>        当他放下手中的钢笔，有些头疼地揉了揉太阳穴，闷哼了一声时，坐在那边的童筱颖突然从沙发上起身，走到他的身后，抬起手来给他按摩了起来。
<br/>
<br/>        傅夜沉对于她的这一举动，惊吓得从老板椅上起身。
<br/>
<br/>        而在童筱颖的潜意识里，费尔南的那句话一直在不停地重复命令着：一定要把连少伺候好！
<br/>
<br/>        所以，她现在所做的事情，就是在伺候连少。
<br/>
<br/>        “你若是累了，去楼上睡吧！”傅夜沉回过神来，有些疲惫地说道。
<br/>
<br/>        他完全专注于自己的事情，丝毫未察觉到童筱颖的异样。
<br/>
<br/>        童筱颖误以为是他“连少”累了，需要去楼上去睡觉，于是踱步上前，伸手揪住了傅夜沉的领带，转过身去，直接拉动着他的身子一起往前走。
<br/>
<br/>        “以……小谢！你——”颈部忽然一紧，傅夜沉话还未说完，就被领带给勒得喘不过气来。
<br/>
<br/>        “哒哒哒”的脚步声。
<br/>
<br/>        童筱颖拽着傅夜沉上楼后，直接将他推倒在了床上，然后她一个跨步，坐上了他的大腿，替他拉松了领带，然后开始解着他衬衫上的纽扣。
<br/>
<br/>        傅夜沉得以呼吸，呛得咳嗽了几声，伸手便抓住童筱颖的手，阻止她接下来的动作。
<br/>
<br/>        “小谢，你真打算用自己的身体去换‘夜惑’？你确定你这么做值得？”傅夜沉低吼，冷冷地质问。
<br/>
<br/>        童筱颖面不改色，目不转睛地看着傅夜沉，回答道：“小谢会伺候好连少！”
<br/>
<br/>        傅夜沉剑眉微蹙，抬起手来，轻轻地摘掉了童筱颖脸上的假面眼镜。
<br/>
<br/>        整个过程，她没有一丝抗拒，就这样愣愣地坐在他大腿上，双目发直地看着他。
<br/>
<br/>        看到这假面眼镜下的面容，真真切切的就是童筱颖时，傅夜沉心中一滞。
<br/>
<br/>        他随之抬起手来，在童筱颖的眼前晃了晃，发现她双目无神，完全找不到目光的焦点。
<br/>
<br/>        “筱颖，你这是怎么呢？”傅夜沉连忙撑起身子坐了起来。
<br/>
<br/>        童筱颖仍旧跨开双腿，坐在他的大腿上，一动也不动地看着他。
<br/>
<br/>        傅夜沉神色一敛，试探性地命令道：“小谢，从我身上下去！”
<br/>
<br/>        童筱颖果然立即照做。
<br/>
<br/>        她挪开双腿，从他身上下来后，跪坐在了他的身边。
<br/>
<br/>        傅夜沉这才意识到，筱颖可能被人下了药。
<br/>
<br/>        “夜惑”里人，还真的是让他大开眼界，居然什么药都有。
<br/>
<br/>        但现在从童筱颖的自身条件来看，她除了不知道自己在做什么，其他似乎并没有什么大碍。
<br/>
<br/>        傅夜沉坐在床上，屈起长腿，一手搭在膝盖手，一手撑在床铺上，支起上半身，意味深长地凝视着面前这个不吵不闹格外安静又听话的童筱颖。
<br/>
<br/>        “小谢，说一句，我爱你，来听听。”傅夜沉不知道自己是怎么了，居然趁机跟童筱颖开了玩笑。
<br/>
<br/>        童筱颖微微偏头，双目无神地看着傅夜沉，讷讷地开口：“我爱你。”
<br/>
<br/>        “你爱我什么？”傅夜沉苦笑着问道。
<br/>
<br/>        是啊！
<br/>
<br/>        她爱他什么？
<br/>
<br/>        这句话，是他让她说的。
<br/>
<br/>        而且，完全不用经过脑子，随口说的。
<br/>
<br/>        童筱颖似乎不会思考了一般，完全不知道如何回答傅夜沉的问题。
<br/>
<br/>        “筱颖，我爱你。”傅夜沉真诚地凝视着童筱颖的眼睛，深情款款地说。
<br/>
<br/>        也就只有这个时刻，他才敢大胆地把藏在心底的爱意说出来。
<br/>
<br/>        童筱颖仍旧木讷地看着他，面对他的告白，她身心上没有任何反应。
