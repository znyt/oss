　　叶少阳沉默半晌，道：“我觉得，很有可能是法术公会的人，除了他们，天下应该没人知道你是谁？”
<br/>　　陈悦皱眉道：“我是谁？”
<br/>　　“我的意思是，藏在你身上的秘密，恐怕只有他们知道。”
<br/>　　陈悦默不作声起来。
<br/>　　叶少阳问起她接下来的打算，陈悦的想法是，一定要弄清楚这一切的真相，不能让师父白死。
<br/>　　“虽然师父是自杀的，但其实她也是被人逼死的，我一定要弄明白事情的真相！”
<br/>　　“那……你跟我一起下山？”
<br/>　　“你如果不是很忙，可以等我一天，我总要收拾一下，毕竟这么多年了，总有些东西想要带走的。”
<br/>　　当天，陈悦依然没有开山门，在里面收拾东西，填满了两个箱子，下午煮了自己种的茶，请叶少阳一起喝。
<br/>　　“真要走了，还真有些舍不得，我是真想留在这里，哪怕只有我一个人。”黄昏的时候，两人一起来到道观后面的绝顶上，站在小亭子里，陈宇远远地望着道观，满怀不舍的说道。
<br/>　　“可是你师父做出这么大牺牲，就是想要保护你，你留在这里，会被人找到的。”
<br/>　　“我知道，我只是不舍得……也许将来，如果有一天我能像你这么厉害，我一定会回来，一个人住在这里。”陈悦转冲他笑着，“到时候，你可一定要来找我，我请你喝茶。”
<br/>　　两人相视一笑。
<br/>　　“老大，你们看啊，道观里有人！”
<br/>　　瓜瓜从亭子顶上跳下来，指着道观方向说道。
<br/>　　叶少阳定睛看去，的确是有一个白衣人，在道观里走着，距离太远，连男女都看不清。
<br/>　　“门已经锁了，香客不可能进来的！”陈悦道。
<br/>　　“看看去！”
<br/>　　叶少阳飞奔下山，从道观后门进去，一进院子，立刻看到了那个白衣人，愣了一下道：“你怎么来了！”
<br/>　　居然是林三生！
<br/>　　林三生书生打扮，手里拿着一把折扇，冲他笑道：“自然是来找你。”
<br/>　　叶少阳跟他互相留了魂信，需要找对方的时候，能通过它来感知到对方的位置，不过魂信只在现实空间有用，在一切虚幻空间里都会失效。
<br/>　　林三生看到他身后的陈悦，拱手道：“见过姑娘。”
<br/>　　“你好。”之前在虚幻世界中，陈悦也多次见过他，只是没有说过话，算不上熟。
<br/>　　叶少阳忙问他：“什么事？”
<br/>　　林三生道：“不请我坐下慢慢说吗？”
<br/>　　“你一个鬼，坐什么坐。”
<br/>　　叶少阳嘴上这么说，还是带他进了房间，让瓜瓜点了一炷香给他吸。
<br/>　　“这地方可以，纵然不是什么名山大川，但灵气充盈，一两个人清修是足够了。”林三生站在窗前，张望着四周说道。
<br/>　　“你很闲吗？”叶少阳嘟囔道，“有事就快说啊。”
<br/>　　陈悦起身道：“你们聊吧，我去师父坟前再烧一刀纸，然后回来做饭。”
<br/>　　叶少阳想挽留她，听见林三生干咳了一声，这时陈悦已经离开了。瓜瓜瞪了林三生一眼，刚要做，林三生道：“瓜瓜你回来！”
<br/>　　瓜瓜只好站住。
<br/>　　叶少阳抱怨道：“她是自己人，你有什么就说，怕什么。”
<br/>　　林三生道：“防人之心不可无，你不要见个人就是自己人。”
<br/>　　“行吧，懒得跟你解释，你说吧，什么事。”
<br/>　　林三生道：“两件事，第一，黎山老母他们最近来到人间，你可知道。”
<br/>　　“什么！”
<br/>　　叶少阳大吃一惊。
<br/>　　“黎山老母，摇光仙子，还有几个阐教金仙，偷偷来了人间，去了泰山，据说是要封禅泰山，寻找东岳大帝的信物……”
<br/>　　“东岳……大帝？”叶少阳不相信自己的耳朵。
<br/>　　“是了，他们要请东岳大帝降临，好带领他们扫清空界，驱逐尸族。”
<br/>　　叶少阳怔怔地看着他。“东岳大帝……在泰山？”
<br/>　　“自然不在，传闻他是在九重天上，但我们都知道，没有真正的神界，他一定是在某个不同位面的空间里，类似无量界，想要请他降临，需要有他的信物……而泰山是东岳大帝的道场，也他得道成仙的地方，传闻他的信物就在泰山。”
<br/>　　叶少阳听他说完，刚要开口，林三生摆手说道：“你不要问我更详细的，因为我也不知道，我又不是阐教中人，我只是得到了线报，这次寻找东岳大帝信物的行动，是摇光仙子提议的，他们姐妹已经悄然来了人间，或许还有几个人，后续我将继续调查，提供给你最准确的情报。”
<br/>　　叶少阳这才反应过来，站起来说道：“什么情报，跟我有什么关系！”
<br/>　　林三生严肃看他，道：“我要你去抢夺信物！”
<br/>　　“疯了你！为啥要这么做！”
