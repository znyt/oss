一个黑衣男人在开车，另外三个黑衣男人围着夏夕绾，他们很满意这一次的货色，果然是极品。
<br/>
<br/>        开车的黑衣人已经迫不及待了，他催促道，“你们速度快一点，赶紧玩，玩好了来换我。”
<br/>
<br/>        “不要急，面对这样的极品，我们怎么快的起来啊，你们先来，我来拍视频，雇主可是交代了，多拍点照片和视频。”
<br/>
<br/>        一个黑衣男人拿出了手机，捏着夏夕绾的脸就来了一个特写，“大家来看一看，这张脸多漂亮啊，快点把她衣服扒了，我们来看一看她的身体。”
<br/>
<br/>        “我来了。”另一个黑衣人两只手来到了夏夕绾里面的T桖领口上用力的一嘶，伴随着布料被撕开的声音，夏夕绾莹润的香肩暴露在了空气里。
<br/>
<br/>        这几个黑衣人看的双眼都直了，“哇喔，这肌肤跟牛奶白似的，这手感，滑的像丝绸一样，艹，我感觉自己像吃了兴奋剂一样的兴奋，我已经忍不住了，我先来！”
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，可是睁不开眼，肌肤沁凉，耳畔都是那几个男人猥琐亢奋的笑声，让人作呕。
<br/>
<br/>        这几个男人动作十分的利索，应该是高薪聘请来的，他们并不想要她的命，而是将她带到车上轮了然后拍摄视频。
<br/>
<br/>        对于一个女孩子而言，这是最大的耻辱了，简直比死还难受，很明显这个幕后主使者是想折磨她，毁了她的声誉。
<br/>
<br/>        夏夕绾动了动，想挣扎，但是浑身提不来力气。
<br/>
<br/>        这时一个黑衣人已经迫不及待的扑了过来，压在了她的身上，伸手就去扯她身上的裤子，“我还从来没有玩过这样娇滴滴的小美人，你们都在我后面排队，让我先爽一下哈哈。”
<br/>
<br/>        夏夕绾闭着眼，这么一刻感官特别的清晰，她已经感觉到了这些人的侵犯，但是她无力反抗。
<br/>
<br/>        这种无力的感觉就像是她坠入了一摊死水里，然后慢慢的往下沉，就要沉溺了。
<br/>
<br/>        她已经很久没有体会到这种无助的感觉了，上一次还是在她九岁那一年，妈咪和爷爷相继离开，她变成了一个人。
<br/>
<br/>        都说帝都卧虎藏龙，原来危险一直蛰伏在她的身边，她错信范恬，这就是她必须承担的后果。
<br/>
<br/>        夏夕绾眼里热热的，里面像是有什么滚烫而湿润的东西要流出来了，这么绝望的一刻，她满脑子都浮现出了一张精致英俊的脸庞，是陆寒霆。
<br/>
<br/>        是她的陆先生。
<br/>
<br/>        这么一刻，真的好想好想他。
<br/>
<br/>        后面的黑衣人在扯夏夕绾的裤子，这时前面开车的黑衣人突然叫了起来，“不好了，后面有车冲了过来！”
<br/>
<br/>        什么？
<br/>
<br/>        这几个黑衣人相当警觉的，他们当即趴在后面的挡风玻璃上看去，只见后面那辆劳斯莱斯幻影豪车箭一般的冲了过来。
<br/>
<br/>        几个黑衣人瞳仁一缩，盛满了惊悚，他们惊悚的看着开车的那个人，陆寒霆大手搭在方向盘上，目不斜视的盯着前方，他深邃的狭眸透过挡风玻璃如同X光般射在了这几个黑衣人的脸上，他的神色面无表情，漫不经心，看着格外的森然嗜血，让人头皮一麻。
