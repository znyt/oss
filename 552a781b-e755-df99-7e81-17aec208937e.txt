　　叶少阳说要去帮镇长家捉鬼的时候，她以为他只是一个神棍，去骗点钱而已，虽然无可厚非，但是没想过他会成功，结果，他竟然真的做到了。
<br/>　　他说要送自己一个那种样式奇怪的西洋钟，她以为他只是随口一说，结果他居然也做到了。
<br/>　　甚至他很随意地就将巨额的报酬中的大部分给了自己，这是一笔自己一辈子都赚不到的钱，他却眉头都没皱一下……
<br/>　　自己活了这么大，从来没有见过这么了不起的人，他跟自己所有认识的人都不一样，简直可以用奇迹两个字来形容。在翠云心中，叶少阳就好像是一个不属于这个世界的人……她不想就这么跟他分开。
<br/>　　这个念头在她脑海中盘踞不走，让她感到很难受，很矛盾。
<br/>　　也许……他只是自己人生中一个匆匆的过客吧。翠云叹了口气，让自己不要再想下去。
<br/>　　第二天一早，叶少阳起床收拾东西，自己那些东西其实没什么好收拾的，主要是之前帮秋镇长捉鬼的时候，叶少阳借故找了弄来不少朱砂、黄纸这些通行法器，留着将来用。
<br/>　　收拾好之后，吃了翠云煮的粥，翠云把送他的东西打成了一个很大的包袱，叶少阳背在身上，出门的时候遇到几个邻居，翠云解释是送表弟离开，寒暄几句，两人出发，一路往镇上去。
<br/>　　叶少阳心中也有一些离别的伤感，嘱咐了她一些事情，并且保证自己将来一定会来看她。翠云只是点头不吭声。
<br/>　　马车要到县城去坐，两人到镇上之后，找了个茶馆，喝茶休息了一会，然后步行出发。
<br/>　　一路走到县城，都已经中午了。
<br/>　　叶少阳还是第一次来上蔡县的县城，感觉跟电视剧里看到的民国县城也差不多，只是建筑物之类的要破很多，大部分都是古建筑，人也没有那么多，不过沿街还是有些镇上看不到的商铺。
<br/>　　叶少阳跟着翠云，一路走马观花地从镇上走过，心里想的是这样一个原汁原味的民国县城假如能够保存到百年之后，那可真是能吸引大批游客，虽然不知道这个上蔡县具体是百年后的什么地方，但是想到将来这里翻天覆地的变化，尤其是自己走在这些古人的中间，这种感觉……真的是有点恍若隔世的感觉。
<br/>　　翠云带着叶少阳来到车行，是一个非常的大的厅子，里面摆着很多板凳，稀稀拉拉地坐着一些人，对面就是马厩，能看到一些人在给马匹套鞍投食之类，叶少阳四下看了一下，这里应就是候车厅一类的所在了。
<br/>　　翠云留下叶少阳看行李，自己去办手续，回来的时候告诉叶少阳，还有两个客人没到，需要等一会。
<br/>　　叶少阳听了很是纳闷，问道：“别的客人跟我有啥关系？”
<br/>　　“大家坐一辆车啊，人没到齐，马车怎么出发啊。”
<br/>　　叶少阳怔了怔，问道：“不该是一个人一辆马车吗，怎么还有别的人？”
<br/>　　翠云指着马厩里一辆最大的马车说道：“是这个样子的马车啦，八个人一起坐的，你说的那种一人一匹的也有，不过太贵了，我就没租，这种大马车就是到皖南边界的，人齐就可以出发啦。”
<br/>　　叶少阳顺着她目光看去，这辆马车大是很大，但是很破，有四匹马拉着，看上去远不如边上那些小马车精致美观。
<br/>　　叶少阳问道：“价格相差的很大吗？”
<br/>　　翠云道：“是啊，大马车一个人一块大洋，小马车三块，贵不少啊。”
<br/>　　叶少阳无奈，说道：“你不用给我省钱啊，我现在也不是没钱。”
<br/>　　翠云道：“需要省的，将来你要娶媳妇，多留点钱，可以多买些房子和地，将来会过的好一点。”
<br/>　　叶少阳无语。
<br/>　　“那个，我走之后，你好生过日子，有什么困难就去找秋镇长，他肯定会帮忙的。”临别之际，叶少阳想要多嘱咐几句，谁料翠云摇了摇头。
<br/>　　“嗯？”叶少阳说道，“你不要不要意思，他答应我会照顾你的。”
<br/>　　翠云抿嘴一笑，仿佛是下了什么决心似的，说道：“我不需要任何人照顾了，我决定了，我要跟你走。”
<br/>　　叶少阳当场怔住。“跟我……走？”
<br/>　　“对，跟你走。这一路这么远，你身边没个女人照顾，我不放心的。”
<br/>　　“这……”叶少阳挠着头。
<br/>　　翠云黯然低头，说道：“你要是不想带着我，那就算了。”
<br/>　　叶少阳摆手说道：“那倒不是，跟你在一起挺舒服的，你很会照顾人，只是……这孤男寡女的，似乎有点……”
<br/>　　翠云笑道：“你想什么呢，可不许误会我，我要做你姐姐，我以后就把你当成弟弟了。我对你任何照顾，你都不需要多想。”
<br/>　　姐弟……叶少阳有点吃惊，不过内心还是很感动，点头说道：“我是没问题，只是……你家里房子什么的不要了？”
<br/>　　“就一套破房子和几亩薄田，不要也罢。我陪你去找你媳妇，将来你成家了，我可以给你管家，如果你到时候不要我，大不了我再回来就是了。”
<br/>　　叶少阳笑道：“你这样跟我走，别人会以为你跟我私奔的了。”
<br/>　　“随便吧。我反正决定了。”翠云深吸了一口气，“我想了一个晚上呢，走就走，我也很想去外面看看呢。”
<br/>　　叶少阳心中很是感动，同时内心深处是希望她离开小镇，看看外面的世界，这样就不用过那种苦行僧似的寡妇生活，更不要说翠云是自己救命恩人，又照顾自己那么久，当下凝视着翠云的双眼，说道：“我也没有姐姐，从今往后，你就是我姐姐。”
<br/>　　翠云咧嘴笑起来。
<br/>　　等那两个客人来了之后，车行的人开始招呼他们上车。
<br/>　　一共八个客人，除了有一对是夫妻，其余都是男的，打扮都很朴素，在上车之前，翠云就跟他说过，经常走远途的，都是做买卖的，把这边的东西卖去外地，然后再把外地特有的商品采购一些回来，卖给当地人。
