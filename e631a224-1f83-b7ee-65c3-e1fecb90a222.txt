　　在每一个夹角之中，都有一根竹子，伸到中间去。如果从高处俯瞰下去，这个水池就像是一个八角形的巨大的车轮。
<br/>　　在这个水池的中间、八根毛竹撑着的地方，那看上去像是“车轮”轴承的位置，放着一口古铜色的钟，有两米来高，上面全是符文，钟的四周，荡漾着一层层的妖气，与碧绿的潭水结合，仿佛一块炙热的烙铁，将池水煮沸，不断散发出水蒸气，却不消散，而是围绕着这口钟旋转，不断注入钟的表面上。
<br/>　　这就是东皇钟。上古神器，东皇太一的遗物……
<br/>　　林三生站在附近，呆呆地看着。
<br/>　　虽然自从法坛搭好之后，自己一直守在边上，但是近距离地打量这口东皇钟，还是会有一种很奇怪的感觉，林三生说不上来。
<br/>　　东皇钟的正面，那一块如同铜镜一样明亮的地方，映照出了一道艳丽的人影，几乎是半透明的，完全看不清，不过，在水蒸气弥漫的空隙中，能依稀看见女子的相貌，倾国倾城，天下无双。
<br/>　　林三生一直盯着她看，倒不是贪图她的容貌——他心中只有婉儿，退一步说，就算没婉儿，他也不会为东皇钟上这个女子动心，朋友妻不可欺，林三生饱读诗书，这个道理还是严格恪守的。
<br/>　　他之所以一直盯着看，是期望上面影像越来越清晰，也就说明神念越来越凝聚。
<br/>　　他希望看到她神念重聚，虽然魂魄和肉身还没有来历，不过这总算是第一步成功。
<br/>　　东皇钟上的女子，当然就是九尾天狐。
<br/>　　不知道少阳看到她元神重聚，是什么样的心情……林三生经常这样想，然后，再想到自己的处境，内心再度挣扎起来，他舍不得婉儿，但是他又迫切地想要去跟叶少阳和一干好友重聚，他也放不下他们，自己在这里跟婉儿每天你侬我侬的，叶少阳却不知道在民国遭受着什么样的局面，他于心不安，总觉得自己是叛徒。
<br/>　　“军师……”
<br/>　　一个声音，懂面前的东皇钟方向飘来，林三生一惊，抬头看去，却看到白雾消散，东皇钟上小九的影像更加明亮真实起来，而且已经睁开了眼睛看着自己。
<br/>　　“小九，你醒了？不，你……恢复元神了？”林三生十分激动，能听见小九的声音，这说明徐福的法子成功了，虽然只是第一步。
<br/>　　“感觉好像重生了一样。”小九喃喃说道，然后透过东皇钟，看着外面的世界，皱眉说道：“这是什么地方？”
<br/>　　“这是……”林三生一时语塞，他也不知道这是什么地方，想了一下，说道：“徐福老祖找了一个清静的地方，让我布置法坛，帮助你聚拢元神。”
<br/>　　“哦……”小九沉吟起来，感受了一下周围的环境，从外面看，她是出现在东皇钟上的一抹影像，就好像电视屏幕一样，不过在小九自己感受，自己就好像漂浮在一片无尽的虚空之中，浑身不能移动，甚至没有什么感知，自然也谈不上运功之类的了。
<br/>　　“少阳呢？”
<br/>　　“这……”
<br/>　　小九看他的表情，忍不住一惊，说道：“怎么了，快说，少阳在哪！”
<br/>　　林三生于是把星宿海一战后续的情况详细告诉了他。
<br/>　　小九听说叶少阳被送到了民国时期，有些担心，不过也是松了一口气。“很好呢，至少少阳还活着。我也没有白死一场……”
<br/>　　林三生道：“小九你不要难受，徐福老祖既然费了这么大力气帮你聚拢元神，一定有办法帮你复原，说不定连修为也可以恢复，还能继续做你的妖王。”
<br/>　　小九点点头，她知道，自己内丹并没有碎裂，现在就在东皇钟里，正是借助这一点内丹之力，自己的元神才能保存下来……
<br/>　　“那你呢，你为什么没有去追随少阳，反倒在这里？”
<br/>　　“我……”林三生一时语塞，只好把自己情况说了一遍，当日，在星宿海一战开始之前，徐福找到他，就告诉了他一切，当时为了跟婉儿在一起，也为了在最后时刻帮叶少阳一把，他答应了徐福，之后徐福兑现承诺，带他去找到了婉儿……
<br/>　　林三生说完，自己有些尴尬，不太敢看小九，小九却是没太在意，思索了一下说道：“我觉得，你可以先带着你的婉儿一起去见少阳，如果他有什么困难，帮帮他，结束之后，你可以带着你的婉儿离开，继续回来这里隐居。军师，少阳需要你。”
<br/>　　需要我……
<br/>　　林三生眼前，浮现出了当初与叶少阳最初相识，叶少阳帮自己各种寻找婉儿、安排一切的局面，又想起了之后那一次次战斗，他与叶少阳和其余捉鬼联盟成员一起并肩作战的场景，如今，叶少阳一行人被困在民国，虽然不知道在干什么，但是按照徐福的说法，肯定是在经受什么磨难，甚至有生命危险。
<br/>　　徐福带他来是有自己的目的，不过至少有一点，就是渡劫。渡劫这种事，林三生当然是明白的，生死都只在一念之间而已……
<br/>　　小九的话，帮助他做出了最后的决定，当下冲着小九微微一笑，点点头。
<br/>　　对这一切的一切，不管是发生在什么年代的事，叶少阳都毫不知情，对他来说，只是好好睡了一个晚上，一早起床，洗漱之后，他去隔壁敲门，叫毛小方和妙心起床。
<br/>　　“等我们吃完早饭，这就出发去古墓？”叶少阳问妙心。
<br/>　　“要等我几个朋友到了才行，光凭咱们的实力，恐怕有危险。”
<br/>　　“你还有朋友要来？”叶少阳听了她的话，很是有点吃惊。
<br/>　　曹雨兴这时候来劲了，挑着眉毛说道：“当然了，妙心姑娘的朋友，那可是了不得，这次有陈晓宇和卢晓清要来，因为在上海有事情没办成，所以特意让我先送妙心姑娘过来。”
<br/>　　“陈晓宇和卢晓清？”毛小方微微吃惊，皱起了眉头。
<br/>　　“不错。等这二位师兄来了，事情就容易多了。”曹雨兴得意地说道。
