　　特招生三个字，很容易使人浮想翩翩，比如关系户，不管在任何地方，关系户这一类人都是存在的。

<br/>　　慕清霄清楚他的意思，只不过后者的眼神让他有些不舒服，脸上的笑容也消失不见，眼神冷漠无比。
<br/>　　“管好自己的眼睛，不该看的就不要看，否则...”
<br/>　　“呃！”
<br/>　　冰冷不含一丝情绪的话语声响起，彻骨的寒意席卷全身，男子汗毛炸立，额头上瞬间冒出一层冷汗。
<br/>　　吞咽了口唾沫，急忙将视线自小医仙身上移开，惊恐的看了一眼慕清霄。

<br/>　　当然，这都是那些导师心目中的要求，在他们看来，这种天赋已经非常不错。
<br/>　　不过，慕清霄看来，除去古薰儿，经过淬体改造的小医仙天赋稍微好一些，其余的都算个什么东西？
<br/>　　而且，这一路上，男子的目光一直盯着小医仙，现在又来套近乎，绝对是道貌岸然的伪君子。
<br/>　　若是在外面，慕清霄估计早就出手废了他，但这里毕竟是迦南学院，他的实力，暂且不能横着走，并不想多生事端。
<br/>　　不再理会身后的小丑，慕清霄视线落在房间内的能量罩上，周围除了能量罩，再也没有其他东西了。
<br/>　　就在他茫然之际，细微的破风声陡然在房间内响起，虽然声音细微，但在这寂静的房间里，却显得极为清晰。
<br/>　　“好像...有东西来了，不会从里面冒出来吧？”小医仙望着能量罩，疑惑的说道。
<br/>　　“这里就只有能量罩，当然得从里面出来。”
<br/>　　慕清霄话语刚落，只见那能量罩中，一团青色光芒爆射而出，旋即化作一道流光，朝着众人疾射而来。
<br/>　　速度虽快，但在他的眼中，就像是蜗牛在爬，慢的不可思议，轻轻抬起手掌，就将青光抓在了手中。
<br/>　　青光消散，最后一卷被能量包裹的青色卷轴出现在手中。
<br/>　　“玄阶高级功法，青元决。”
<br/>　　话语刚落，身后几人也回过神来，东西果真是从里面冒出来啊，第一种就是玄阶高级功法，学院底蕴当真深厚。
<br/>　　就在他们愣神之际，一阵阵破风声再次响起，紧接着一团团色彩缤纷的能量团，铺天盖地的从能量罩中喷射而出。
<br/>　　能量团速度极快，有的甚至快的留下一串残影，随后在巨大的房间中来回穿梭，破风声绕耳不绝。
<br/>　　望着满屋子的能量团，慕清霄将手中的功法松开，青色能量团就成为了能量大军中的一员。
<br/>　　观察片刻，慕清霄又随手拿起一个，右手侵入能量团中，随后就显出一枚碧绿色的果子。

<br/>　　玉朱果，六品高级灵果，能够直接服用，效果能够直接令一名斗皇强者，瞬间提升一星左右。
<br/>　　这种高级灵果极其难寻，哪怕是有，也是有价无市，因为根本没人愿意拿出来卖，可见价值有多么的珍贵。
<br/>　　瞥了一眼玉朱果，慕清霄并没有取出，随手扔进能量团大军中，其中肯定有更好的东西。
<br/>　　斗技功法，或者宝甲，对他来说都没用，所以慕清霄的目标就只有灵药，或者是有价值的丹药了。
<br/>　　“仙儿，自己去挑一个吧，我再看看，其中有没有别的好东西。”望着身边的小医仙，慕清霄温柔一笑，随后目光就在能量团中搜寻起来。
<br/>　　“嗯。”
<br/>　　闻言，小医仙乖巧的点了点头，随后美眸也在屋内搜寻起来。
<br/>　　经过实验与观察，慕清霄发现，能量团传出的波动越大，其中的东西就越是珍贵。
<br/>　　“咻咻...!”
<br/>　　几分钟后，能量层中，猛的响起一道响亮的破风声，慕清霄目光一凝，这是一刻钟下来，最响亮的声音。
<br/>　　周围的能量团，与之比起来，简直是小巫见大巫。
<br/>　　不仅是他，连其他学员也都能听见这明显的破风声，眼中满是激动，身躯紧绷，随时都准备争抢。
<br/>　　破风声越来越响亮，只听见‘咻’的一声，一道金色能量团，瞬间冲出能量罩，以肉眼根本捕捉不到的速度，乱窜起来。

<br/>　　慕清霄低喃一声，这种速度在他看来，已经不是快的程度，而是一种境界，别说其他学员，哪怕是他想要追上，都必须集中精神。

<br/>　　在他的视线中，一团金色能量团，以迅雷不及掩耳的速度直射过来，恐怕斗宗级别的碰到，估计都能一下子撞懵。
