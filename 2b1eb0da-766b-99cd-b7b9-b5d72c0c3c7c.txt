　　第686章
<br/>　　程大雷摸了摸鼻子，刚才的郁闷已变得不太重要。在这一刻，他竟能读懂崔白玉的心意。
<br/>　　将心比心，若是自己落到她这般境地，能不能硬着头皮说一句，来呀，杀我呀，二十年后老子依旧是一条好汉？
<br/>　　估计悬。
<br/>　　程大雷拍拍高力士的肩膀，道：“松了她们的绑绳，给她们送饭菜进来，等下送到城主府。”
<br/>　　高力士一惊：“大当家，如此做万一她们再寻死？”
<br/>　　程大雷摇了摇头，估计崔白玉不会再有寻死的心。引刀成一快的确慷慨，但不负少年头的能有几人。至于活下去的理由，她自己会找到的。
<br/>　　程大雷和高力士退出房间，门口的守卫在为小鹿三女松绑之后，也撤出了房间。
<br/>　　第一时间，三女扑到崔白玉身边，将狼狈的她扶起。看着她此刻模样，三女比自己受辱还要难受，主辱臣死，平日哪里见到过崔白玉露出这般模样。
<br/>　　“郡主，郡主……”
<br/>　　唤了好几声，崔白玉才回过神，她抬头四顾，竟有些茫然。
<br/>　　“郡主，我们带你逃出去？”小刀道。
<br/>　　“逃，又往哪里逃呢？”崔白玉失神道。她虽然沮丧，但脑子还在，凉州城已是程大雷的凉州，凭她们几个女子想从程大雷的魔爪下逃生，无异于痴人说梦。
<br/>　　小鹿也忍不住掉下眼泪，她何尝不知这件事的难度，只是不忍心看到自己主人露出如此模样。斗志、勇气、稳操胜券的镇定自若现在都消失了，杀人易，诛心难，现在崔白玉一颗心已经是七零八落。
<br/>　　“成与不成，总是要试一试，落在那魔头手中……”小鹿已经不忍说下去。
<br/>　　其实她说这种话已经有些越级，因为她在与崔白玉商量。这在从前是难以想象的事情，崔白玉可以决定诸多事物，算无遗策，怎么会同一个丫鬟商量。
<br/>　　但现在失魂落魄的崔白玉肯定没有意识到这一点，如今也并不是从前了。
<br/>　　这时有守卫从外面进来，粗鲁的将食盒放在桌子上。这些守卫都是女人，自然不会在意崔白玉的美貌，对她的态度自然也谈不上有多好。
<br/>　　小鹿三女各自占据一个位置，封锁了来人的退路。她们是被相府精心培养出来的，刺杀、下毒，甚至如何诱惑男人都是极其擅长的。即便几日没吃过饭，饥肠辘辘，想杀死这名守卫也不费多少力气。
<br/>　　她们在等待，等待崔白玉点点头，便要杀人。
<br/>　　崔白玉没有点头。小鹿三人是幸运的，因为崔白玉是她们的主人，无论怎样的绝境，只要主人活着，她们就有存在的价值。可崔白玉是不幸的，她是自己的主人，她心中的信仰已经被程大雷击碎。
<br/>　　那守卫放下饭菜后离开，临走前还冲崔白玉啐了一口，嘴里嘟嘟囔囔，说些狐狸精之类的话。而她浑然不知，自己从鬼门关前走过一遭。
<br/>　　崔白玉目光落在那食盒上，守卫的态度虽然不如何，但在伙食上并没有亏待自己。如今饭菜散发出诱人的香气，对饥肠辘辘的崔白玉来说，更是致命的诱惑。
<br/>　　不必崔白玉开口，小鹿已经将一碗饭端到她面前。做下人的就要有这样的眼力见，不能等主人开口时才献上，要主动给主人台阶下。
<br/>　　崔白玉将食物吞入口中，丰富的食物瞬间占满味蕾，她第一次感觉，活着原来是如此幸运的事。随之，便有些悲哀，感觉自己满可怜的。
<br/>　　扬名长安的白玉美人，竟为一碗白米饭而感到幸运了。
<br/>　　……
<br/>　　程大雷回到了城主府，心里还寻思着刚才发生的事。宋伯康的王府委实太大了，比蛤蟆城大出十倍还富裕，走在里面空空旷旷的，的确有些冷清。
<br/>　　他找到了苏樱，苏樱坐在账房，正处理最近的账务。程大雷辛苦，苏樱其实也没有闲着。
<br/>　　这次占了凉州城，除了抢到一些女人，也抢到一些金银珠宝。这样事都要由苏樱处理，将这些金银珠宝收入她的小金库中。
<br/>　　花也花不出去，也就过一过小富婆的瘾头。
<br/>　　程大雷走到门前，先往里面张望一眼，见苏樱正在紧锁眉头，正在扒拉着算盘。程大雷鼓起勇气，大笑着踏入房中。
<br/>　　“夫人辛苦，夫人辛苦……”
<br/>　　苏樱抬起头，目光平静的望着程大雷。完全是看傻瓜的眼神，直接就是说：准备做什么对不起我的事，直说无妨。
<br/>　　程大雷无语的坐下，他和苏樱的关系闹到这步田地，未必没有苏樱太过透彻的缘故，这个女人太聪明了，做什么缺德的事也瞒不住她。
<br/>　　“那个……这个……夫人一向辛苦，我也是看在眼里，疼在心头。夫人身边本该有个伺候的人，这不正好，有一批女眷不知如何处置，我也烦恼。不若夫人去拣几个看得过去的人，身边有人照顾我也放心不是。”
<br/>　　苏樱也并不是傻瓜，相反很是聪明。程大雷闪烁其词，她早已明白他的心意。心中又是好笑又是好气，笑的是这种事程大雷本不用和自己商量，气得是程大雷为什么要同自己商量。难道还想自己鼓掌支持不成。
<br/>　　她心里有些不舒服，按她原先的性格，就该点头应承。可忽地想起杨小妹同自己说过的话，苏樱鼓起勇气道：“假若我不许呢？”
<br/>　　程大雷脸上的笑容僵固住，愣了半晌无可奈何道：“便知道你不会答应，不许就不许吧……唉，真的不可以商量商量么？”
<br/>　　苏樱无语的看着程大雷，这人终究不坏，换做旁人，有他现在的地位怕早已妻妾成群，又何必同自己商量。更不必露出如此委屈的模样。
<br/>　　多多少少，自己在他心中还有一席之地。为此，也不枉自己跟他一场。
<br/>　　她忽地大起胆子，屈指在程大雷额头敲了敲，道：“我的好大王，为妻便替你走这一遭，只盼你莫要喜新厌旧。要记得我当初可是被你抢上山的，抢我上山，可不是为了摆着好看。”
<br/>　　程大雷揉了揉眉心，忽觉得自己和苏樱之间横搁的冰山，似乎有了一道裂缝。
<br/>(本章完)
