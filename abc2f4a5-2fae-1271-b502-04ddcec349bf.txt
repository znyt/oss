　　有的学员先发制人，率先将其他学员踢出场地，有的学员则掠到场地边缘，生怕被第一个踢出。
<br/>　　战斗一触即发，看台上就响起一道道激动的尖叫声，响彻云霄。
<br/>　　许多的学员则是扯着嗓子给自己崇拜的参赛者加油助威，显然，这种比赛模式比回合赛更令他们兴奋。
<br/>　　学院高层专用看台上，一名老者道：“副院长的注意真是妙，世上没有绝对的公平，混战更容易培养他们的警惕性，联手的学员也会懂得团队合作的好处。”
<br/>　　“我也只是厌烦了回合赛，换种模式尝试一下，现在看来，效果貌似不错。”
<br/>　　琥乾笑眯眯的说了一声，随后就将目光投入混乱的场地中。
<br/>　　此时，比赛场地中，场面混乱至极，时不时就有人相互摩擦。
<br/>　　但是，明显许多学员都害怕背后的黑手，即使是与学员交手，也仅仅是一击即退，完全不敢缠斗，目光则谨慎的扫向周围。
<br/>　　任何闯进他们攻击范围的学员，都会令他们犹如惊弓之鸟，直接反击或者爆退几步。
<br/>　　虽然比赛才刚刚开始，赛场中极其混乱，待一些智商堪忧的学员被打的吐血飞出场外，其他学员也就学聪明了些。
<br/>　　只要场内有认识的，学员们都会豪言拉拢，若是没有，也只能寻找落单的。
<br/>　　虽然临时组合出的团队默契度与信任都很易碎，却也是现在唯一的办法。
<br/>　　斗气如灰色匹练，缭绕在周身，小医仙站在赛场的边缘，望着挡在自己身前的几名学员，柳眉皱在了一起。
<br/>　　以她现在的实力，哪怕得不到第一名，第二也绝对不是问题。
<br/>　　场面混乱，小医仙美眸瞥了眼对面的古薰儿与萧炎，以及其他组成团队的学员，顿时有些无奈。
<br/>　　古薰儿实力深不可测，刚来学员的萧炎恐怕也不好惹。
<br/>　　她现在独自一人，吃亏的肯定是自己。
<br/>　　混乱的场面并没有维持太久，当白山，吴昊他们各自组成团队，场中的混乱就变成了几强并存的局面。
<br/>　　此时，混乱结束，但谁也不敢轻易出手。
<br/>　　伴随着看台上震耳欲聋的吆喝声，比赛渐渐进入白热化阶段，较弱的学员被不断踢出场外，场内人数开始缩减。
<br/>　　时间在漫天高呼声中流逝，小型团队越来越少，大型团队则越来越强横。
<br/>　　萧炎与古薰儿也纷纷出手，展现出大斗师强横的实力压迫，直接将攻来的小团队横扫出场。
<br/>　　小医仙则长袖一挥，灰色斗气环绕四周，瞬间有人察觉到了一丝不对劲，脸色从正常化作紫色，明显是中毒的迹象。
<br/>　　回过神来，其中一名男子盯着小医仙，面露惊惧。
<br/>　　“你……你是毒修！”
<br/>　　望着躺在地面，脸色紫青口吐白沫的学员，男子吓得脸色惨白，当下撒腿冲出她的攻击范围内。
<br/>　　其他学员也是骤然色变，看台上围观者们的欢呼声戛然而止，气氛一瞬间变得有些压抑起来。
<br/>　　原本，某些倾慕她的学员脸色狂变。
<br/>　　各各目光复杂无比，其中含有一丝惊惧，甚至闪过一抹厌恶之色，在斗破世界中，毒修不被人理解。
<br/>　　“毒修……学院中怎么会有毒修！”
<br/>　　“难道是出云帝国派来的奸细？”
<br/>　　“本姑娘早就看她不顺眼，原来是毒修，见不得光，难怪平日里都用薄纱掩面。”
<br/>　　看台上议论纷纷，学员们开口咒骂，学院高层的看台上，各位长老脸色微变。
<br/>　　至于琥乾，依旧坐在板凳上，脸色一如既往，并未有所动容。
<br/>　　“她是我当初推荐进内院的学生，由于境界不够，她自动选择离开内院，来到外院修炼，如果她是尖细，那老夫岂不是与她一样？”
<br/>　　说着，琥乾语气平淡道：“她是迦南的学生，哪怕是毒修又如何？”
<br/>　　一时间，高层看台上陷入沉默，长老们也不好在说什么。
<br/>　　虽然看台上咒骂声一片，但并未影响比赛的继续，只是小医仙却独自一人站在赛场的边缘，其余学员则抱成一团。
<br/>　　萧炎瞥了眼小医仙，将目光收回，看向古薰儿道：“你早就知道她是毒修？”
<br/>　　“不错，所以才让萧炎哥哥遇到她要小心，她的实力可完全不比白山他们弱。”
<br/>　　此时，场地中的一角，白山目光阴郁的早场内扫视一圈。
<br/>　　无论是吴昊，或者是古薰儿都不好惹，琥嘉也是如此，场地中的人自然是越少，对他们越有利，于是目光就看向了孤零零的小医仙。
<br/>　　后者似乎并不准备站队。
<br/>　　只见白山手掌一挥，身旁八名不弱的参赛者与他一起向小医仙包围过去，看样子似乎是准备动手了。
<br/>　　小医仙抬起臻首，看向朝自己包围过来的白山等人，薄纱低下的脸色不变，柔美的声音中参杂着一抹冰冷。
<br/>　　“怎么，你们想动手？”
<br/>　　白山手掌一握，一根通体银白的长枪闪烁而出，枪身流转着淡淡银光，一看就知道是一根镶嵌着魔核的武器。
<br/>　　“你们八个守着，别让吴昊他们有机可乘，让我先将她解决了。”
<br/>　　听到白山的命令，其身后八人顿时分散开来，硬是挤出一片空间，留给白山与小医仙。
<br/>　　望着杀向小医仙的白山，萧炎手掌轻轻握住巨尺，却被一只玉手拦下。
<br/>　　“熏儿？”
<br/>　　只见古薰儿微微摇头，道：“萧炎哥哥不必担心，你们都小看她了，区区白山根本就不是她的对手，我们先解决其他人。”
<br/>　　在白山对付小医仙的时候，萧炎与古薰儿，吴昊与琥嘉等队伍都开始清扫杂鱼。
<br/>　　场面再度变得混乱起来，团队与团队间的较量场面终于珊珊来迟，这一刻周围看台上，尖叫呐喊的助威声宛如雷鸣。
<br/>　　望着攻击越来越凌厉的白山，小医仙体内一缕缕灰色毒气流转，最后沿着经脉狂涌而出，恐怖的气息提升到了巅峰。
<br/>　　二星斗灵！
<br/>　　灰色毒气肆虐，令得看台上的围观者们惊呼出声。
<br/>　　甚至有人吓的不断往后退，生怕被毒气波及。
<br/>　　白山脸色狂变，阴郁的目光变得阴沉如水，咬牙道：“小医仙姑娘，你隐藏的可真深，当初听闻你是内院出来的，现在我信了。”
