　　萧逸一下子成了众矢之的。
<br/>　　“那家伙不过是先天五重修为，却能打败长风公子和不群少爷。”
<br/>　　“也只可能是拥有地阶武技了。”
<br/>　　一些武者边自语着，边蠢蠢欲动起来。
<br/>　　“听说这个萧逸毫无背景，也没有任何势力。”
<br/>　　“就算我们....”
<br/>　　这个武者的话没有说下去，但谁都明白他的意思。
<br/>　　萧逸皱起了眉头，他知道地阶武技对于武者来说是多大的诱惑力。
<br/>　　“哈哈哈哈。”旁边的顾长风得意地大笑起来，“萧逸，我知道我不是你的对手。”
<br/>　　“但，那又如何？”
<br/>　　“我打不过你，你又能在这数千武者的围攻下活命吗？”
<br/>　　顾长风一副阴谋得逞的样子，似乎已经能看到萧逸在数千武者围攻下竭力而亡的画面。
<br/>　　“你很得意？”萧逸冷冷地看着他。
<br/>　　“是又如何。”顾长风冷笑一声，“我已经迫不及待要看你瑟瑟发抖的样子了。”
<br/>　　“恐怕，你没那个机会看了。”萧逸脸色一冷，一把捉住他的脖子。
<br/>　　而后，身影一闪，瞬间消失在原地。
<br/>　　“嗯？”数千武者顿时着急起来，“千万别被他跑了。”
<br/>　　数千武者，自发地包围了起来，方圆百米之内，一只苍蝇也飞不出去。
<br/>　　下一秒，萧逸的身影出现。
<br/>　　他没有逃，而是来到赵不群的身边。
<br/>　　“你想做什么？”赵不群顿时一惊，面露恐慌。
<br/>　　萧逸一把夺过他和顾长风的令牌，将两人的分数夺光。
<br/>　　赵不群见自己的分数被夺光，似乎预料到萧逸下一步想做什么。
<br/>　　“不...不要杀我。”赵不群求饶着，装出一副痛哭流涕模样。
<br/>　　他很清楚，自己在萧逸手上一招都接不住，萧逸想杀他，易如反掌。
<br/>　　而且，如今令牌在萧逸手上，他连释放禁制保命都做不到。
<br/>　　堂堂百武城大家族之一赵家的少家主，北山榜14的天才，如今竟跪地求饶。
<br/>　　只不过，在萧逸看来，他的求饶很是虚伪。
<br/>　　这时，一直被萧逸扣住脖子的顾长风冷声道，“赵不群，求他做什么，他不敢杀我们的。”
<br/>　　“是吗？”萧逸的声音，忽然变得极度冰冷。
<br/>　　那双原本清亮的双眸，变得杀意四溢。
<br/>　　几乎是一瞬间，一股澎湃得可怕的杀气在空气中快速凝聚。
<br/>　　就连原本包围着他的武者，也不禁打了个冷颤。
<br/>　　“好恐怖的杀意，这个家伙到底是过多少人。”一些武者开始不自觉地咽了咽口水。
<br/>　　而就在萧逸身旁的顾长风和赵不群，对这股杀意的感觉更为清晰和直接。
<br/>　　有那么一瞬间，他们仿佛看到了面前的少年，成为了一个满手血腥的侩子手，让人胆战心寒。
<br/>　　“不...不要杀我。”赵不群没有发现，自己的声音哆嗦了起来。
<br/>　　顾长风甚至眼泪鼻涕都流了出来。
<br/>　　这两人，看似心狠手辣，但其实心理承受力极差，只是两个脓包。
<br/>　　“你...你这个怪物，去死吧。”顾长风宛若失去了理智般，手中风吟剑全力刺出。
<br/>　　萧逸皱了皱眉头，轻松夺过风吟剑。
<br/>　　而在看到顾长风那丑恶的嘴脸和不堪的模样时，那鼻涕差不多要滴到自己手上，萧逸脸上一阵厌恶之色。
<br/>　　“滚。”萧逸冷喝了一声，重重将其甩飞。
<br/>　　“你也滚。”萧逸将赵不群也一脚踹飞。
<br/>　　随手将风吟剑放入自己的乾坤袋，萧逸冷眼瞥了一眼包围住他的武者。
<br/>　　而后，他的脚步动了，嘴中喃喃自语着，“最好别逼我杀人，我手上的性命，足够多的了，不差你们这数千条。”
<br/>　　一些武者开始后退，脸上尽是踌躇之色。
<br/>　　一方面是对地阶武技的狂热，一方面是对刚才浓郁杀气的惊怕。
<br/>　　但，数秒后，人性的贪婪，还是战胜了恐慌。
<br/>　　“不要让他跑了。”一个武者大喝一声，并站了出来。
<br/>　　人便是这样，有人带头，立马紧跟其上。
<br/>　　“要么留下你身上的宝物，要么留下你的性命，自己选吧。”
<br/>　　数千武者，立刻扬起了兵刃，齐刷刷指向了萧逸。
<br/>　　萧逸停下了脚步，淡淡地自语着，“你们，非要逼我大开杀戒吗？”
<br/>　　话音刚落，他理了理自己有些凌乱的衣衫，显然，他已经起了杀心，要动真格了。
<br/>　　恰在此时，四道人影从人群中奔袭而来。
<br/>　　“找死。”萧逸冷哼一声。
<br/>　　当他看到来人竟是林劲四人时，眼中闪过一丝失望。
<br/>　　但下一秒，他眼中的失望，很快便褪去了。
<br/>　　只因，秦飞扬、林劲、铁牛和柳嫣然来到他面前后，毫不犹豫地转过了身，背对着他。
<br/>　　他们，竟将毫无防备的后背，交给了萧逸。
<br/>　　这代表的，是无比的信任。
<br/>　　“你们这群家伙。”萧逸发自内心地笑骂了一声。
<br/>　　四人都背对着萧逸，手中兵器冷冷地指向了周围的数千武者。
<br/>　　“哈哈。”林劲也笑骂一声，“小气的家伙，以为我们也要抢你的武技吗？”
<br/>　　萧逸笑道，“我想过，但我又相信你们不会这样做。我也犹豫过，幸好，你们没有让我失望。否则....”
<br/>　　“否则什么？”林劲问道。
<br/>　　萧逸淡笑道，“否则，若是连你们三个奇葩都是那样的人，我会对整个世界都失望掉的。”
<br/>　　“你个混蛋。”林劲差点没笑岔气。
<br/>　　铁牛憨笑道，“萧逸兄弟，我还以为你要说什么，把俺吓一跳。”
<br/>　　柳嫣然美眸闪烁，嗔怒道，“萧逸，你竟然怀疑我，我很伤心喔。”
<br/>　　秦飞扬甩了甩头发，傲然道，“本公子何等人物，会抢你的东西？切！如果你是美女的话，本公子倒是可以考虑劫...个色”
<br/>　　“呵呵。”众人同时一笑。
<br/>　　“好了。”萧逸笑笑，道，“你们不用掺和此事，我承你们的情，这里交给我就行。”
<br/>　　“不，你们先走。”林劲、铁牛、柳嫣然三人异口同声说道。
<br/>　　说罢，三人又同时相视而笑。
<br/>　　三人似乎都对自己充满了信心，一人就可对付这数千武者。
<br/>　　萧逸笑笑，他一直都知道，这三人都隐藏了实力，却又互相没看出来。
<br/>　　“既然都不愿走，便和他们战个痛快吧。”萧逸说着，和四人并肩站到一起，冷眼看着那数千武者。
<br/>　　“秦飞扬，一有意外，立刻驮大伙离开，我会拦下他们，让你们安然离去。”萧逸吩咐了一声。
<br/>　　这时，数千武者渐渐开始围拢，包围圈越缩越少。
<br/>　　“遗言都交待好了吗？”
<br/>　　“杀一个是杀，杀五个也是杀，没差。”
<br/>　　五人对三千，不难想象，战斗将激烈到骇人的地步。
<br/>　　只不过，下一秒，情况竟再次开始转变。
<br/>　　三千武者中，又有数十、近百道人影快速跃出，来到萧逸身边。
<br/>　　正是花剑、遁地鼠、秃鹰、猎妖师王虎等人的小型队伍。
<br/>　　全是之前萧逸等人抢了分数，却又饶过他们，没有逼他们认输的十几支小型队伍。
<br/>　　“萧逸，我不知道你是否有地阶武技，但，我不喜欢欠别人人情。这一次，就当还你半月前的人情。”花剑桀骜地说道。
<br/>　　“萧逸，半月前你放我一马，今日混战中，我已经抢回了不少分数，足够通过考核的了，谢啦。”遁地鼠何苍说道。
<br/>　　秃鹰则道，“虽说如今已经抢回了不少分数，但，我不介意继续多抢些。恰好，萧逸你的敌人似乎比较多，够我抢的了。”
<br/>　　“.....”
<br/>　　众人半开着玩笑，却毫不犹豫地表面了自己的态度。
<br/>　　无论是花剑、遁地鼠、秃鹰、还是王虎，以及队伍中的其它武者，皆是桀骜不羁之辈。
<br/>　　他们就是因为骄傲，才不加入那些大型团伙，哪怕很可能会因此而考核失败，无缘成为裂天剑派弟子。
<br/>　　如今，他们也是因为骄傲，虽然不知萧逸是否真的有地阶武技，但绝不做那等卑鄙之事。
<br/>　　萧逸、秦飞扬、林劲、铁牛、柳嫣然、花剑、秃鹰.....
<br/>　　这近百人，都是这次考核中修为实力处于最上层的一批武者，联合起来，毫无疑问将是一支最强队伍。
<br/>　　萧逸上前一步，冷眼看着那三千武者，淡淡地问道，“你们，准备好与我一战了吗？”
