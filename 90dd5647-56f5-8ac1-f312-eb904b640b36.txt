
<br/>　　<br/>　　    来这一群年轻人质量普遍都挺高的，要用菜刀文的话来讲就是，在KTV点这样的台起码得要四位数起步，这是标准的模特组啊。<br/>　　<br/>　　    四个女的两个男的，青春靓丽组合，姑娘全都是短裤体恤，小伙子牛仔裤配衬衫和短袖，特别是其中一个女人身材嘎嘎高挑，肤白貌美，背着双肩包，大长腿好像从肚脐眼以下就开始分叉了。<br/>　　<br/>　　    一个长得挺乖巧可爱的女生蹦跳着就来到徐闯身边，伸手就挽上了他的胳膊，俏皮地眨了眨眼睛问道：“等多久了？”<br/>　　<br/>　　    “我也是刚到一会，没事”徐闯跟他女朋友说话的时候，脸上明显洋溢着满足的表情。<br/>　　<br/>　　    两人打完招呼，徐闯就指着王惊蛰说道：“和我住在一起的朋友，听说我们来天目山，也想过来逛逛的，他叫王惊蛰”<br/>　　<br/>　　    徐闯的女友朝着他点了下头，说道：“人越多越热闹，那就一起吧”<br/>　　<br/>　　    “打扰了……”王惊蛰笑着回应了一句。<br/>　　<br/>　　    在进站口汇合以后一群人就往车站里走，在自动售票机上买了几张去临安的动车票，半个小时后上车。<br/>　　<br/>　　    等着检票的时候，王惊蛰就安静地站在一旁看着他们聊天，从几人的交谈中他听出来了，这些都是理工大学的学生，徐闯的女朋友叫王清雪，那个肚脐眼以下就分叉了的女人有个很诗意和充满了武侠味的名字，叫李红拂，还有个长得跟权志龙似的嘻哈青年一直都围在她身旁，总得说来这就是一群洋溢着青春荷尔蒙的少男少女们。<br/>　　<br/>　　    本来王惊蛰跟他们的年岁也差不了多少，但站在一旁的他就显得跟这些人有点格格不入了，性子太沉稳，穿着一身长褂，只是没带上那包菜刀，看起来他的装扮就跟个八十年代的知识分子一样，充满了沉闷和腐朽的味道。<br/>　　<br/>　　    王清雪看着王惊蛰，小声地跟徐闯说道：“你朋友看起来怎么怪怪的，这么热的天穿那种衣服不热么，去天目山又不是去化装舞会，这是cosplay啊还是行为艺术呢？”<br/>　　<br/>　　    徐闯尴尬地一笑，憋了半天后说道：“可能，他就这一件衣服吧？”<br/>　　<br/>　　    确实，自从见到王惊蛰到现在，徐闯真的就只看到过他穿这一件衣服，从来都没有换过，也是因为这一点，徐闯才经常请他吃面的，可能这哥们的条件有点差。<br/>　　<br/>　　    李红拂很少说话，多数的时候都是安静地站在人群里摆弄着手机，但她的眼神有好几次都从王惊蛰的身上掠过了，她不认识对方，但却看出了他身上穿着的瑞福祥褂子。<br/>　　<br/>　　    其实李红拂和王惊蛰在这之前还真曾经有过一面之缘，只是当时天色太黑，双方的注意力又都没在彼此的身上。<br/>　　<br/>　　    不知道是第几次打量王惊蛰的时候，两人的眼神就对上了，对上眼后双方停顿了大概不到一秒钟，就一触即逝了。<br/>　　<br/>　　    “这是个有意思的姑娘啊”王惊蛰心里嘀咕了一句。<br/>　　<br/>　　    “这是个有故事的男人……”李红拂差不多也同时在心里念叨了一句。<br/>　　<br/>　　    只一个眼神，他们互相之间就瞧出来了，对方各有不同之处。<br/>　　<br/>　　    二十分钟后检票上车，一路无话，车程四十多分钟到了临安，下车后他们直接就上了通往天目山景区的大巴。<br/>　　<br/>　　    上午十点左右，到了天目山，一行人跟欢脱了似的顺着山路往山上走。<br/>　　<br/>　　    天目山的风景还是不错的，云雾缭绕，景色秀丽，俗称大树华盖闻九州，但再好的山水人气一多之后少了原始的自然味，就显得特别俗了，特别是被开发成景区的地方，到了十一的时候，你都不知道自己是来看山水还是来看人头的。<br/>　　<br/>　　    几个人顺着山路漫漫地走着，欣赏着两边的风景，他们现在去的地方是天目山的主峰仙人顶，海拔一千五百多米。<br/>　　<br/>　　    登山的时候，王惊蛰就落在队伍后面闲庭阔步地跟着，他来天目山跟游玩没啥关系，全是为了徐闯，期间他也曾经观察过另外几人，让他颇为担忧的是，除了徐闯以外他女朋友王清雪还有一个叫赵文亮的人也都有了麻烦。<br/>　　<br/>　　    “这帮糟心的孩子，到底是要沾上什么事呢……这山上人多的走走路都容易擦枪走火，我怎么就没看出来，这问题在哪啊？”<br/>　　<br/>　　    王惊蛰略显迷茫，今个十一，来天目山旅游的人放眼望去都成人山人海了，但却偏偏就徐闯他们那有问题，这种古怪让他都有点懵逼了。<br/>　　<br/>　　    一路爬到仙人顶，游览了几个景点之后，时间就到了下午四五点钟了，距离天黑还有三四个小时左右，一日游差不多也该结束了。<br/>　　<br/>　　    “现在下山我们正好能赶上一趟车回去，看你们好像没有返回的意思呢？”王惊蛰来到徐闯身边问道。<br/>　　<br/>　　    王清雪雀跃地挥舞着小手说道：“我们今天不回去了”<br/>　　<br/>　　    “啊？”<br/>　　<br/>　　    “好不容易出来玩一次，当然要玩地尽兴点了，七天假期总不至于出来一天就回去了吧？我们之前商量过了，去租几顶帐篷，然后去山里露营，体现下自然风情什么的……”<br/>　　<br/>　　    “唰”王惊蛰一听脸色就变了，他皱眉说道：“露营啊，山里晚上全是蚊虫可能还有走兽什么的，不太安全吧”<br/>　　<br/>　　    王清雪瞥了瞥小嘴，说道：“这么多人呢，你们四个男人还怕啊，多好的当护花使者的机会啊”<br/>　　<br/>　　    王惊蛰幽幽地叹了口气，王清雪一句话就把问题给点出来了，他们要露营。<br/>　　<br/>　　    古人都说，遇水莫渡，逢林莫入，这些生长在红旗下受着现代化教育的孩子们，恐怕把老一辈人常念叨的谚语都给抛在脑后了，听老人一言得少吃多少亏啊。<br/>　　<br/>　　    王惊蛰劝了一句就不再劝了，算他一共七个人，有五个赞同的，就他这一个反对的，有个屁用？
