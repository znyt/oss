　　“管好你们自己就行。我还用不着你们来救。”林峰瞥了他们一眼道。
<br/>　　“呦呵，还挺狂。现在的年轻人啊，这群00后，没多少本事不说，还特别喜欢装逼逞英雄。”
<br/>　　一个汉子冷笑道。
<br/>　　“行了，变异狼出现了，立即出发开始猎杀行动！”
<br/>　　吴刚打断了他们的争吵立即带着小队人出发了。
<br/>　　“哼，小子，最好躲起来，可别吓尿了。”
<br/>　　三人嗤笑的旋即也都跟着小队出发。
<br/>　　林峰也跟在后面出发了。
<br/>　　下了楼，地上躺着许多丧尸，看样子是这个猎豹小队之前歼灭的。
<br/>　　“看，就在那边！”
<br/>　　吴刚指着远处到。
<br/>　　在远处可以看到一头看起了像是头狼的玩意在啃咬丧尸。
<br/>　　“的确是四阶的。”
<br/>　　林峰眯着眼道。
<br/>　　“到时候我们会先发射麻醉剂，然后就靠你们主攻了，这怪物虽然的皮毛很厚实，若是子弹无法集中关键点，效果不大，这几天下来，我们弹药不多了，所以这一战，得靠你们。”
<br/>　　吴刚道。
<br/>　　“没问题，一头四阶的罢了，我们足足三个四阶高手，还怕他？”
<br/>　　“就是，吴队长你就带着那个拖油瓶小子在后面躺着吧，保证带你们躺赢。”
<br/>　　吴刚没有多说什么，他虽然是训练有素的特战队员，在技巧和军事素养上，他自认他比所有人都要强，但是没办法，人家是四阶的，实力比他要强许多，在这实力差距面前，他曾经日复一日训练的那一套压根没什么用处。
<br/>　　不说别的就是单单一个力量训练，他们哪怕是想提升十来斤的力量，那得付出多少汗血。
<br/>　　可是现在，只要提升一个级别，那就是几千斤力量的升华。
<br/>　　这就是来自于基因蜕变的力量。
<br/>　　“队长，这群人可以吗？”
<br/>　　几个特战队员道。
<br/>　　“他们都是四阶的，问题不大。”
<br/>　　吴刚道。
<br/>　　说着他对林峰道：“小兄弟你就和我们呆在这里吧，这一次任务，若是完成了，我还是会让你通过的。”
<br/>　　林峰愣了，本来他也想去猎杀的，看来这吴刚不怎么相信他啊。
<br/>　　不过也无所谓，既然躺着都可以拿到奖励，那就躺着好了，就当是看大戏了。
<br/>　　“全员原地警戒！”
<br/>　　话说。
<br/>　　三个四阶战士冲了过去，他们的武器都是些砍刀。
<br/>　　那正在吞食丧尸的变异狼察觉到了他们，顿时转过身来龇牙咧嘴，在他身上可以看到许多弹孔，都是猎豹小队之前射杀留下的。
<br/>　　“扁毛畜生，去死吧！”
<br/>　　三个人二话不说就正面冲锋。
<br/>　　林峰见此摇了摇头，这作战经验也太傻了吧，这变异狼的力量肯定在人之上。
<br/>　　居然还要正面交锋。
<br/>　　而且狼的头是最坚硬的，他的弱点就在腰上面，铜头豆腐腰说的就是狼。
<br/>　　要是林峰和陈勾他们要对付这个变异狼的话，那肯定就是让陈勾正面去勾引，毕竟陈勾两百斤大胖子肉多，绝对有吸引力，然后林峰就侧面偷袭狂攻对方的腰肢和菊花。
<br/>　　不过很显然，这三个人压根没有什么战斗意识，或者说他们就是对自己自信心爆棚了。
<br/>　　轰！
<br/>　　三人攻击倒也颇为的凶猛，但是落在那变异狼身上，只是将变异狼给击退了。丝毫无损，甚至反而激怒了变异狼的怒气。
<br/>　　“怎么可能。”
<br/>　　“屁事没有？”
<br/>　　“该死，我听说狼的头很硬，我们刚才应该攻击他软肋！”
<br/>　　三人面色微微一变。
<br/>　　“攻击软肋？怎么，你怕了？怕就滚过去和那个乳臭未干的毛头小子待在一起躺着！老子就是要和他正面刚！”
<br/>　　“没错，怂个毛，继续正面刚！”
<br/>　　两人冷笑道旋即再次冲了过去。
<br/>　　另外一人犹豫了一下也是跟着过去继续砍杀。
<br/>　　变异狼龇牙咧嘴，怒吼着也是冲了过来。
<br/>　　“去死吧！”
<br/>　　砰！
<br/>　　三个人直接被这股巨大的力量给撞飞了，在地上滚了七八圈，一个个感觉自己一口鲜血都差点喷出来了。
<br/>　　“怎么可能，这么大的力量？”
<br/>　　“我们三个联手居然都不是他的对手？”
<br/>　　“该死，这头狼的力量少说也是趋近于五阶的存在了！”
<br/>　　三个人都面色难看到了极点，他们虽然和变异狼都是一个级别的，但问题是，他们的力量差距太大了，他们是四阶的最低门槛，而这个变异狼却是最高门槛，都快要五阶了。
<br/>　　这就像是千万富翁，一千万身价也是千万富翁，九千九也是千万富翁，但是彼此之间差了不少。
<br/>　　“队长，他们似乎无法拿下那个变异狼啊！”
<br/>　　猎豹小队一人道。
<br/>　　“该死，这变异狼居然这么强大。”
<br/>　　吴刚面色难看。
<br/>　　“跑啊！这变异狼太厉害了，我们不是对手。”
<br/>　　“妈的，先撤！”
<br/>　　三个人爬起身来就是掉头就跑，显得无比的狼狈。

<br/>　　说着他就带队离去了，他们仰仗的就是这三个四阶的，可是这三个四阶的很显然已经没有战意了。

<br/>　　这时。
<br/>　　林峰忽然开口道。
<br/>　　“你来对付？”
<br/>　　吴刚愣了完全不知道对方这是什么意思。
<br/>　　“小兄弟现在不是逞能的时候，这个时候了，你还愿意挺身而出，不得不说，你十分的勇敢，但有些问题，不是勇敢就能解决的。还是先撤离这里！”
<br/>　　吴刚道。
<br/>　　这个时候。
<br/>　　三个四阶战士也跑了过来。
<br/>　　“吴队长快跑，这个变异狼太厉害了，我们三个人不是对手！”
