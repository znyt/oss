　　帮助当地警方打掉了那个毒窝，从警方手里拿到了奖励后，李霄力消失了，没有人知道他去了哪里，所有的联系方式都中断了，没人能联系到他，仿佛从人间蒸发了一样。
<br/>　　一年后，李霄力换了一张脸，换了身份，出现在杭山市。
<br/>　　在消失的这一年时间里，李霄力去了南鲜国，在那里他找到了一家有名的整容医院，改变了自己的容貌，然后花钱搞了一个南鲜国的国籍，从此原来的李霄力消失了，他摇身一变，变成了南鲜国华裔，父母是南鲜国的华人商人。
<br/>　　李霄力来到杭山市后，利用之前从他原来老大手里收集来的资料自立门户，表面上开着酒吧，实际上则是以毒品拆家暗中大肆敛财。李霄力在南鲜国的时候搭上了“毒皇后”玛丽科娃的线，从境外以比国内低数倍的价格拿货，然后在以他在国内、甚至东南亚区域所掌握的销售网络，把东西提价卖出去，赚取中间的差价。
<br/>　　几年的时间，李霄力身价倍增，在杭山市的酒吧也开了好几家，俨然是一个成功人士。
<br/>　　一辆红色保时捷卡宴停在酒吧门口，范思哲衬衫、阿玛尼西裤、手工鳄鱼皮鞋，一副成功人士打扮的李霄力迈步走下卡宴，推开酒吧门走了进去。
<br/>　　“力哥！”
<br/>　　“力哥！”
<br/>　　“力哥！”
<br/>　　看到李霄力走进酒吧，负责在酒吧中看场子、维持这里秩序的手下纷纷走上来，向李霄力问好。
<br/>　　李霄力走到吧台前，从手包中拿出一根雪茄，剥掉锡纸，也没有修剪直接咬在嘴上，用打火机点燃，吸了一口，对这里的酒吧经理问道：“事情办得怎么样了？”
<br/>　　“差、差不多办好了。”酒吧经理擦了擦额头上的汗回答道。
<br/>　　尽管酒吧中打着空调，可汗水还是不断从他额头上渗出来。
<br/>　　“差不多办好了？”李霄力眼睛中寒芒闪烁，一把揪住酒吧经理的领带，拽到自己面前，冷声说道：“办好了就是办好了，没办好就是没办好，差不多办好了是什么意思？”
<br/>　　“力、力哥，有个娘们不听话……”酒吧经理哆嗦着回答道。
<br/>　　嗤啦——！
<br/>　　“啊——！”
<br/>　　酒吧经理的话还没说完，李霄力夹在手里的雪茄就对着他的脸烫了下去，瞬间就在他脸上烫出了一个水泡，令得酒吧经理哀嚎不已。
<br/>　　“废物！”
<br/>　　李霄力一脚把酒吧经理踹倒一边，对旁边一个马仔说道：“三儿，以后这里归你管，做好了有赏，做不好他就是你的下场！”
<br/>　　“明白了，力哥！”
<br/>　　被称为“三儿”的马仔忙不迭的点头，他没有被这突然从天上掉下来的“馅饼”给砸晕。说好听了，这是天上掉下来了一块大馅饼正好砸在了他的头上；说不好听了，这他妈掉的压根就不是馅饼，是一颗炸弹而且还是一颗极不稳定，随时都有可能会爆炸的炸弹。
<br/>　　三儿很清楚，自己老大从不开玩笑，做好了的确有赏，钱、房子、车子、女人，要什么有什么，一夜之间就能变成人上人；做不好，那就会死得很惨，会被了李霄力当做运毒工具发挥最后的价值，然后被送到境外，就算是死了，也不会有人知道。
<br/>　　“那个不听话的娘们在哪儿？”李霄力扔掉手中已经熄灭的雪茄，看也不看倒在地上跟一只煮熟的虾一样，蜷缩着身体的酒吧经理，对三儿说道：“老子就不信这个邪了。”
<br/>　　“力哥，他怎么办？”三儿看了眼躺在地上酒吧经理，问道。
<br/>　　“老规矩办。”李霄力毫无感情的说道。
<br/>　　听到李霄力的话，酒吧经理慌了，也顾不得被烫出大水泡的脸，抓着李霄力的裤脚哀求道：“力哥，我错了，我真的错了。力哥，我不要当‘皮箱’，我还有老婆孩子，我不能当‘皮箱’！”
<br/>　　李霄力眼神冷漠的看了眼抓着自己裤脚的酒吧经理，抬脚踢在他的下巴上，把他的下巴踢得都脱臼，然后抬脚踩在他脸颊水泡上，把水泡踩破，并且用力碾压着，说道：“要是没有我，你就是个臭吊丝，你这辈子也就只能电脑撸，你的孩子注定只能射在纸巾上！你现在的一切都是我给你的，我早就说过了，事情做好了，你就是人上人，事情做不好，你他妈连吊丝都做不成！看在你这两年帮我做事情的份上，我可以向你保证不动你的老婆孩子，要是不听话，我就把你老婆孩子给卖了！”
<br/>　　李霄力抬起脚，没有再去看被自己踩得脸颊血肉模糊的酒吧经理，朝着酒吧存酒的房间走去。
<br/>　　这个房间表面上是用来存酒的，但是搬开酒箱子会看到地板上有一块长方形的颜色比周围颜色略深，就像是破损以后重新修补过的样子。
<br/>　　三儿扣住长方形地板上的凹槽，把那块长方形地板掀开，露出地下室的入口。
<br/>　　地下室刚打开就能听到从地下室中隐隐有女人传来的抽泣声，李霄力沿着阶梯走进地下室。这个地下室才是酒吧最黑暗的地方！
<br/>　　地下室的面积就跟酒吧面积一样大，地下室被分隔出好几个隔间，这些隔间都安装了通了电的铁门，铁门背后关押着“皮箱”，也就是身体藏毒运毒的人。
<br/>　　这些人中有男人，也有女人，其中女人的数量要明显多于男人。毕竟女人比起男人，要更容易控制。
<br/>　　地下室最大的一处隔间就是用来包装毒品，并且让“皮箱”吞毒的地方。
<br/>　　桌子上放着一个盘子，里面有着好几个用安全套包裹好的毒球，桌子旁边的地上，跪着一个年纪在二十岁左右的年轻女子，正在对着逼着她吞毒的马仔苦苦哀求。
<br/>　　她看到过那些人吞毒的过程，也看到有人在吞毒后，因为毒球在体内破裂，口吐鲜血很快死去的场景。毒球破裂导致吞毒人死亡后，这里的这些马仔做的第一件事情，就是把死去吞毒人的肚子剖开，将里面还没有破裂掉的毒球拿出来，然后才是把尸体装入铁桶，灌入水泥，等到天黑后沉入到江水中。
