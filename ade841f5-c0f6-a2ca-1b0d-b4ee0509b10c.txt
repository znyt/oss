　　五个亿灵石，便是五百枚皇品灵石，五万枚王品灵石。
<br/>　　曾经白萱在虚空海闯荡许久，才得到一枚皇王品灵石，现在为了拍下这极道仙兵的残片，几乎已经是疯狂了。
<br/>　　这众人的震惊，还没有消散。
<br/>　　在下一刻，又是一道声音传来。
<br/>　　“老夫出十亿！”
<br/>　　这一道声音，如同一刻惊雷般，爆炸出来，将所有的价格，都给死死的压了下去。
<br/>　　这声音的主人，正是二号包厢之中的百蜈魔尊。
<br/>　　这百蜈魔尊的一句话，直接是让这极道仙兵的残片，再次翻了个倍。
<br/>　　这翻倍，可不是那么简单，足足是一下子涨了五个亿的灵石。
<br/>　　“这极道仙兵的残片，为什么不是我得到的！”
<br/>　　下方一尊尊武者，眼神都是嫉妒的发狂。
<br/>　　这极道仙兵的残片，若是他们谁得到的话，可就是真正的发了大财了。
<br/>　　“可恶，这地魔门的人，居然也来了。”
<br/>　　“我们准备的财富不够了！”
<br/>　　天字包厢之中，许许多多不甘心的声音响起。
<br/>　　可是这十亿的价格，已经是足以让在场绝大部分人退去了。
<br/>　　百蜈魔尊自信大笑道。
<br/>　　“哼，若是无人出价的话，这极道仙兵的残片，便是属于老夫了。”
<br/>　　“我出二十亿！”
<br/>　　百蜈魔尊的话音刚落，一道极为轻蔑的声音，突然响起。
<br/>　　这声音，却是陌生的一句声音，之前从来没有开口过。
<br/>　　“是谁？”
<br/>　　“天字一号包厢？”
<br/>　　一尊尊武者，望了过去，才察觉到，这开口的声音，居然是来自天字一号包厢之中。
<br/>　　这天字号的包厢，其中的排列，也代表着各大势力的地位，天字一号包厢，便是属于通天阁的。
<br/>　　在通天阁拍卖会上的东西，居然是被通天阁的人拍走？
<br/>　　无数质疑的眼光，看向看台上，这看起来，似乎更加像是一个托儿。
<br/>　　“这天字一号包厢，虽然是属于我们通天阁的，不过其中的阁下，却并非是我们通天阁的人，还请诸位放心。”
<br/>　　灵尊真人笑着解释道。
<br/>　　“二十亿灵石，不知道还有没有那位阁下愿意加价。”
<br/>　　“二十五亿！”
<br/>　　二号包厢之中，百蜈魔尊再次开口道。
<br/>　　“三十亿！”
<br/>　　“三十五亿！”
<br/>　　“四十亿！”
<br/>　　不管是百蜈魔尊如何开口，这一号包厢的中的人物，却都是极为气定神闲，反而是毫不在意这无数财富一般。
<br/>　　“这一号包厢中的人，到底是谁？”
<br/>　　“能够得到通天阁的包厢，这可是通天阁都需要去巴结拉拢的人！”
<br/>　　“此人的财富，也是多的吓人！”
<br/>　　楚风眠也听得出来，这一号包厢中的人物开口，这种气定神闲，绝非是伪装出来的，而是这价格，对他来说，都算不得什么。
<br/>　　“我来看看，到底是谁？才拥有如此财富！”
<br/>　　楚风眠心神一动，灵识直接钻入到了一号包厢之中，一样便是看见其中，站着七八个年轻人。
<br/>　　这一个个年轻人的境界，都是神力境九重，其中的实力，都不在落长恨之下，最为强大的那一尊，甚至是比起洪狱，都要强大许多。
<br/>　　楚风眠感觉到，此人的实力，还有着极大的隐藏，现在只是如同一头沉睡的雄狮一般。
<br/>　　一旦是全部力量展现出来，只怕是足以跟生死境高阶的强者抗衡的人物。
<br/>　　“这一股气息，是圣王不败秘法，是圣王宗的人？”
<br/>　　楚风眠察觉到这些人身上的气息，一瞬间便是明白过来。
<br/>　　这些人，居然都是来自七大宗门之一，圣王宗的人。
<br/>　　这圣王宗，是七大宗门之中都称得上古老的宗门，比起荒古门，甚至还要几分。
<br/>　　“乾元师兄，这极道仙兵的残片，您是势在必得了。”
<br/>　　一尊圣王宗弟子，正对着那一群人中的最强者，讨好的开口道。
<br/>　　“极道仙兵的残片，这可是我们中域都未曾出现过的宝物，您若是可以得到，只怕会一跃成为我们圣王宗的大人物。”
<br/>　　“哼，这次来到北域，本来以为这偏远之地没什么，没想到居然是让我碰到了阴阳魔尊的洞府，我得到了他的财富，足足是有着数百亿灵石，我倒是要看看，这极道仙兵的残片，谁可以跟我来抢。”
<br/>　　乾元嚣张的开口道，其中言语中，自信到了极致。
<br/>　　“这些北域土著，怎么可能跟我能抢，我倒是要看看，这些土著还有什么能耐。”
<br/>　　“北域的家伙，不过都是一些土著罢了，这极道仙兵的残片落到了他们的手中，也是蒙尘，只有落入到乾元师兄的手中，才是真正的有德者得。”
<br/>　　一尊圣王宗弟子，急忙拍马屁说道。
<br/>　　“乾元师兄你看，这二号包厢中的土著，已经喊不出价格了。”
<br/>　　现场，沉默了许久，从二号包厢之中，才再度传出来一个声音。
<br/>　　“四十二亿！”
<br/>　　这价格，天蜈魔尊喊出的一刻，便是他能够拿出的最高价格了。
<br/>　　喊出这一声后，便是好像将身上的所有力气，都给抽干了一样。
<br/>　　“四十三亿！”
<br/>　　乾元冷冷一笑，又是轻描淡写之中，报出了一个价格。
<br/>　　这一个价格，直接是让天蜈魔尊的脸上露出几分失望。
<br/>　　他的所有财富，已经用尽，再也无法竞争。
<br/>　　“四十三亿！这简直是一个天价，通天阁的拍卖会上，好像还从未出现过如此价格吧。”
<br/>　　“不知道这一号包厢中的人到底是谁，居然有这如此财富。”
<br/>　　一尊尊武者议论起来。
<br/>　　似乎已经是尘埃落定。
<br/>　　就连乾元的脸上，都是露出了几分自信，看向这极道仙兵残片，眼神中已经满是贪婪。
<br/>　　想到下一刻，这极道仙兵的残片便是就将属于他，他的心中便是无比激动起来。
<br/>　　“我出五十亿！”
<br/>　　在这沉默之中，楚风眠突然出言了，如同一匹黑马，直接杀了出来，将这价格，再次拔高。
<br/>　　“什么？十一号包厢中的人，居然再次开口了！”
<br/>　　“之前天魔九宝的时候，此人都没有出价，本以为他的财富已经用光了，现在居然又出价了！”
<br/>　　“五十亿……”
<br/>　　一些武者，听到这数字的一刻，都是瑟瑟发抖，五十亿灵石，到底是一笔多么庞大的财富，让他们想象都无法想象出来。
