　　与他相对的是，玉玑子却气的七窍生烟，从莲叶上站起来，立刻就要作法，将他们这些骂自己的人全杀了，耳边突然听见影魅大喝一声：“助手！”
<br/>　　吓的立刻不敢动了。
<br/>　　这一幕小马看在眼里，虽然被罡风吹得浑身都难受，还是开心的不行，双手拢在嘴边，冲他喊道：“你个老不死的变态，你想让别人当狗，其实你才是最狗的，你比所有的狗都狗！”
<br/>　　玉玑子气的差点岔气，随即恶狠狠地看了那些泡沫里的生灵一眼，大声吼道：“好啊，你们有骨气，那就一起去死！”
<br/>　　法阵的缺口合上了。
<br/>　　四个盘古僧停止了作法，其中一个带着愤怒质问影魅：“我等是佛门弟子，助你擒敌，本也是为了超度而来，怎可又节外生枝，种下这般事端！毕竟是几百条性命，本该搭救，如今却要看着他们死吗？”
<br/>　　叶少阳听见这话，忍不住对那盘古僧说道：“几位，就凭你们这番话，将来我横扫轩辕山之时，一定不杀你们盘古僧，至少不杀你们几位！”
<br/>　　鲲鹏大笑：“大言不惭，人间那小地方让你自信爆棚了是吗，居然还打轩辕山的主意？”
<br/>　　叶少阳望着他，冷冷笑道：“你最好能活到那一天，少不了你的份！”
<br/>　　一边招呼碧清作法，腾起波浪，将这些生灵接下来，泡沫碎裂之后，这些生灵又重新落在了水里，感受着阴阳罡风，一个个疼的死去活来。
<br/>　　叶少阳看在眼里，心中很惭愧，冲他们说道：“连累各位了，其实，你们可以骂我的，我并不生气……”
<br/>　　那蜘蛛精疼的现出了本尊，八条腿一起打颤，却咬牙说道：“叶天师这是哪里话，本来这一切都是法术公会干的，你唤醒我们，要带我们离开，我们怎么能忘恩负义，之前念咒，已是铸成大错，再让我们骂你，那我们是什么了，就算今天活下去了，将来也不配做人了！”
<br/>　　“好！”
<br/>　　小马一巴掌打在蜘蛛脑袋上，却用力过猛，把他拍到水里，急忙又捞出来放在莲叶上，激动地说道：“你这妖怪很讲义气啊，你以后就是我白云城主的兄弟，以后就跟着我混吧！对，还有你们，以后大家都是兄弟！”
<br/>　　大伙也深受鼓舞，为自己的选择感到骄傲，但就在这时，阴阳罡风更加强烈，波涛也汹涌澎湃起来，连碧清也掌控不住了，只能勉强用莲叶护住他们，但作用不大，而且经过这一段时间的消耗，她的修为也耗费了不少，愈发感觉到吃力了。
<br/>　　“四位，请继续念咒。”影魅客气地请求了一声。
<br/>　　“法阵都关闭了，他们也都做出了选择，还念什么！”几个盘古僧很是不爽。
<br/>　　“你们念咒，让叶少阳跟着念。”
<br/>　　听见的人都诧异地看着他。
<br/>　　小马冷哼道：“又疯了一个，怎么都是疯子。”
<br/>　　叶少阳心中却咯噔了一下，果然，影魅交代完之后，转头望着叶少阳（他通体乌黑，就像一团影子，根本看不到脸上五官，只能通过他转头的方向来判断应该是看着自己），缓缓说道：“叶少阳，你知道你为什么有机会活下来？如果当时我们想杀你，你早就死了，你认吗？”
<br/>　　“我认。”叶少阳回答得很干脆，他早就猜到，自己之所以没死，不可能是因为星月奴闭关、没人下达命令的缘故，肯定还有更深层的原因。
<br/>　　“不杀你，有两个原因，”影魅语气平静地跟叶少阳解释，“第一，我们想要山海印，你死了，就再也找不到了。”
<br/>　　叶少阳点点头。
<br/>　　“第二，可能你猜不到，因为，你是天选之子、应劫而生。”
<br/>　　叶少阳皱起眉头，深吸一口气，说道：“你们也相信这个？”
<br/>　　“至少，三界之内，所有势力都相信。法术公会想要的，是接管人间法术界，不管你信不信……我们是希望带改造人间，带领人间法术界平安度过天劫。不然，我，清长风，还有这几位师父，为什么要帮助法术公会？”
<br/>　　几个盘古僧听了这话，沉默不语。
<br/>　　叶少阳冷笑道：“别说得这么高大上，你们干的坏事也不少。”
<br/>　　“对，我们每个人，都有自己的私心，但这私心是建立在法术公会能一统人间的基础上……你是酆都大帝选中的应劫之人，杀了你，阴司那边没法交代，其余的势力，也会因此而攻击我们，就算我们守住风声，没了你，天劫之时，三阶大乱，到时候要一个破碎的人间，又有什么用？”
<br/>　　这番听上去推心置腹的话语，反倒让叶少阳怔住了，缓缓摇头说道：“不，之前清长风和星月奴，都是想杀我的，还有你……”
<br/>　　“他们一直不相信你是应劫之人，最近，他们相信了，我从来没想杀你，我只想擒住你，给你洗脑，让你加入我们这边。”
<br/>　　叶少阳“呵呵”了两声回应，“你说完了没有，你说完了，我倒是要问你一个问题：现在我被你困住了，你为什么要跟我说这么一大堆，总不会是良心发现吧？”
<br/>　　“叶少阳，不到最后一步，我依然不想杀你，我说这么多，是想告诉你，只要你做一件事，一切过去的恩怨，一笔勾销，你知道是什么……跟着念咒吧。”
<br/>　　对那四位盘古僧使了个眼色，在神识中嘱咐了几句，四人开始念咒，却不是之前的咒语，而是《接引咒》，传闻是当年准提道人在人间接引恶人用的，一旦跟着念咒，自身一切业障全消，就跟着准提走了。
<br/>　　后来这一段《接引咒》，就成了南传佛门为刚入门的弟子剃度时候念的经。
<br/>　　这时候念接引咒，意义再明显不过了。
<br/>　　“呵呵，小叶子，你说这帮人是不是疯了，他让你念，你难道就得跟着念啊。”
<br/>　　一转头看叶少阳，却脸色惨白，忙问他怎么回事。
<br/>　　叶少阳咬着牙关，吐出四个字：“杀人诛心！”
<br/>　　影魅并没有威胁他，但是他说了这么多，叶少阳早就听明白了，自己而今最大的软肋，就是身边这些人……这些先背叛了自己、关键时刻又选择站在自己这边的诸多生灵。
