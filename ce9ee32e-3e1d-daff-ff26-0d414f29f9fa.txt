　　无论如今武风云的内心再不相信。
<br/>　　眼前的这一幕，却是丝毫无法改变的。
<br/>　　“废物，就连我的实力都不知道，还敢号称镇压我，简直是个笑话！”
<br/>　　楚风眠看了一眼那武风云，眼神中冷笑连连。
<br/>　　“你也给我跪下，去陪着你的那些狗腿子吧！”
<br/>　　说着，楚风眠便是大手一挥，一股庞大灵力，顿时从天而降，压迫下来。
<br/>　　武风云的身躯，在空中被一股巨力束缚，直接落下，无法反抗，跪在了楚风眠的面前。
<br/>　　这样一尊人榜第十一位的风云人物，武风云，如今居然是败倒在了楚风眠的手中。
<br/>　　这个结果，可以说是出乎任何人的意料，谁也不会想到，武风云居然会真的败给一个新人。
<br/>　　“楚师兄饶命啊。”
<br/>　　“楚师兄我们错了，求您放过我们。”
<br/>　　那些跪在地上的风云会弟子，看到武风云都被楚风眠镇压，跪在地上，一个个已经是脸色苍白到了极点，急忙求饶道。
<br/>　　武风云都败了，他们这些狗腿子那个还敢跟楚风眠抗争，一个个脸上都是献媚讨好道。
<br/>　　“求饶？知道错了？可惜晚了。”
<br/>　　楚风眠表情冰冷，这种献媚讨好，对楚风眠可没有任何用处。
<br/>　　“每个人，都将功勋值全部交出来。”
<br/>　　上一次，楚风眠击败牛峰，马允的时候，倒是忘了这件事，不过这两人都新人，只怕也没有功勋值。
<br/>　　但是眼前这些风云会的弟子，不少都是在外门中摸爬滚打的老油条了，一个个只怕都积攒了不少功勋值。
<br/>　　楚风眠如今要兑换圣山上的洞府，需要大量的功勋值，倒是没什么，比起直接抢来的更快了。
<br/>　　这些风云会弟子既然是送上门来了，楚风眠当然不会放过。
<br/>　　“什么，要我们将所有功勋值交出来？”
<br/>　　那些风云会弟子，听到楚风眠的话，一个个脸色煞白。
<br/>　　这功勋值，可是他们积攒了数年的，如今要全部交出来，简直是心痛不已。
<br/>　　面对楚风眠，他们虽然极为惧怕无比，但是却没有任何一人愿意主动交出来。
<br/>　　“怎么，你们不愿意？”
<br/>　　看着这些人的表现，楚风眠的眼神，冰冷到极点，言语中也是露出了几分不悦道。
<br/>　　“既然如此，那我就将你们身上的圣者精血抽走九成吧，就算是弥补我的损失了！”
<br/>　　说着楚风眠，便是就要动手。
<br/>　　“我愿意交。”
<br/>　　一尊风云会弟子，急忙的说道，直接是拿出了身份牌来，递给了楚风眠。
<br/>　　剩下的那些风云会弟子，也都是急忙的一起，将身份牌统统递了上去，生怕一慢，楚风眠就出手了。
<br/>　　楚风眠的话，可不是说着玩的，之前的牛峰，马允二人，已经是一个例子了。
<br/>　　要是被抽去了九成精血，他们也就会变成跟那牛峰，马允一样，在北邙学院外门中垫底的存在，可谁也不想这样。
<br/>　　如今这些风云会弟子，纷纷的交出了身份牌来。
<br/>　　楚风眠心神一动，其中的功勋值，便是全部转入到了楚风眠的身份牌中去。
<br/>　　“才两百多点？这么少？”
<br/>　　楚风眠不由的眉头一皱道。
<br/>　　这数量，倒是比起楚风眠想象中的要少许多。
<br/>　　楚风眠这一下子可是抢走了七名风云会弟子的身份牌，可是现在，居然只有两百多点功勋值。
<br/>　　要知道兑换一次圣山修炼，至少是需要一万点的，这两百多点，简直是九牛一毛。
<br/>　　“这，已经是我们积攒了数年的了。”
<br/>　　一尊风云会弟子，听到楚风眠的话，不由苦笑道。
<br/>　　“一次任务，能够给个十点功勋值，都不错了，像是我们外门弟子，实力不强，只能接取这种任务，那种一次上百点，数百点功勋值的，只有内门弟子，甚至是核心弟子才能接取的。”
<br/>　　外门弟子，在北邙学院中的地位，便是底层。
<br/>　　这圣山修炼，本身都是极为昂贵的一件事，这倒是楚风眠有些忘记了。
<br/>　　“好，既然你们都乖乖交出了功勋值，全都离开吧。”
<br/>　　楚风眠瞥了一眼那些风云会弟子，冷声道。
<br/>　　他心神一动，压迫在那些风云会弟子身上的压力，直接消散。
<br/>　　“是，是。”
<br/>　　这些风云会弟子，一个个如同大赦，直接纷纷离开。
<br/>　　对于这些人，楚风眠还算是网开一面了，不过这倒不是因为楚风眠心慈手软，而是楚风眠不愿意太过张扬。

<br/>　　这两个人不过是两个新人来，没有任何的关系，废了就废了，也不算麻烦。
<br/>　　但是这些风云会弟子，不少都是北邙学院中的老油条了，一个个甚至是跟许多内门弟子，甚至是长老都有些关系。

<br/>　　如今这些人虽然都放了，但是风云会，也彻底完了。

<br/>　　一些弟子，不由有些感叹，在外门中，风云会已经算是一个小势力了，没想到居然就这样毁在了楚风眠这样一个新人的手中。

<br/>　　众人心中，不免都有些大快人心，风云会的名声可不好，欺压弟子，夺取资源，今日被毁灭，不少人都是为之叫好。
<br/>　　“看来，你的风云会完了。”
<br/>　　处理完那些风云会弟子后，楚风眠便是将所以注意力，放在了武风云身上。
<br/>　　“让你交出功勋值，看起来你是不打算按照我的吩咐去做？”
<br/>　　楚风眠看着武风云，饶有兴趣的开口道。
