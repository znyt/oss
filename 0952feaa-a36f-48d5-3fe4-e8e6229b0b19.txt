　　“滚开！”
<br/>　　叶尘直接又是一记八极镇魔神拳打出，
<br/>　　砰！
<br/>　　祁老登时便被打飞了出去。
<br/>　　“还想跑？”
<br/>　　叶尘一拳轰飞了祁老，身体微微一晃，便已经追到韩月公子的身后，一把抓住他的后颈，犹如老鹰抓小鸡一般，将死死擒住。
<br/>　　而此时，
<br/>　　远处上古外，目睹了这一切的三大家族的众人，顿时全部石化了。
<br/>　　在他们眼中，无比尊贵的韩月公子，还有那两位，强悍无比的擎苍宗强者，在那个看似普通的年轻人面前，竟然犹如玩偶一般，被他玩弄于鼓掌之中。
<br/>　　这世界怎么了？
<br/>　　疯了吗？
<br/>　　众人顿时感觉自己的脑袋有点不太够用。
<br/>　　而之前，曾经出言嘲讽过叶尘的那个雷洛风，更是一屁股坐倒在了地上，浑身瑟瑟发抖。
<br/>　　“啊！！”
<br/>　　“你不能杀我！我可是擎苍宗的核心弟子，你杀了我，擎苍宗绝对不会放过你的！”
<br/>　　韩月公子被叶尘擒住之后，顿时吓得魂都要飞出去了，不由得大声嘶吼起来。
<br/>　　叶尘冷冷一笑，
<br/>　　“你似乎忘记我刚刚说过什么，早晚有一天，我会屠灭你们擎苍宗！”
<br/>　　“你！”
<br/>　　韩月公子顿时一滞，一股冰冷的寒气，直接从脚底冲到了头顶，心中无比绝望。
<br/>　　之前他无论闯了多大的祸事，只要将擎苍宗核心弟子的身份搬出来，从来没有敢多说一个字。
<br/>　　可是眼前这个小子，到底是从哪里蹦出来的怪物？
<br/>　　不但一点不怕，反而还嚣张的声称，要灭掉擎苍宗！
<br/>　　莫非是疯子不成？
<br/>　　“不过……”
<br/>　　就在韩月公子心中无比绝望之时，叶尘忽然话锋一转，
<br/>　　“今日我不会杀你！你应该认识绝无敌吧？我想让你帮我给他传个话，你可愿意？”
<br/>　　韩月公子听到这话，当真是喜从天降，哪敢说半个不字，
<br/>　　“愿意！愿意！您请吩咐！我一定帮您把话带给绝无敌师兄！”
<br/>　　“很好！”
<br/>　　叶尘点了点头，
<br/>　　“回去告诉绝无敌，半年之后，我会在迷罗星海等着他，他若是够胆，就过了找我，否则的话，今后我遇到擎苍宗弟子，见一个便杀一个，见两个便杀一双！！”
<br/>　　“滚吧！”
<br/>　　叶尘说完这话之后，随手像丢垃圾一般，将韩月公子丢了出去。
<br/>　　不过，就在他将韩月公子丢出去的那一瞬间，一道微弱到几乎不被任何人察觉的猩红色火焰，被打入到了他的体内。
<br/>　　韩月公子原本都以为自己必死无疑了，怎么也没有想到，叶尘居然真的不杀自己，慌忙连滚带爬，朝山谷外狂奔而去，不过眼眸之中，却透着浓浓的恨意，心中在不停的狂吼，
<br/>　　“小子！本公子不管你到底是什么来头，你竟然敢挑衅擎苍宗，挑衅绝无敌师兄，还如此羞辱本公子，你就等着付出惨痛的代价吧！”
<br/>　　叶尘自然知道，这个韩月公子，心里肯定对自己恨意滔天，不过他并不在乎。
<br/>　　相比于绝无敌，韩月公子不过就是一个小喽啰罢了，而且他表面上放了此人，却早已经在他身上，暗暗做了一点手脚。
<br/>　　解决了韩月公子等人之后，叶尘直接身体一晃，一头扎入下方的那处深渊之中。
<br/>　　深渊很深，足足有数万米！
<br/>　　而且，随着叶尘往下方深入，那道毁灭的气息，就越发的强大，仿佛能够毁灭这世上所有的事物。
<br/>　　不但如此，随着不断深入，空气的温度也开始急速飙升，很快便已经超过了千度以上，而且依旧在飞速提升。
<br/>　　渐渐地，那不停飙升的高温，以及那道充满毁灭的气息，就连叶尘，也开始感受到了一丝压力。
<br/>　　好在这种情况并没有持续太久，就在叶尘已经大汗淋漓，浑身都被汗水浸透的时候。
<br/>　　他总算来到了深渊底部。
<br/>　　然而，
<br/>　　等他看清楚深渊底部的情况，顿时满脸凝重之色。
<br/>　　只见，在深渊最底部，竟然是一片炙红无比的岩浆！
<br/>　　岩浆之中，不知道是何种物质组成，温度高的可怕，按照地球上的量度来估算，至少也在七八万度以上，甚至越往下，里面的温度还要更高！
<br/>　　而那道毁灭的气息，赫然正是来自这些炙热的岩浆下方深处。
<br/>　　“难怪之前擎苍宗在这里耗了这么久，也没有得到灭世魔刀，单单是这些岩浆表面的温度，寻常化神境之下的修士，即便是动用护体罡气，只怕也撑不了多久……”
<br/>　　在心里暗暗感慨了一句之后，叶尘缓缓吸了一口气，将状态调整到最佳，继而心念微微一动，
<br/>　　“五昧真火！开！”
<br/>　　呼！
<br/>　　一团猩红色的火焰，顿时便出现在叶尘周身上下，将他整个人包裹在了其中。
<br/>　　叶尘还没有踏入化神境，无法施展外道神像的神通手段，但是以五昧真火包裹在护体罡气之外，在眼下这种情况之下，比寻常化神天君的外道神像，会更加的实用。
<br/>　　用五昧真火将全身包裹起来之后，叶尘这才缓缓踏入岩浆之中，然后慢慢朝下方继续下沉。
<br/>　　轰！
<br/>　　一进入岩浆之内，温度攀升的速度，顿时更加快了！
<br/>　　十万度！
<br/>　　十五度！
<br/>　　两十万度！
<br/>　　……
<br/>　　才深入不过数十米，周围的温度，竟然已经攀升到了三十万度！
<br/>　　一时间，
<br/>　　就连叶尘也有些犹疑了，
<br/>　　按理说，
<br/>　　即便是那些仙器，在如此炙热的高温之下，恐怕也要被融化了吧？
<br/>　　灭世魔刀，当真在下面吗？
<br/>　　叶尘想要释放出神识去扫描，可惜周围温度实在太高了，超过十米，就会被那恐怖的高温严重干扰，便什么也感应不到了。
<br/>　　终于，在叶尘下潜到百米左右的时候，周围岩浆的温度，已经达到了恐怖的五十万度！
<br/>　　纵然全身上下，有五昧真火包裹着，叶尘身上的法衣也已经尽数融化，叶尘周身的皮肤也开始慢慢发黑、发红，犹如随时都要燃烧起来的黑炭一般！
<br/>　　“不行！倘若再继续往下，我的肉身很快就会扛不住，非得被烧成灰烬不可！”
<br/>　　“难道我注定无缘得到这灭世魔刀吗？”
<br/>　　就在叶尘一阵沮丧，准备退出去的时候，忽然隐约看到，旁边的岩壁之上，似乎有金光闪烁！
