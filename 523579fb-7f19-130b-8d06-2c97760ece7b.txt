　　转眼之间，时间便是过去了数天。
<br/>　　楚风眠来到这归墟天之中，已经是足足一个月，距离归墟天的再度开启，再过几个时辰，便是归墟天的出口关闭的时候。
<br/>　　如今楚风眠来到这归墟天之中，也是逐渐的弄清楚了一些关于归墟天的事情。
<br/>　　这归墟天之大，比起楚风眠曾经想象之中的还要大得多，没有边际，每三年才能开启一次，一共持续一个月的时间。
<br/>　　若是在这一个月的时间内不出去的话，便是只能在去等待三年的时间。
<br/>　　“这附近，看起来已经是没有无上剑气的踪影了。”
<br/>　　在归墟大陆的一处山峰之上，在楚风眠的面前，却是一道庞大无比的剑阵，足足是有着六十道无上剑气，组成的剑阵，这向着楚风眠轰杀而来。
<br/>　　在这几日的时间之中，楚风眠一直都是在寻找着无上剑气，不断的收复这。
<br/>　　现在楚风眠的手中，也已经是有着足足三百多道无上剑气了。
<br/>　　这些无上剑气，全部展现出来的一刻，蕴含的锋利剑锋，都足以是将周围的一切生灵，全部斩杀。
<br/>　　这三百多道无上剑气，也是让楚风眠的九域剑术，得益于再进一步。
<br/>　　只要是楚风眠的实力再度突破的话，便是就足以体现在实力之上。
<br/>　　这无上剑气的奥秘，也逐渐的是被楚风眠所参透，在剑道领悟上，楚风眠足以称得上是天下第一也不为过。
<br/>　　曾经的剑道之主，都是自叹不如，这无上剑气中玄妙之际的剑术，让一些寻常剑修来参悟，只怕是穷其一生都无法全部参悟。
<br/>　　可是对楚风眠来说，却是轻轻松松的，便是将其中的剑术全部参悟透了，更是融入到了九域剑术之中，成为了楚风眠自己的力量。
<br/>　　楚风眠的境界，已经是达到了生死境七重的巅峰，随时都有可能突破到生死境八重。
<br/>　　他的九域剑术，逐渐的完善，就算是如今的九域剑术拿出去，都已经不亚于任何一道天级剑术，足以是让无数剑修去争抢。
<br/>　　在剑道上，楚风眠便是不折不扣的巨头。
<br/>　　“一群无上剑气，也想要来反抗我？统统都给我镇压！”
<br/>　　楚风眠看着眼前的那些无上剑气，组成的剑阵，他的嘴角上露出了几分冷笑，突然之间便是手持祖龙至尊剑，冲了上去。
<br/>　　在楚风眠身边那三百多道无上剑气，其中的力量也是全部加持在了无上剑气之上，顿时令剑锋的锋利程度，要恐怖的百倍不止。
<br/>　　一剑刺出。
<br/>　　顿时这剑阵，便是被彻底打破，随着楚风眠大手一挥，这六十多道无上剑气，都被楚风眠直接收取，吞噬到了她的身躯之中去，一下子都被炼化。
<br/>　　楚风眠身上的剑意，如今便是又再强一步。
<br/>　　“四百道了！”
<br/>　　在收取这六十多道无上剑气之后，楚风眠身上无上剑气的数量，便是已经达到了整整四百道。
<br/>　　这已经是比起当初楚风眠斩杀荒圣君的时候，还要多上了四五倍。
<br/>　　“好！拥有着四百道无上剑气，就算是碰到生死台阶前三重的武者，我都有着一战之力，足以立于不败之地了！”
<br/>　　楚风眠感觉着身上的力量，心中不由的感叹一声。
<br/>　　生死台阶，一共分为九重。
<br/>　　这生死台阶的前三重，便是碎命之境，长生之境，不死之境，这三重境界便是生死台阶的前三重。
<br/>　　当初在来到归墟天之前的时候，就算是一名碎命之境的武者出手，楚风眠便是也只能够去逃走，没有其他任何办法抵挡。
<br/>　　虽然说足以抗衡一二，但是却是必败，没有任何悬念可能。
<br/>　　而之前楚风眠，便是轻而易举斩杀了碎命之境的荒圣君，对现在的楚风眠来说。
<br/>　　就算是他碰到生死台阶二重长生之境，甚至是第三重不死之境的武者，楚风眠都足以有着一战的资本，甚至有可能战胜。
<br/>　　以现在生死境八重的境界，便是足以做到这一切，说出去绝对无人会信。
<br/>　　可是楚风眠身上得到的奇遇实在是太多，云羽天宫的传承，无上剑气，祖龙至尊剑，造化之力，甚至是那战龙决。
<br/>　　那一件不是一旦出事便是惊天动地的机缘，这一切如今都被楚风眠得到，才让楚风眠现在有着如此可怕的实力。
<br/>　　不过楚风眠真正足以击败的，也就是这前三重。
<br/>　　生死台阶九重，分别为碎命，长生，不死，无暇，神武，炼心，天命，至天，不朽九重。
<br/>　　生死台阶的前三重，碎命，长生，不死三重境界，主要还是生命力的变化，对实力没有质的变化。
<br/>　　但是等到这生死台阶的第四重，无暇之境的时候，可就截然不同了。
<br/>　　生死台阶的第四重，无暇之境，便是代表着武者身躯的无暇，达到这一重境界，将会将武者身躯之中的一切杂质清除，令身躯达到一种完美的地步。
<br/>　　一旦是成功蜕变，实力将会暴涨无数倍。
<br/>　　碰到无暇之境的武者的话，以现在楚风眠的实力，能不能与之抗衡，楚风眠心中都还没底。
<br/>　　不过楚风眠倒是也并不担心，他如今有着新的底牌，云羽天宫，一旦是遇到强者不敌，楚风眠也可以退回到云羽天宫之中，足以利于不败之地。
<br/>　　来到这归墟天，楚风眠最大的收获不是无上剑气，而是这云羽天宫，一个云羽天宫，足以是抵得上楚风眠的一条性命。
<br/>　　“是时候，也该离开了！”
<br/>　　楚风眠已经是感觉得到，归墟天的法则已经是开始变化这，即将要关闭。
<br/>　　虽然楚风眠可以利用云羽天宫中的法阵离开，但是这样的话，很有可能会让其他人怀疑楚风眠有着可以进出云羽天宫的办法。
<br/>　　楚风眠可还不想就这样早早的暴露。
<br/>　　“走！”
<br/>　　楚风眠身躯一动，遁光入剑，直接向着归墟天的出口飞去。
<br/>　　这一次楚风眠回归，他也想要看看，那些夜家的人，还能如何对付他。
<br/>　　楚风眠的遁光，一直飞行，在前方便是突然之间，出现了一道裂痕。
<br/>　　这裂痕的背后，就是九州。
<br/>　　“嗖！”
<br/>　　楚风眠的遁光，毫不停留的，直接冲了出去。
